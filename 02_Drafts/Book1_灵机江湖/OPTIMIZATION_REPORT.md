## 八、技术细节

### 优化方法论

**第6章优化实例**：
```
1. 问题诊断：收服过于简单 → 缺乏情感深度
2. 解决方案：增加创伤记忆 → 37次被拒绝
3. 情感设计：共情 → 看见 → 哭泣 → 治愈
4. 哲学升华：淡墨之美 → 对抗完美主义
5. 文化融入：中国画理论 → 自然嵌入对话
6. 字数控制：4000字 → 6000字（+50%）
7. 质量验证：催泪测试 → 通过
```

### 写作技巧应用

1. **"冰山理论"**：只展示1/8，暗示7/8
   - 37次被拒绝：只详写3次，其余用数字暗示

2. **"契诃夫的枪"**：出现的元素必有用
   - 铸角上的纹路 → 后续成为"活"字的载体

3. **"显示而非告知"**：用行动展示情感
   - 不说"铸很孤独"，而是描写它独自吃草的动作

4. **"情感真实性"**：避免完美角色
   - 麦洛也承认"我也被说不够好"，增加共鸣
