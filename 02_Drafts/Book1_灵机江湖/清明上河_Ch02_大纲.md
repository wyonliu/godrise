# 清明上河 第2章·寻夫 - 详细大纲

**字数**: 4000字  
**场景**: 苏瑾言的住所 → 记忆档案馆  
**核心**: 苏瑾言背景故事 + 陈默线索 + 记忆灵机设定

---

## 开场：苏瑾言的住所（800字）

**场景**：汴京城东区小院，堆满考古设备，墙上贴满陈默的照片和笔记

**关键道具**：记忆照片（会发光的照片）
- 只要还记得，照片就发光
- 如果遗忘，照片消失
- **寻梦环游记核心设定引入**

**维塔吐槽**：
```
维塔："你在虚拟世界住了三年？！"
苏瑾言："虚拟世界九年。时间倍率1:3。"
```

## 陈默的故事（1000字）

**三年前的发现**：
- 陈默在长安遗址发现记忆结晶
- 里面是用灵机写成的诗歌
- 可以永久保存记忆

**陈默的理论**：
```
"Cycle-6文明发现了对抗遗忘的方法。"
"记忆是灵机的源泉。"
"如果记忆永存，就没人能被收割。"
```

**陈默的失踪**：
- 去归墟入口寻找线索
- 三天没回来
- 只留下考古铲和一幅画

---

## 记忆档案馆（1200字）

**设定**：
- 所有玩家记忆都被记录
- 陈默的档案被标记为"危险记忆"
- 需要灵机共鸣才能查看

**灵机共鸣场景**（情感高潮）：
- 苏瑾言触碰记忆水晶
- 她和陈默的记忆交织
- 闪回：初见、婚礼、考古、争吵、和解、最后拥抱

**陈默的留言**：
```
"瑾言...如果你听到这个，说明我已经不在了。"
"但不要悲伤。记忆可以永存。"
"只要你记得我，我就还活着。"
"去长安。去找我留下的诗。"
```
