# 清明上河 第4章·少年 - 详细大纲

**字数**: 4000字  
**场景**: 长安城门 → 遇到周行云  
**核心**: 周行云登场 + 少年的自卑与不屈 + 第一次战斗

---

## 开场：长安城门（600字）

**场景描写**：
- 长安废墟的城门
- 比汴京更古老，更破败
- 但城门上刻着诗句
- 诗句在发光

**诗句内容**：
```
"长安一片月，万户捣衣声。"
"秋风吹不尽，总是玉关情。"
```

**苏瑾言的解释**：
```
"这是李白的诗。"
"Cycle-6文明的诗人，将诗句刻在城门上。"
"用灵机封存。"
"只要有人读，诗就不会死。"
```

**维塔的发现**：
```
维塔："城门里有人！"
麦洛："什么人？"
维塔："一个少年。他在...战斗？"
```

---

## 周行云登场（1000字）

### 第一次见面

**场景**：
- 城门内，一个少年在和记忆怪物战斗
- 16岁，穿着破旧的剑客服
- 手持一把生锈的剑
- 战斗得很吃力

**记忆怪物**：
- 形态：扭曲的人形
- 特征：没有脸，只有空洞
- 能力：吸食记忆

**周行云的战斗**：
- 剑法很笨拙
- 被怪物打得节节败退
- 但他不放弃
- 一次次爬起来

**麦洛的出手**：
```
麦洛："他需要帮助！"
苏瑾言："等等，这可能是他的试炼——"
麦洛（已经冲出去）："管不了那么多了！"
```

**联手击败怪物**：
- 麦洛用贯虹道攻击
- 周行云趁机补刀
- 怪物消散

---

### 周行云的故事

**第一印象**：
```
周行云（喘气）："谢...谢谢..."
麦洛："你还好吗？"
周行云："我...我没事..."
（试图站起来，但腿软了）
维塔："你明明有事！"
```

**自我介绍**：
```
周行云："我叫周行云。16岁。"
"我是...废物剑客。"
麦洛："废物？"
周行云（苦笑）："是的。我学剑三年，还是最弱的。"
"我的师父说，我没有天赋。"
"我的师兄们说，我是耻辱。"
"但我...不想放弃。"
```

**来长安的原因**：
```
周行云："我听说长安有'剑魂'。"
"Cycle-6文明最强剑客的灵魂。"
"如果我能得到剑魂的认可..."
"我就能证明，我不是废物。"
麦洛："那你为什么一个人来？"
周行云："因为...没人愿意和我组队。"
维塔："为什么？"
周行云："因为我太弱了。会拖累别人。"
```

## 林墨的观察（400字）

**林墨一直在看周行云**：
```
林墨："你的剑法...很像一个人。"
周行云："谁？"
林墨："我妹妹。"
"她也学剑。也很笨拙。"
"但她从不放弃。"
周行云："你妹妹...现在在哪里？"
林墨："她死了。"
周行云（沉默）："对不起..."
林墨："不用道歉。"
"但看到你，我想起了她。"
"所以...我想帮你。"
```

**林墨的提议**：
```
林墨："我可以画出剑魂的样子。"
"如果你能看到剑魂，也许就能理解剑道。"
周行云："真的吗？！"
林墨："但有条件。"
周行云："什么条件？"
林墨："你要和我们一起行动。"
"我们也在找东西。"
周行云："好！我答应！"
```

## 第一次团队战斗（1200字）

### 遇到记忆怪物群

**场景**：
- 进入长安城内
- 街道上出现大量记忆怪物
- 它们从废墟中爬出来
- 数量...至少三十只

**维塔的警告**：
```
维塔："不好！这是记忆潮汐！"
苏瑾言："记忆潮汐？"
维塔："当大量记忆同时被遗忘，就会产生记忆潮汐。"
"这些怪物...是被遗忘的人。"
麦洛："被遗忘的人？"
苏瑾言："是的。Cycle-6文明的人。"
"他们死后，记忆被封存在这里。"
"但随着时间流逝，记忆开始消散。"
"最后...就变成了怪物。"
```

### 战斗分工

**苏瑾言的指挥**：
```
苏瑾言："麦洛，你负责前排！"
麦洛："好！"
苏瑾言："林墨，你用画困住它们！"
林墨："明白！"
苏瑾言："周行云，你保护维塔！"
周行云："我...我能行吗？"
苏瑾言："你能。相信自己。"
```

**战斗过程**：

**麦洛的战斗**：
- 使用贯虹道突击
- 一剑刺穿三只怪物
- 但怪物太多

**林墨的画**：
- 他快速画出一个"牢笼"
- 画完成时，牢笼变成真实的
- 困住了十只怪物

**周行云的保护**：
- 维塔在扫描怪物弱点
- 有怪物偷袭
- 周行云挡在前面
- 虽然剑法笨拙，但很拼命

**危机时刻**：
- 一只巨大的记忆怪物出现
- 它是"记忆领主"
- 实力远超其他怪物
- 一击就打飞了麦洛

**周行云的觉醒**：
```
周行云（看到麦洛受伤）："不...不能让她受伤..."
"我要保护她！"
"我不是废物！"
"我...我能做到！"
```

**灵机爆发**：
- 周行云的剑开始发光
- 那是他的灵机
- "不屈"之灵
- 他冲向记忆领主
- 剑法依然笨拙
- 但每一剑都充满决心
- 最后一剑...刺穿了领主的核心

**胜利**：
- 记忆领主消散
- 其他怪物也消失了
- 周行云跪在地上，喘着气

## 章节结尾（800字）

### 战后

**麦洛的夸奖**：
```
麦洛："你做得很好！"
周行云："真的吗？"
麦洛："真的！你救了我们！"
周行云（眼眶湿润）："这是...第一次有人夸我..."
维塔："你的剑法确实很笨拙。"
周行云："......"
维塔："但你的心很强。"
"笨拙的剑法可以练。"
"但强大的心...是天生的。"
```

**苏瑾言的认可**：
```
苏瑾言："欢迎加入队伍。"
周行云："我...我可以吗？"
苏瑾言："当然。我们需要你。"
周行云（哭了）："谢谢...谢谢..."
```

**林墨的承诺**：
```
林墨："我会帮你找到剑魂。"
"但在那之前，我先画一幅你的画。"
周行云："我的画？"
林墨："是的。今天的你。"
"击败记忆领主的你。"
"这是你的高光时刻。"
"值得被记住。"
```

**五人组成立**：
- 麦洛（14岁，战斗核心）
- 维塔（侦察+吐槽）
- 苏瑾言（35岁，队长+记忆追踪）
- 林墨（28岁，记忆画师）

**各自的目标**：
- 麦洛：寻找父亲
- 苏瑾言：寻找丈夫陈默
- 林墨：复活妹妹林染
- 周行云：找到剑魂，证明自己

**共同的信念**：
- 不放弃
- 相信记忆的力量
- 保护彼此

### 前往长安深处

**场景**：
- 五人站在长安的街道上
- 远处，有一座塔
- 塔顶有光
- 那是...记忆之塔

**苏瑾言的话**：
```
"陈默的线索，应该在那里。"
"记忆之塔。长安的核心。"
"那里封存着Cycle-6文明最珍贵的记忆。"
```

**维塔的警告**：
```
维塔："我的数据库显示，记忆之塔有九层。"
"每一层都有守护者。"
"很多玩家进去后...就再也没出来。"
麦洛："那我们更要去了。"
维塔："为什么你总是这样？！"
麦洛（笑）："因为危险的地方，才有答案啊。"
```

**最后画面**：
- 五人向记忆之塔前进
- 夕阳下，他们的影子拉得很长
- 像是五个战士
- 走向未知的战场
