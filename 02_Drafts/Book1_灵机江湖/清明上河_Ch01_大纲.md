# 清明上河 第1章·入城 - 详细大纲

**字数**: 4000字  
**场景**: 汴京城入口 → 虹桥市集  
**核心**: 繁华展示 + 苏瑾言登场 + 记忆的第一次异常

---

## 开场（500字）

麦洛和维塔从千里江山的边境石碑走出，眼前的景象让她屏住了呼吸。

**视觉冲击**：
- 不是荒野，不是山水
- 而是一座**巨大的城市**
- 城墙高耸，城门如山
- 城门上方悬浮着全息投影："清明上河·山海2.0"

**维塔的吐槽**：
```
维塔："这是...城市？"
麦洛："好大..."
维塔："不是'好大'！是'超级大'！我的地图数据库显示，这座城市的面积是千里江山的十倍！"
麦洛："那很厉害啊。"
维塔："厉害？！这不科学！虚拟世界不应该这么大！渲染引擎会崩溃的！"
麦洛："但它没有崩溃啊。"
维塔："......你的逻辑又让我过载了。"
```

**情感基调**：好奇、兴奋、期待

---

## 进城（800字）

### 城门检查

**NPC守卫**：
- 穿着宋朝铠甲，但眼睛是电子的
- 扫描麦洛的"身份令牌"
- 令牌显示：千里江山通关者

**守卫的话**：
```
守卫："千里江山通关者？14岁？"
麦洛："有问题吗？"
守卫："没有。只是...你是今年最年轻的通关者。"
维塔（骄傲）："那当然！我家麦洛可厉害了！"
守卫（看向维塔）："三尾灵宠？稀有度A级？"
维塔："什么A级！我是S级！"
守卫："系统显示是A级。"
维塔："系统错了！"
麦洛（拉住维塔）："别闹了。我们可以进去了吗？"
守卫："可以。欢迎来到清明上河。"
```

### 进城的瞬间

**异常发生**：
- 麦洛踏入城门的瞬间
- 感到一阵眩晕
- 眼前闪过一个画面：
  - 同样的城门
  - 但是千年前的样子
  - 没有全息投影
  - 只有木质的城门
  - 一个穿着古装的女人站在那里
  - 她转过头，看向麦洛
  - 眼神...悲伤

**画面消失**：
- 只有一瞬间
- 麦洛揉揉眼睛
- "刚才...是什么？"

**维塔的检测**：
```
维塔："你还好吗？"
麦洛："我好像...看到了什么..."
维塔："我的传感器没有检测到异常。"
麦洛："可能是我眼花了。"
维塔："你才14岁，不会眼花的。"
麦洛："那就是...幻觉？"
维塔："也许是清明上河的'欢迎特效'？"
```

**伏笔**：这是记忆的第一次泄露

---

## 虹桥市集（1200字）

### 繁华的展示

**视觉描写**（参考《银翼杀手2049》+《清明上河图》）：

**街道**：
- 宽阔的石板路
- 两旁是木质建筑
- 但建筑上有全息广告
- "最新灵器！九折优惠！"
- "灵机修行指导！包教包会！"
- "寻找组队伙伴！高手优先！"

**人流**：
- 玩家和NPC混杂
- 有人穿着古装
- 有人穿着现代服装
- 有人穿着科幻装甲
- 各种风格混搭

**虹桥**：
- 巨大的拱桥
- 横跨一条"数据河"
- 河水是流动的代码
- 桥上桥下都是店铺
- 桥中央有一个巨大的全息屏幕
- 播放着各种信息

**麦洛的反应**：
```
麦洛："这里...好热闹..."
维塔："这是清明上河的核心区域。虹桥市集。"
麦洛："为什么叫虹桥？"
维塔："因为桥的形状像彩虹。而且，传说这座桥连接着'过去'和'现在'。"
麦洛："什么意思？"
维塔："我也不知道。这是清明上河的七大谜团之一。"
```

### 第一次遇到"记忆NPC"

**场景**：
- 麦洛走在桥上
- 一个老人坐在桥边
- 他在画画
- 画的是...虹桥
- 但画中的虹桥是千年前的样子

**对话**：
```
麦洛（好奇）："老爷爷，你在画什么？"
老人（抬头）："虹桥。"
麦洛："但你画的...和现在的不一样。"
老人："因为我画的是...记忆中的虹桥。"
麦洛："记忆？"
老人："是啊。千年前的虹桥。"
维塔（扫描）："这是NPC。但他的对话逻辑...很奇怪。"
老人（看向维塔）："小猫，你说我奇怪？"
维塔："我不是猫！我是灵宠！"
老人（笑）："都一样。"
```

**老人的话**：
```
老人："小姑娘，你知道吗？这座城市...是活的。"
麦洛："活的？"
老人："是啊。它有记忆。它会记住每一个来过这里的人。"
麦洛："那...它记住我了吗？"
老人（看着她，眼神突然变得深邃）："它不仅记住了你...它还在等你。"
麦洛："等我？"
老人："是啊。等你...想起来。"
```

**老人消失**：
- 话音刚落
- 老人突然变成半透明
- 然后...消失了
- 只留下那幅画
- 画上的虹桥...在发光

**维塔的惊讶**：
```
维塔："他...他消失了？！"
麦洛："这是bug吗？"
维塔："不...我的数据库显示，他是'记忆NPC'。"
麦洛："记忆NPC？"
维塔："清明上河的特殊NPC。他们不是程序生成的，而是...记忆生成的。"
麦洛："什么意思？"
维塔："我也不太懂。但据说，这些NPC是Cycle-6文明的记忆残影。"
```

**伏笔**：记忆NPC的设定

## 苏瑾言登场（1000字）

### 相遇

**场景**：
- 麦洛捡起老人留下的画
- 画突然发出强光
- 光芒吸引了周围的人
- 其中一个女人快步走来

**苏瑾言的外貌**：
- 35岁左右
- 穿着考古学家的装束
- 卡其色夹克，工装裤，登山靴
- 但眼神...疲惫而坚定
- 像是找了很久很久的人

**对话**：
```
苏瑾言："这幅画...你从哪里得到的？"
麦洛："一个老人给我的。"
苏瑾言："老人？什么样的老人？"
麦洛："他在画虹桥...然后就消失了。"
苏瑾言（眼睛一亮）："消失了？那是记忆NPC！"
维塔："你知道记忆NPC？"
苏瑾言："当然。我是考古学家。我专门研究清明上河的记忆系统。"
```

**苏瑾言的请求**：
```
苏瑾言："这幅画...能给我看看吗？"
麦洛："可以。"
（苏瑾言接过画，仔细观察）
苏瑾言（颤抖）："这是...他的笔触..."
麦洛："他？"
苏瑾言："我的丈夫。陈默。他也是考古学家。"
维塔："也是？"
苏瑾言："他失踪了。三年前。在清明上河。"
```

**苏瑾言的故事**（简短版）：
```
苏瑾言："三年前，我和陈默一起来到清明上河。"
"我们在研究Cycle-6文明的记忆系统。"
"有一天，他说他发现了一个重要的线索。"
"然后...他就消失了。"
"我找了他三年。"
"我知道他还在这里。在某个地方。"
"因为...我能感觉到他。"
```

**麦洛的共鸣**：
```
麦洛："我理解你的感觉。"
苏瑾言："你也在找人？"
麦洛："我在找我爸爸。"
苏瑾言（看着她，眼神温柔）："那我们是同路人。"
维塔："所以...我们要一起找？"
苏瑾言："如果你们愿意的话。"
麦洛："我愿意。"
```

**组队**：
- 苏瑾言加入队伍
- 她的等级：灵根境（比麦洛高）
- 她的能力：记忆追踪
- 她的武器：考古铲（可以"挖掘"记忆）

## 第一个任务（500字）

### 苏瑾言的计划

```
苏瑾言："这幅画是线索。它指向一个地方。"
麦洛："什么地方？"
苏瑾言："长安遗址。"
维塔："长安？那不是在清明上河的西边吗？"
苏瑾言："是的。那里是Cycle-6文明的艺术中心。"
"如果陈默留下了什么...一定在那里。"
麦洛："那我们现在就去？"
苏瑾言："不。我们需要先准备。"
"长安遗址很危险。那里有...记忆怪物。"
维塔："记忆怪物？！"
苏瑾言："是的。被遗忘的记忆，会变成怪物。"
```

### 章节结尾

**场景**：
- 三人站在虹桥上
- 看着远方的夕阳
- 夕阳下，城市的轮廓若隐若现
- 像是两个时代重叠在一起

**苏瑾言的话**：
```
苏瑾言："清明上河...不只是一座城市。"
"它是一座墓碑。"
"埋葬着Cycle-6文明的所有记忆。"
"而我们...是盗墓者。"
```

**麦洛的决心**：
```
麦洛："那我们就把这些记忆...挖出来。"
苏瑾言（微笑）："对。挖出来。让它们重见天日。"
维塔："听起来很危险..."
麦洛和苏瑾言（同时）："但很值得。"
维塔："......你们两个真像。"
```

**最后一个画面**：
- 镜头拉远
- 虹桥上的三个身影
- 城市的灯光开始亮起
- 但在灯光之间
- 有一些...影子
- 那些影子...在动
- 它们是...记忆

**黑屏，第一章完**
