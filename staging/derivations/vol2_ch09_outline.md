**字数**: 4000字  
**场景**: 前往洛阳 → 地宫入口  
**核心**: 权力的诱惑 + 武则天记忆 + 第一道考验

## 开场：离开长安（500字）

**场景**：
- 五人离开长安艺术区
- 前往洛阳
- 路上，林墨一直看着剑歌
- 他在想林染

**林墨的变化**：
```
麦洛："你还好吗？"
林墨："我很好。比以前更好。"
"因为我知道...她还在。"
"在我的记忆里，在所有人的记忆里。"
苏瑾言："这就是记忆的力量。"
林墨："是的。我现在明白了。"
"我不是在为死者画画。"
"我是在为生者画画。"
"让生者记住死者。"
```

## 洛阳地宫入口（1000字）

### 场景描写

**洛阳城**：
- 比长安更宏伟
- 但也更阴森
- 城中央有一个巨大的地宫入口
- 入口是一个龙头
- 龙口张开，黑暗深邃

**守门石碑**：
```
"权力之路，通往地狱。"
"欲望之门，永不关闭。"
"进入者，需面对内心的黑暗。"
```

**维塔的警告**：
```
维塔："我的传感器检测到...这里的灵机很奇怪。"
麦洛："怎么奇怪？"
维塔："它在...诱惑。"
"像是在说：'来吧，我给你力量'。"
苏瑾言："这是权力的灵机。"
"Cycle-6的统治者，将他们的权力欲望封存在这里。"
"这些欲望...会诱惑进入者。"
```

### 进入地宫

**第一道门**：
- 门上有一个问题
- "你想要什么？"
- 需要回答才能进入

**五人的回答**：
```
麦洛："我想要...找到爸爸。"
（门开了一条缝）

苏瑾言："我想要...陈默回来。"
（门又开了一点）

林墨："我想要...染染永远被记住。"
（门继续开）

周行云："我想要...证明自己不是废物。"
（门几乎全开）

维塔："我想要...理解人类的情感。"
（门完全打开）
```

**进入**：
- 五人走进地宫
- 门在身后关闭
- 黑暗吞没了他们

## 第一道考验：权力的诱惑（1500字）

### 幻境开始

**场景变化**：
- 黑暗消失
- 五人发现自己在一个宫殿里
- 宫殿金碧辉煌
- 中央有一个宝座

**声音响起**：
```
"欢迎来到权力之殿。"
"这里，你们的愿望都能实现。"
"只要...你们愿意付出代价。"
```

### 麦洛的幻境

**幻境内容**：
- 麦洛看到父亲
- 船长站在宝座上
- 他说："女儿，来吧。"
- "只要你坐上这个宝座。"
- "你就能拥有一切。"
- "你就能找到所有答案。"

**诱惑**：
- 麦洛走向宝座
- 她的手触碰到宝座
- 感觉到...无尽的力量
- 她可以控制一切
- 她可以改变一切

**觉醒**：
```
麦洛（突然）："不...这不对..."
"爸爸不会这样说..."
"他不会让我用权力换答案..."
"他会让我自己找到答案..."
（推开宝座）
"我不要这个！"
```

**幻境破碎**：
- 宝座消失
- 父亲消失
- 麦洛回到现实

### 其他人的考验

**苏瑾言**：
- 幻境：陈默复活，但代价是她要成为统治者
- 选择：拒绝，"我要的是真实的他，不是权力换来的幻影"

**林墨**：
- 幻境：林染复活，但代价是他要控制所有人的记忆
- 选择：拒绝，"我不能为了一个人，牺牲所有人的自由"

**周行云**：
- 幻境：成为最强剑客，但代价是他要杀死所有比他强的人
- 选择：拒绝，"我要的是证明自己，不是踩着别人的尸体"

**维塔**：
- 幻境：成为真正的生命，但代价是失去对麦洛的记忆
- 选择：拒绝，"我宁愿是AI，也不要忘记她"

## 武则天的记忆（1000字）

### 守门人出现

**场景**：
- 五人通过考验
- 一个女人的身影出现
- 她穿着龙袍
- 眼神...威严而孤独

**武则天**：
```
"你们通过了第一道考验。"
"很好。"
"大多数人，都会被权力诱惑。"
"但你们...选择了拒绝。"
麦洛："你是...武则天？"
武则天："我是她的记忆。"
"真正的武则天，死了一千多年。"
"但她的权力欲望...还在。"
"封存在这个地宫里。"
```

### 武则天的故事

**简短回忆**：
```
武则天："你们知道权力的代价吗？"
苏瑾言："什么代价？"
武则天："孤独。"
"当你站在权力的顶峰..."
"你会发现...没有人能理解你。"
"没有人能信任你。"
"你只能...一个人。"
"永远的...一个人。"
```

**武则天的遗憾**：
```
武则天："我用一生追求权力。"
"我成为了唯一的女皇帝。"
"但我失去了...一切。"
"爱情、友情、亲情。"
"最后...连自己都失去了。"
"所以...不要像我一样。"
"不要为了权力，失去自己。"
```
