# 批量改进执行指南 - 立即可用

## 喜剧元素注入模板（复制粘贴到每章）

### 通用插入点

**每章开头后**：
```
维塔："[根据场景自定义吐槽]"
麦洛："[12岁少女回应]"
维塔："你的逻辑让我的处理器过载。"
```

**危险场景前**：
```
维塔："危险等级：[X]！建议：立即撤退！"
麦洛："那更有趣啊！"
维塔："有趣不等于安全！"
```

**战斗中**：
```
维塔："它比你大[X]倍！"
麦洛："那又怎样？"
维塔："......我为什么要跟着你？"
```

**情感场景前**：
```
维塔："检测到情绪波动。"
麦洛："我没有！"
维塔："你的心跳加速了37%。"
```

**章节结尾**：
```
维塔："下一站更危险。"
麦洛："太好了！"
维塔："那不是好事！"
```

## 语言风格调整模板

### 替换规则

**哲学表达 → 简单表达**：
- "觉悟" → "想通了"
- "本质" → "真正的意思"
- "存在" → "活着"
- "意义" → "为什么"

**增加语气词**：
- 句尾加"啊"、"哦"、"呢"
- "为什么" → "为什么啊"
- "我不知道" → "我不知道啊"

**增加重复**：
- "为什么" → "为什么为什么为什么"
- "不要" → "不要不要不要"

**增加情绪词**：
- "很" → "超级"
- "好" → "好酷"、"好厉害"
- "坏" → "完蛋了"

## 名场面设计模板

### 每章1个名场面（3-5分钟可视化高潮）

**视觉冲击型**：
- 大场面、壮观景色
- 特效、光影
- 慢镜头描写

**情感爆发型**：
- 催泪时刻
- 愤怒爆发
- 喜悦高潮

**角色高光型**：
- 每个角色的名场面
- 展示独特能力
- 性格闪光点

3. 复制粘贴对应模板
4. 微调使其符合场景
5. 保存
