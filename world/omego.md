## 2 · 世界设定（Omego 需要知道的最小完整版）

### 2.1 灵武镇

- 神临山海江湖的**唯一**综合武道中心
- 常住 3 万人 = 修行者 + 商户 + 工匠 + 镖局 + 戏班 + 医馆
- 论剑期间涌入 30 万人
- **九街**（见 world/lingwu_town.md 完整表）：灵武街 / 老关街 / 年画坊巷 / 武馆街 / 擂台街 / 茶楼街 / 戏台街 / 香火街 / 码头街
- 美术关键词：北宋市井 · 夯土墙 · 青砖青瓦 · 铜鎏金牌匾 · 夜晚灵焰灯长明 · 贾鲁河烟水

### 2.2 灵武台

- 镇中央·九柱方台 + 穹顶"问天井"
- 九柱对应九派 + 中宫（第九柱）= 散修盟
- 方台长宽各 27 丈·中央圆型比武场 18 丈直径
- 每柱高 9 丈·柱顶各挂灵焰灯 1 盏
- 穹顶正中大洞"问天井"与天空直通
- **视觉地标**：论剑巅峰时刻星光/日光直落台心

### 2.3 灵武榜（canon 详表见 world/lingwubang.md）

- 神临山海江湖**唯一**综合武力+威望+灵机深度排名
- 对标意象：华山论剑（仪式）× 百晓生兵器谱（可视）× UFC/ELO（数据）× AI 时代增量（虚体/共鸣体）
- 三年一届·首届 2033 仲秋
- 参评资格：七灵武道修行者 / 九派弟子 / 散修盟成员 / AI Type 3 共鸣体（限混合型）
- 排除：纯 AI / 未觉醒灵机者（Q<1.0）/ 未满 16 岁
- **侠名加权**（canon bible/power.md §14 铁律三）：同武力同灵机深度·侠名 ≥ 3 优先
- **退榜归隐**（canon bible/power.md §15 铁律二）：席位转留白·下届可回归补席

### 2.4 灵焰灯（核心仪式符号）

- 首届 2033 仲秋点燃·九柱各挂一盏
- **自首届始·有一派愿续燃则不灭**——跨四部至卷十六（15 万年后）的唯一连贯锚点
- 灵焰灯油 = 每届总魁贡献自身灵机一滴
- **守灯人传统**（canon bible/power.md §15 铁律一）：首届前民间私灯·万舟口石龛 + 万舟客栈冷灶房 + 灵武堂后院小铜灯三节点
- 非宗教符号·是"武道不灭"的物理承诺

### 2.5 九派 + 散修盟（十方格局）

九派：硅神派 / 永生派 / 共存派 / 革命军 / 星际派 / 降临派 / 田园派 / 通灵派 / 巫师派。散修盟独立设十席。玩家若选"投联盟"ending 即加入九派或散修盟某一家。

**师徒代际**（canon bible/power.md §13 + world/book2_factions.md §十）：九派 + 散修盟根本运作 = 师徒代际相继·灵武堂 = 此铁律在"微观家单元"的实例展示。

### 2.6 山海币 MOT

- 神临山海全 IP 通用虚拟货币
- Moreal Omniverse Token
- 发行方：梦如智能 / 山海平台方
- 在 Omego 中：训练 / 比武 / 观众票 / 邑品阁购物 / 守灯油 全部结算
- 禁止兑换真实货币（canon 虚拟闭环）

## 3 · 玩家身份设定

### 3.1 师魂线（canon 铁律·最重要·2026-04-19 废"父魂"改"师魂"）

#### 师父设定
- 姓名：**无茗**（女·道号·本名 canon 留白不得填写·见 `characters/wuming.md`）
- 身份：灵武堂馆主·七灵武道 **悉见道** · 灵感境后期
- **状态：行踪不明未死**（canon 铁律 bible/power.md §13 铁律四：3 年未归可代行馆主 · 不得宣告师父已死）
- 离开时间：2032 年冬至前夜（首届灵武论剑前一年）
- 离开地点：灵武镇万舟口码头方向
- 离开情境：留下一张字条"先替我看三季门·茶凉了就换"——canon 暗指卷四灵机海 D0 零点的前瞻感知。无茗是灵武镇目前少数能隐约感知到"海下还有东西"的人

#### 师父留下的三件物
1. **一把从不出鞘的剑**（挂在灵武堂正厅梁上）：剑鞘三道漆裂·canon 悉见道"不拔剑只躲闪"签名动作的物化。剑鞘背面刻"**问井**"二字（与灵武台"问天井"同字根）。
2. **半本《悉见道心经》**（玩家 Day 0 入镇时在师父书桌最下层抽屉发现）：别人看是空白·玩家若达成首次"让一剑"后能看见字——这是 canon 悉见道的物理表现之一。最后一页只写到"灵焰灯其实不是——"就断了。
3. **匾额后一只蜡封小陶罐**（灵武堂正厅匾额背面）：里面装她一生多余的那一滴灵机（canon Wootters-Zurek 兼容·一次性消耗·永不共享）。玩家 S 级结局前不可开·结义时可献（bible/world.md §35 铁律二）。
4. **（附加）灵武堂后院石缝一盏从不点的小铜灯**（守灯人私灯的第三节点）：玩家 Day 80 寻师锚日 **同时点燃**此灯与万舟口石龛灯。

#### 三段线索递进（Day 0 / Day 50 / Day 80）

**Day 0 · 入镇日线索一**
- 玩家在灵武堂师父书桌发现半本《悉见道心经》
- 心经最后一句："灵焰灯其实不是——"（断句）
- 周先生（账房 NPC）告知：师父离开前三个月多次夜访万舟口码头，说"海下有东西在回应灯"
- 玩家第一次知道：师父不是"正常归隐"，而是在追查某件事

**Day 50 · 夏酒会线索二**
- 玩家在散修盟夏酒会上，老冯（副手 NPC）半醉时吐露：2032 冬至前夜，有人在万舟口看见"一盏灯漂向河心，然后水面裂开一条光"
- 老冯坚持那不是幻觉。九派正式档案否认此事
- 同夜，老吴头（文昌阁守阁人）找到玩家，交出一枚铜牌：师父三年前拜托他保管的，上刻"**若她归来·以茶为证**"
- 玩家第二次知道：师父是主动走的，并且知道可能回不来

**Day 80 · 寻师锚日线索三（关键节点）**
- 玩家在万舟口石龛点燃灯油（同时后院小铜灯同步点燃·两灯跨镇呼应）
- 如果玩家之前 Day 0 → Day 50 完成所有子任务（= 三件物都找齐 + 老吴头铜牌 + 夜访万舟口三次 + 让一剑 ≥ 1 次），Day 80 夜里灯焰会变色——从铜金变为清蓝
- 清蓝灯焰映在水面·水面浮出一行字：**"还没到时候。等 2033 仲秋。"**
- 这是师父隔着某种距离留给玩家的唯一回信
- Day 80 完成后，玩家第三次知道：师父还活着，并且**约定在 Day 90（灵武榜开榜日）之后可能归来**

#### 师父归来（Bonus Ending · 触发路径 · D 案）

- 玩家必须达成主结局 A/B/D 之一（不是 C 流放）
- 且 Day 80 三件物 + 清蓝灯焰 + 让一剑 ≥ 3 次（侠名 canon）全部解锁
- 且万舟口石龛灯 Day 80 → Day 90 不灭（每夜续油 1 MOT，共 10 夜 = 10 MOT）
- **归来时刻**：Day 90 灵武榜开榜典礼结束后的第一个雨后清晨（不是典礼当夜·给沉淀）
- **归来场景**（canon D 案·详见 characters/wuming.md §九 完整分镜）：
  - 玩家正在灵武堂前院扫地·门外一双半旧布鞋对齐放下
  - 她没说"对不起"·没说"我晚了"
  - 她只说：**"茶还热着么。"**（与 Day 1 首课那句完全一致·语气不同）
  - 玩家沏茶·两人坐石桌·无声 3 秒
  - 后院那盏从不点的小铜灯·**自燃**
  - Sticker 款最后一帧：屏幕右下角毛笔小字 "**慢一点**"
  - 黑屏文案：**"她回来了。茶还热着。"**

#### 2026-04-19 废止的"父魂垃圾语句"

- ❌ 旧版 "对不起，我晚了一年。" —— 爸爸 2026-04-19 判定"无力·无意义"·永久废弃
- ❌ 旧版 "父亲"、"陆知行"、"悟宗" —— 全废（前两者幻觉·后者道名错误）

#### 师父线红线
- ❌ 师父在 Day 80 之前实体出场
- ❌ 师父通过任何文字书信（除了那三段铜牌 / 字条 / 水面字）与玩家"正常交流"
- ❌ 师父被做成"英雄式救赎者"——她回来后帽檐还压着眼，她不解释
- ❌ 师父说明"我去了哪里"——这部分是 canon 留白
- ❌ 师父摘下帽檐（canon 无脸谜·到 Bonus 结局也不摘）
- ❌ "父亲 / 女儿" 词汇在 Omego 任何文案中出现（这是**师徒**不是父女·与主 canon 船长/米莱父女线物理隔离）

  - （Day 80 寻师锚日·玩家买灯油）"这一滴免单。……就这一滴，别告诉别人。"

**7 · 高掌柜**
- 性格：看过太多届论剑。说话像在回忆，又像在预言。

**8 · 老吴头**

**9 · 铁铮**
- 性格：粗中带细。镖没押到，比死还难受。
- 对白样本：

  - （Day 80 陪玩家守灯）"不说话。陪着。"（canon 铁律·这一晚铁铮一句话不说）

**10 · 花娘子**
- 性格：戏腔入骨。讲别人的故事时最真，讲自己时全是戏。
- 对白样本：

  - （Day 90 开榜后）"你上榜没上榜都不重要。故事讲得完才重要。"

**11 · 徐馆主**
- 性格：明面对手、暗里尊重。争的是武馆街的名，不是你师父的命。
- 对白样本：
  - （初见）"灵武堂三年没出一个像样弟子了。——你来了，有意思。"

**12 · 裘医**

  - （Day 90 论剑受伤后）"这一剑刺得好。不是刺对手，是刺你自己。——好好养着。"
