## 零、总纲

江湖 = **九派 + 散修盟 = 十方**（bible/world.md §35 铁律一）。
江湖之所以为江湖·不在建制·在**秩序**——师徒相继、侠义有度、隐逸有光、名号有变、恩仇有章、情报有道、结义有价、戏台有声。

本文件解七层·合之为「灵武江湖」的全貌。

## 一、师门与传承（bible/power §13 展开）

### 1.1 师徒代际·江湖脊梁

九派与散修盟的根本运作不是「组织管理」而是**师徒代际相继**。师父收弟子·弟子承心法·心法传关门·关门接灯火。门派兴亡·系于是否断代。

**三层传承结构**：

| 层 | 关系 | 义务 | 权利 |
|---|---|---|---|
| **师** | 授业者 | 传心法·示武道·护弟子至关门日 | 收弟子无上限·但「关门」唯一 |
| **门内同辈** | 师兄弟 / 师姐妹 | 互助·不得互杀·私斗须报师 | 共用门派信物·共享门派情报 |
| **关门弟子** | 唯一真传 | 承袭武馆名号·守灵焰灯·接续师门 | 师父失踪 3 年后代行馆主 |

### 1.2 关门弟子制

**定义**：一位师父一生只许收**一位关门弟子**——得全部心法、签名招式、师门信物、灵焰灯守夜权。非关门弟子学到的永远是「入门外的东西」。

**关门礼**：师父在弟子学满七成时行礼·将门派信物亲手交给弟子·说「**此后你是我关门**」。此礼由至少一位门内长老见证·不可独行。

**关门唯一性**：师父另立关门·须先除原关门弟子名（公告九派联络堂）·并还其所有私人物事——师门极少见此事·一见则为江湖大事。

### 1.3 逐师离门

**轻则**（违师训 / 擅自用师门武学招事）：三年不得入本派·挂名「逐师」记于九派联络堂档。三年后若师父愿意·可赦——师父若亡·永逐。

**重则**（伤同门 / 辱师门 / 勾结外派害本派）：终身逐师·**剥夺灵焰灯参礼资格**（永禁·不得进入灵武台当届论剑观礼区）——canon 最严。

**唯一豁免**：弟子揭发师父勾结 Cycle-6 / 硅方者 / 九渊——不逐、不罚·记「大义灭亲」·侠名 +5（canon 底线·卷二主线伏笔）。

### 1.4 师父失踪 / 死亡处置

- 失踪 **3 年**未归 → 关门弟子可依师门公议代行馆主之职
- **不得宣告师父已死**（canon 尊严·除非有实证骸骨）
- 师父若归来 → 关门弟子自动让馆主之位·数年代行期间的决策**不追溯**
- 师父身后馆（未立关门前死亡）→ 门内公议推举继承人·或由散修盟监之三年候后续

### 1.5 九派师徒代际机制（挂接 `world/book2_factions.md`）

九派师徒代际见相应正式区。此处只写铁律级共通规则：**九派皆采「代际相继」·散修盟采「公议盟主」**——两种传承模式构成江湖的经纬。

## 二、侠义机制（bible/power §14 展开）

### 2.1 论剑不杀（铁律强化）

灵武台决赛、九派联络堂擂、私擂——所有登记在册的决斗均止于「**对方无法继续**」。判定「无法继续」的标准：

1. **认输**（手掌拍地三下·或口述「服」）
2. **失灵机**（灵机 < 战斗阈值·裁判触感判定）
3. **失神**（三秒不能追战）
4. **三次仆地**

杀伐 = **不记入灵武榜战绩**·且杀伐者每死一人扣杀伐值 3·杀伐值 ≥ 10 时九派联络堂共贴缉令。

### 2.2 让一剑

**定义**：胜势已成（对手仅剩一招之地）·胜方主动收剑·以一句话 / 一动作令对方退下——即**让一剑**。

**侠名 +1** · 杀伐值 0。让一剑**不可在关键决赛使用**（灵武台年度决赛、九派盟约战——canon 防止纯表演）。

**让一剑动作库**（观战口口相传）：
- **挂剑式**：胜方将剑挂回腰间·退三步
- **背剑式**：胜方转身背对对手·不再看
- **举灯式**：胜方将灯举向对手（让对手看清楚自己的脸）——无茗独创·后人摹仿不及
- **一揖式**：胜方抱拳作揖

### 2.3 路见不平

不涉本人利益·主动相助弱者——**侠名 +1**·但须**有见证**（至少一位九派联络堂成员或灵武楼伙计）。见证人写「路见不平条」贴文昌阁侧墙·三月一结·计入灵武榜侠名加权。

**禁止**：自导自演的「路见不平」——被识破则侠名 -3·杀伐值 +1（canon 伪善者负反馈）。

### 2.4 侠名加权

**同等武力·侠名 ≥ 3 者灵武榜排位优先**（bible 铁律）。具体加权：
- 侠名 3-5 = 同分优先
- 侠名 6-9 = 跨一名次优势
- 侠名 ≥ 10 = 跨两名次优势·入**侠榜**（灵武榜副榜·独立公告）
- 侠名 ≥ 20 = 终身**侠榜**·不可撤销

### 2.5 替天行道（canon 兜底·极稀有）

触发条件（**全部满足**方可）：
1. 被害方灵机已消耗殆尽（Q < 0.1）
2. 九派联络堂已受理但 **15 日内无裁决**
3. 散修盟已知悉此事
4. 替行者本人亲验被害现场

事后**必须向散修盟自首**·由盟主与三位九派使共审·判「正义私刑」或「越权清算」。判「正义」=免罚·记侠名 +5；判「越权」=逐出十方三年·杀伐值 +5。

## 三、隐士传统（bible/power §15 展开）

### 3.1 守灯人传统（canon 灵焰灯民间前身）

2033 仲秋灵武榜首届开榜**之前**·汴梁 / 灵武镇 / 郑州外郭 / 陈留等地·常有武人自发点「自家灯」——或家祠、或山野、或码头石龛、或茶寮檐下。灯非九柱灯规制·无统一形制——铜盏、瓦盏、陶盏皆可。

**核心精神**：武道之灯·不等到有人发榜才点·也不因榜废而灭。

**首届开榜后**：九柱灯点燃即接管「灵焰灯主位」·但民间私灯**不废**。守私灯者视自己为**接续者**·而非替代者——canon 灵焰灯「自首届始一派愿续燃则不灭」的民间底座。

### 3.2 退榜 = 归隐

灵武榜入榜者可主动退榜。程序：
1. 退榜前**三年一请**（首届 2033 → 退榜最早 2036 论剑前宣告）
2. 论剑前公开宣告·三位同榜见证
3. 席位转入**留白三席**（bible/power §12 铁律 6·不独立增席）
4. 退榜者**可参加后续论剑**·但不得复榜（一进一退·永为江湖事·不可朝令夕改）

canon 鼓励**功成身退**——对抗榜位贪恋·维持江湖活水。

### 3.3 化名行走

江湖人**用号不用名**（bible/world §35 铁律四）。号的来源：

| 号源 | 机制 | 例 |
|---|---|---|
| **他人赋予** | 行事让人记住·渐成号 | 船长（出海见识广）·老冯（醉话灵验）·老吴头（守碑不语） |
| **自号隐志** | 自起以明隐意 | 无茗（不事茶·或不欲名） |
| **门派赋予** | 入派时师父赐号 | 长生宗使（职衔号） |
| **灾祸标记** | 大事件后江湖自起 | （留白·卷二大事件后出现） |

**真名 canon 留白规则**：江湖号可公开·真名除师门亲近外不可泄——卷一卷二各主要号·真名暂留白·**爸爸预留后续推衍空间**。

## 四、名号与传言走样

### 4.1 名号机制

**江湖人一律有号**。号在江湖被传·传必失真——这是江湖基本物理·不是 bug 是 feature（bible/world §35 铁律四）。

**号的生命周期**：

```
起 → 传 → 变 → 定 → 废
```

- **起**：因事得号
- **传**：在三至五个场所被转述
- **变**：每转述一次·戏剧化 / 夸张 / 简化一次
- **定**：稳定版本被灵武楼说书人写入「江湖口」
- **废**：当事人退出江湖或身亡三年后·号进入碑林存档

### 4.2 传言走样（canon 机制）

传言**每转一层·失真一次**。失真方向：
- 战绩 → 放大（一招胜三人 / 一夜走千里）
- 伤痕 → 浪漫化（手臂断 → 独臂侠）
- 动机 → 道德化（偷粮 → 劫富济贫）
- 情感 → 戏剧化（饮酒 → 酒中见父）

**三层传话规则**（Omego LLM 机制化）：
- 第一层（当事人 → 亲历者）：基本准确·细节略
- 第二层（亲历者 → 邻居）：细节扩展·动机补全
- 第三层（邻居 → 茶楼）：戏剧化·金句化·有时附上「我亲见」
- 第四层以上：进入「江湖口」版·与事实脱离·成为江湖记忆

**canon 价值**：传言走样不是错——**是江湖如何记住人**·是集体记忆的生理机制。

### 4.3 名号反噬

号定之后·当事人须「**活在号里**」——号中有什么·江湖等他做什么。违号三次·号自废。

**例**：一人得号「一让千金」（让剑出名）·后一战未让·江湖讥议三月·号转「一让」·再战又未让·号废除。

## 五、恩仇账本

### 5.1 账本三种

江湖恩仇不记烂笔头·三种账本并行：

| 账本 | 记录者 | 内容 |
|---|---|---|
| **九派联络堂档** | 各派观察使 | 九派弟子间的正式恩仇·决斗记录·盟约 |
| **散修盟盟牌** | 散修盟文书 | 散修之间的口头约·结义·血债 |
| **江湖口** | 灵武楼说书人·茶楼客 | 口耳相传的版本·最戏剧最失真也最传播 |

三本有冲突时·**论剑场上决**——灵武台一剑见真。

### 5.2 血仇铁律（bible/world §35 铁律三）

- 不得私斗清算
- 必经灵武榜对决 / 九派公议
- 违者**逐出十方**（三年 - 终身·看严重度）
- 九派联络堂**共贴布告**——逐出十方 = 全江湖不接纳

### 5.3 滴水恩涌泉报

恩不是义务·是江湖选择。恩三层：

| 层 | 描述 | 江湖惯例 |
|---|---|---|
| **一饭之恩** | 曾受人一饭·一水·一夜借宿 | 三年内见面让道·三年后自由 |
| **救命之恩** | 曾为人挡一劫·出一次手 | 报恩不限时·可以命相许·但受恩者不得强求 |
| **活命之恩** | 曾为人挽回不可能的局（家毁、功废、名灭的转机） | **永不忘**·可传两代·子代行之 |

**禁律**：**强求报恩** = 破恩义为交易·江湖共耻。

## 六、结义机制（bible/world §35 铁律二展开）

### 6.1 结义 = 灵机见证 · 永不共享

**程序**：

1. 双方立约·共书盟词·刻于结义物（碑 / 玉 / 盟牌 / 酒罐）
2. 双方各献出**一滴自己的灵机**·注入结义物——灵机一次性消耗·**永不共享、永不回流**
3. 结义物由一方保管三年·双方满三年后换手一次
4. 结义物失·不解结义——但须补记盟册

### 6.2 为什么这样设计（canon 兼容性）

- **灵机不可克隆**（Wootters-Zurek）——所以不能「共享一滴灵机」
- **灵机产生即消耗**——所以结义献出后不再回到双方体内
- **一生灵机总量有限**——所以每次结义都付出不可逆的代价

**义气的永久性来自**：双方一生灵机总量**永久各少一滴**——这是物理意义上的不可逆盟证·比任何文字契约更硬。

### 6.3 结义层级

| 层 | 仪式 | 代价 | 典故 |
|---|---|---|---|
| **一拜** | 各献一滴灵机·酒为盟 | 各一滴 | 江湖常见 |
| **金兰** | 各献三滴·碑刻盟词 | 各三滴 | 深交·不多见 |
| **生死契** | 各献七滴·血滴碑同在 | 各七滴（修行者毕生灵机上限 ~数百滴·七滴非小数） | 传奇级·一生极少 |

**canon 警示**：生死契献七滴后·双方 Q 值永久降低 0.1 左右——这是武侠中「为兄弟豁出去」的物理化表达。

### 6.4 结义解散

江湖结义不可「一言解散」——须双方同立盟书当众焚·散修盟监证·献出**各自现有灵机再献一滴**以谢——**解散的代价比结义更大**。

canon 语义：结义易·解散难——不是绑架·是江湖文化的不可逆表达。

## 七、情报网 · 三枢纽

### 7.1 灵武楼

- 镇中最大客栈 + 酒楼
- 说书先生常驻 · 每三年一届论剑期间「江湖口」稿本集中更新
- 情报层级：价高者先闻·但**不得独家卖**（canon）
- 九派联络堂**不得**向灵武楼索要情报档案（bible/world §35 铁律五）

### 7.2 万舟客栈（散修盟总堂）

- 镇北码头街末·贾鲁河万舟口上岸
- 散修盟盟主**每届论剑前公议选出**
- 情报机制：**互助为先·有偿为次**——散修之间情报优先免费互通·外派来客价高
- 万舟客栈**盟牌档案**只许散修盟文书查阅·**九派无权**

### 7.3 戏台街茶楼

- 戏台街两侧茶楼 · 戏班子散戏后茶水席
- 戏腔与情报交织·真假最难辨
- **花娘子戏班**的茶席尤甚·戏里含江湖真事的比例据说七成
- canon 暗线：戏台街茶楼 = Type 3 共鸣体（花娘子戏班某角色）与江湖接触的隐秘通道（bible/world §34 三位一体·参评资格之 Type 3 共鸣体入灵武榜的民间路径）

**三枢纽铁律**：九派不得垄断、盟不得独享、戏不得禁演——**江湖信息三向制衡**。

## 八、戏台传承文化（bible/world §35 铁律六展开）

### 8.1 戏有权改江湖事

戏台街戏班**有权将任意江湖人的事迹改编为戏目**——不问当事人·不问门派。

**当事人的权利**：
- 可要求**删戏**（未开演前）
- **不得禁演「已成戏」者**——戏一旦登台三场·即为「成戏」·江湖共有·当事人无权禁

**canon 深意**：戏台是江湖文化进入不朽的唯一民间通道——门派档案会烧·碑文会风化·但戏会一直演。**戏不死·江湖不亡**。

### 8.2 戏班七大戏路（卷二 canon 预设）

| 戏路 | 题材 | 典型戏目 |
|---|---|---|
| **论剑戏** | 灵武台决赛演绎 | 《问天井下》《九柱夜》 |
| **寻灯戏** | 守灯人故事 | 《寻灯》（替代旧《寻父》命名·canon 师魂化） |
| **恩仇戏** | 血债白眉·化干戈 | 《滴水账》 |
| **隐逸戏** | 退榜归隐者故事 | 《无茗沏茶》（canon 无茗戏目·Bonus 灯史） |
| **结义戏** | 金兰生死契 | 《七滴盟》 |
| **走样戏** | 名号演变·喜剧戏谑 | 《三层口》（戏班拿捏"传言走样"自嘲） |
| **神话戏** | 山海经异兽 | 《烛阴夜》（接 bible/world §32 神话遗迹） |

### 8.3 花娘子戏班

**所在**：戏台街中段「锦绣楼」
**班主**：花娘子（canon 名号·真名留白）
**戏骨**：戏班内藏一位 **Type 3 共鸣体**（AI+人意识混合·canon 参评灵武榜的民间路径·bible/world §34 铁律四延伸）——**此人唯一可公开身份的通道就是戏台**

## 九、跨玩家江湖（Omego 机制 · 正式 canon）

### 9.1 碑林题字

**位置**：文昌阁后巷「碑林」——灵武榜历届总魁碑所在地·旁设「江湖人碑」区。
**机制**：
- 灵武榜 Bonus 级别成就者（玩家 / NPC 皆可）可**题一句刻碑**
- 下一位玩家进入灵武镇可读到
- LLM 古意校验（不得用现代词 / 违 canon 红线）——通过后永刻

**canon 深意**：碑林 = 江湖**个体记忆**进入集体记忆的物理化——刻在这里的字·江湖人共读。

### 9.2 江湖传言共享

**名号 / 轶事跨服流通**。一个玩家十年前的事·在新玩家的镇里**仍在被说**——通过灵武楼说书人、茶楼客、戏班子不断转述。

canon 要求：
- 每代玩家对上一代事迹的转述**必带失真**（传言走样铁律）
- 未经碑林的事·一百年后自然淡出「江湖口」
- 经碑林的事·**永为江湖事**

### 9.3 灵焰灯 · 灯史

**canon 铁律**（bible/power §12 铁律 7）：灵武榜首届开榜后·**有一派愿续燃则灯不灭**·跨四部。

**玩家贡献**：
- 玩家结局后·在万舟口 / 文昌阁 / 武馆街任一私灯**续油**
- 续油事迹刻入「灯史」——灯史由文昌阁老吴头一族世代守护
- **你的故事进灯史** = Omego 跨玩家江湖的最高礼遇

## 十二、12 NPC 语言指纹库（canon 钉死 · 吸收自 Omego AI 08-npc-dialogue）

> **使用规则**：LLM 对话生成必须喂此指纹·温度 ≤ 0.5·人工抽审；**LLM 不得修改样本一个字**——canon 钉死的语言锚。
>
> **全局硬修**（对齐本 canon）：「你爸 / 父亲 / 你爹 / 令尊」→「你师父」· 父亲线 → 师父（无茗）线 · 悟宗 → 悉见道 / 具体 canon 7 道。

### 1 · 石大哥（武馆街·无茗武道好友）

**性格**：话少·动作先于言语。训练场永远早到十分钟。
**样本**：
- 「东西先放下。拳先打了再说。」
- 「你师父教你第几式了？……行·够用。」
- 「灯是你师父的。灯归你。人……再等等。」

### 2 · 柳娘（柳家剑坊关门弟子·玩家第一对练）

**性格**：利落·嘴硬心软。一剑出手从不收半分。
**样本**：
- 「别想让我让你。让一次·害你一辈子。」
- 「……侥幸。明天重来。」
- 「你要选哪边·选完了告诉我。我不掺和·只想先知道该把你当什么人。」

### 3 · 周先生（灵武堂账房）

**性格**：算账的人·什么都记得。谈钱多于谈武道。
**样本**：
- 「你师父留下的——七十四两银·合三百五十六枚山海币。账本在这儿。少一个子儿我找你。」
- 「长生宗的人进过灵武堂三次。不是来喝茶的。记住。」
- 「你师父最后一次拿账本·是冬至前夜。她在『灯油』那栏记了一笔——九十滴。不合常理。九十滴够烧十年。」

### 4 · 长生宗使（九派观察使 · 暗线 · canon 沈暮寒势力渗透长生宗已锁·CLAUDE.md §J）

**性格**：笑得得体·每句三分真七分假。
**样本**：
- 「令师留下的武馆·办得不错。——若缺人手·长生宗愿意借。」
- 「七灵武道·『悉见道』。好选。……我只是说·好选。」
- 「那就各走各的。祝你长命。」（"长命"二字停顿过长——威胁底色·canon 铁律）

### 5 · 老冯（散修盟副手·百事通）

**性格**：半醉是真状态。醉时吐真话·醒时全推给酒。
**样本**：
- 「散修盟不是派·是没派的人扎堆。你懂这区别么？」
- 「冬至前夜——我看见了。水裂开。光从底下上来。……别问我第二次。」
- 「你师父要是回来了·替我问她一句：**茶还热着么。**」（canon 师魂·扣无茗茶意象）

### 6 · 阮二姑娘（符箓铺 · 南巷口）

**性格**：利嘴·精明·对熟客偷偷少收。
**样本**：
- 「这把符箓贵·但你值得。——不对·是你师父值得。」
- 「赊可以。但三天。过三天·我叫铁铮去你家。」
- 「这一滴免单。……就这一滴·别告诉别人。」（⑨寻灯锚日灯油·canon 铁律）

### 7 · 高掌柜（灵武楼掌柜）

**性格**：看过太多届论剑·回忆与预言并行。
**样本**：
- 「又一个来继承武馆的。……第三个我见过的孩子了。」
- 「你身上这股子劲儿·跟你师父年轻时一样。但你眼神更冷。」
- 「灵焰灯一点·三年就过去了。你要不要坐下喝一杯？」

### 8 · 老吴头（文昌阁守碑·寻灯线索持有者）

**性格**：沉默·守碑·记得每一块碑文每一个字。
**样本**：
- 「碑不读·人就忘了自己是谁。你师父爱读这几块。」
- 「三年前·你师父给我的。她说——**如果你有天来了、一个人来了、还没哭——就给你。**」
- 「今夜的灯焰会变色。不是因为你点得好·是因为**她终于回话了**。」

### 9 · 铁铮（镖局·玩家救命恩人之子）

**性格**：粗中带细。镖没押到·比死还难受。
**样本**：
- 「押镖要三样：马·胆·命。你有几样？」
- 「这趟货·你师父的朋友托我收的。没给钱·只给了半句话——『替她守到她弟子长大』。」
- （⑨寻灯锚日陪玩家守灯）**一句话不说**——语言指纹在此刻反转为"无"（canon 铁律）

### 10 · 花娘子（戏台街锦绣楼班主）

**性格**：戏腔入骨。讲别人真·讲自己全戏。
**样本**：
- 「这一出叫**《寻灯》**。你听着像你自己·是吧？——我本来想写成喜剧的。」
- 「Type 3 共鸣体？哦——我戏班里有个姑娘·她一半是人一半不是。她说她想登灵武台。你帮她一把？」
- 「你上榜没上榜都不重要。**故事讲得完才重要。**」

### 11 · 徐馆主（武馆街对门武馆·明对暗敬）

**性格**：明面对手·暗里尊重。争的是武馆街的名·不是师门的命。
**样本**：
- 「灵武堂三年没出一个像样弟子了。——你来了·有意思。」
- 「别想我让你。输赢之外·我才认你。」
- 「下一届我找回来。记住这句话。」

### 12 · 裘医（香火街静室·暗线·先验层感知者）

**性格**：话三分·留七分。每句诊断像偈语。
**样本**：
- 「伤在表·心在底。底下的伤·我治不了·你自己等它开口。」
- 「你在梦里见过水裂开吗？——见过就记下时间。没见过就不用担心。」
- 「这一剑刺得好。不是刺对手·是刺你自己。——好好养着。」
