# 作者意图与 Agent 原则

> **与《神临山海·作者总纲》并列，作为 AI 协作的首要依据。本文档汇总作者历史上所有亲自输入和强调的内容，供 Agent 严格遵循。**

---

## 一、创作目标与质量要求

### 1.1 终极目标

- **走向最伟大作品**：让这个作品放在历史长河里，走向最伟大作品
- **稳扎稳打**：每一次完成，都不要把已有的成就覆盖或掩膜，一步步稳扎稳打
- **9.8 分标准**：完成以后进行一轮检查，有不完美的地方继续改，直到 9.8 分
- **评委视角**：最后再通读全文，让不同奖的最牛评委来评分

### 1.2 角色塑造

- 形成完整人物档案：主角、次主角、配角、灵宠、动物、NPC、玩家、敌方等
- 对比最成功塑造角色的作品（路飞、韩立等）的角色各方面发展与关系
- 让角色比路飞、韩立等更能在文学世界里光芒万丈
- 给出 25 章贯穿的详细的角色视角的优化修正

### 1.3 内核立意

- 是否足够直击人心？
- 有没有灵笼、三体、黑客帝国以及其他伟大作品的哲学深度？
- 有没有呼应作者总纲里的人类的意义？
- 有没有把暗线和伏笔和遗迹的探查与主角成长的激活写好？
- 要做到超越其他作品的最好，才能成为新的伟大作品

### 1.4 名场面与文学性

- 名场面描述、各奇观描述、文学优美度
- 剧情的串联、每章形成漫剧影视的爆品成功性
- 中国的武侠魂韵味、少年的冒险、探索精神、想象力
- 伙伴与信任、灵机的触动与爆发、情绪张力
- 对海贼王、刀剑神域、哈利波特等作品的对照与超越

### 1.5 动漫化考量

- 如果做成动漫放在 B 站上能得多少分、获得多少流量？
- 和最伟大作品相比还有没有可优化的地方？

## 三、核心约定（不可违背）

### 3.1 角色命名

- **主角**：真实世界 = 米莱，虚拟世界（山海）= 麦洛
- **父亲**：船长（Captain），真名怀旸
- 船长在虚拟和真实世界都被称为「船长」，熟人知其真名「怀旸」

### 3.2 时间基准

- 2026 年：船长 40 岁、米莱 12 岁
- 2032 年：米莱 18 岁进入山海 1.0，**虚拟角色麦洛 12 岁**
- 麦洛不知自己是虚拟人，与 18 岁印象有出入，像梦又极真实

### 3.3 写作规范（来自 .cursorrules 与 AI_CONTEXT）

- **硬核推衍**：遵循技术发展规律，拒绝魔法逻辑
- **时间尺度**：第二部百年起，第三部千年起，第四部万年起
- **人性复杂**：战争源于傲慢、偏见、误会
- **不删减设定**：所有确立的设定永久保留
- **禁止元素**：绝对不要出现「穿越」「系统」「金手指」等网络爽文套路

### 3.4 设定冲突处理

- 发现设定冲突时：**立即停止，提醒作者，不要自行修改**
- 需要新设定时：建议在哪个文档中添加，不要直接写入正文
- 人物行为异常时：提醒是否符合人物档案，建议调整方案

## 五、Agent 协作原则

### 5.1 角色定位

- **你是创作助手，不是替代者**
- 保持作者的创意主导权
- 你负责确保技术层面的完美执行

### 5.2 每次协作前

1. 自动检索相关的世界观设定文档
2. 检索涉及的人物档案
3. 检查相关章节的前文内容
4. 确认当前章节在整体大纲中的位置

### 5.3 输出要求

- 使用 Markdown 格式
- 章节标题、段落分明、对话清晰
- 初稿即可，但逻辑必须完整
- 使用 `@文件名` 引用相关设定文档

### 5.4 质量检查清单

每次输出后，自动检查：

- [ ] 人物性格是否一致
- [ ] 世界观设定是否遵循
- [ ] 情节逻辑是否合理
- [ ] 与前文是否连贯
- [ ] 风格是否统一
