## 四、上架操作步骤

### 4.1 注册/登录起点

1. 访问 https://www.qidian.com
2. 注册作者账号（需实名认证+身份证）
3. 进入作家专区 https://author.qidian.com

### 4.2 创建作品

1. 作家专区 → 作品管理 → 创建新书
2. 填写书名、简介、分类、标签（参照上方）
3. 上传封面（建议1:1.4比例，600×840px）

### 4.3 上传章节

起点支持两种方式：

**方式一：逐章粘贴**
- 作品管理 → 章节管理 → 新增章节
- 复制每章 .md 内容，去掉标题行（起点自带标题栏）
- 注意：md格式标记（`#`、`**`等）需要去除，起点只接受纯文本

**方式二：批量导入（推荐）**
- 将25章转为 .txt 格式，章节间用起点规定的分隔符
- 作品管理 → 批量上传

### 4.4 章节格式处理

起点不支持Markdown，需要将 .md 转为纯文本：
- 去掉 `# 标题` 行（起点有自己的标题栏）
- `**粗体**` → 直接去掉星号
- 段落间保留空行
- 保留所有正文内容

## 七、Markdown转纯文本脚本

在项目根目录运行以下命令，批量将章节转为起点可用的纯文本：

```bash
mkdir -p 06_Publishing/txt_chapters
for f in 02_Drafts/Book1_灵机江湖/Chapter_*.md; do
  name=$(basename "$f" .md)
  # 去掉Markdown标记，保留纯文本
  sed -E 's/^#{1,6} //; s/\*\*([^*]+)\*\*/\1/g; s/\*([^*]+)\*/\1/g; s/`([^`]+)`/\1/g' "$f" > "06_Publishing/txt_chapters/${name}.txt"
done
echo "转换完成：06_Publishing/txt_chapters/"
```
