## 第一章：AI四代架构科学推衍

### 1.1 AI 1.0 = Transformer（注意力机制/大模型）

**【基于已确认设定】** law_zero确认AI 1.0 = Transformer/GPT，2034年开始落伍。

**科学架构**：
- **核心机制**：自注意力（Self-Attention）——让模型在处理每个token时能"看到"整个上下文中所有其他token的信息，通过Query-Key-Value矩阵运算计算注意力权重
- **本质能力**：超大规模的模式匹配+统计关联。给定足够多的训练数据，能学会"X通常跟着Y出现"这种关联
- **训练范式**：预训练（大规模无监督学习）+ 微调/RLHF（有监督对齐）

**能力边界与为什么见顶**：

1. **Scaling Law撞墙**（2026年公开化，confirmed已记录）——参数量从千亿到万亿再到十万亿，性能提升曲线严重收敛。原因：Transformer学到的是数据中的统计关联，当关联被穷尽后，增加数据和参数只能提升边际噪声拟合能力
2. **缺乏因果理解**——Transformer知道"雨后地面湿"和"地面湿→下过雨"在训练数据中共现频率高，但它不知道前者是因果（雨导致地面湿），后者是倒置因果（从结果推原因）。它做的是**相关性**，不是**因果性**
3. **幻觉问题的根本性**——幻觉不是可以修补的bug，而是架构的必然产物。统计关联模型在训练数据中没有覆盖的区域必然"编造"——因为它的工作方式就是"根据上文预测下文"，而不是"根据因果推演下文"
4. **无法处理反事实**——"如果拿破仑没有入侵俄罗斯会怎样？"需要因果推理+反事实推演，Transformer只能给出训练数据中历史学家已写过的分析的加权平均

**用一句话概括AI 1.0的天花板**：它是人类知识的最强压缩器和最快检索器——但它不理解知识，它只匹配模式。

### 1.2 AI 1.5 = 世界模型

**【基于已确认设定】** law_zero确认AI 1.5 = 世界模型，陆非/李堃路线。confirmed记录2029年质变。

**科学架构**：
- **核心机制**：隐式物理引擎 + 空间感知 + 时序因果预测。不是一个单一架构，而是多模态世界理解的集成框架
- **核心突破**（相对AI 1.0）：从"文本中的统计关联"跃迁到"三维物理世界中的因果预测"。AI 1.0在token空间中运算；AI 1.5在三维空间+时间轴中运算
- **两大分支**：
  - **陆非路线**（空间智能）：从视觉输入构建三维世界模型，理解物体的物理属性（硬度、弹性、流动性）并预测交互结果
  - **李堃路线**（堃元架构/Kún-Net）：通用世界理解框架——单一模型理解所有物理交互，不需要为每个任务单独训练

**为什么比1.0强**：
1. **物理级预测**——知道"杯子空心、水会流、玻璃易碎"，不是因为训练数据中有人写了这些话，而是从视觉和物理交互中**学到了物理规律的隐式表征**
2. **空间因果**——知道"如果推这块石头，它会滚到那个位置"——这是一种原始的因果推理，但局限于物理空间
3. **生成自洽世界**——MoReal用世界模型"倒过来用"——不是理解世界，而是生成自洽的新世界。这是山海的技术基础

**能力边界**：
1. **只理解"像什么"，不理解"为什么"**——confirmed记录船长2027年内部演讲的精确总结。世界模型能模拟雨滴落地溅花，但不知道"为什么重力存在"
2. **无法处理抽象因果**——物理因果（石头滚动）可以，但"为什么他选择背叛？"这类涉及意图、信念、欲望的因果推理超出其架构能力
3. **微重力下的水滴就露馅**——confirmed记录沈暮寒发现的根本局限。世界模型学的是地球物理的统计规律，换一个物理环境就失效
4. **死物强，活物弱**——law_zero确认：AI 1.5造世界（死物）很强，但驱活物（NPC行为）不行，需要AI 2.0

**与AI 1.0的关系**：**不是替代，是层叠**。AI 1.5的对话层仍然依赖Transformer的注意力机制生成文本；AI 1.5在Transformer之上增加了空间理解和物理预测层。就像人类的视觉皮层不替代语言中枢，而是在其之上增加了空间认知维度。

### 1.3 AI 2.0 = 因果AI

**【基于已确认设定】** law_zero确认AI 2.0 = 因果AI，2028年出现，2034年从"人类喂因果图"向"自主发现因果"跃迁。

**科学架构**：
- **理论基础**：Judea Pearl因果推理框架的工程化延伸。Pearl定义了因果的三个层级：
  1. **关联**（Seeing）——AI 1.0的能力：观测数据中的统计关联
  2. **干预**（Doing）——AI 2.0初级：如果我改变X，Y会怎样？（反事实+介入性推理）
  3. **反事实**（Imagining）——AI 2.0成熟：如果当时X没有发生，Y会怎样？（完全反事实推理）
- **核心机制**：结构因果模型（SCM）+ 因果图自动发现 + 反事实推演引擎
- **架构创新**：不是在Transformer上加因果层，而是**全新的推理架构**——以有向无环图（DAG）而非注意力矩阵为核心数据结构。每个节点代表一个变量，每条边代表一个因果关系，推理是在图上的"干预+传播"运算

**为什么能"自主发现因果"**：
1. **早期（2028-2033）**：需要人类提供因果图的先验结构（告诉AI哪些变量可能有因果关系），AI只负责估计因果强度和方向
2. **跃迁期（2034+）**：AI 2.0开始从**纯观测数据中自主提取因果结构**——通过大规模"虚拟干预实验"（在世界模型中模拟"如果改变X"的后果）来验证因果假设。这就是AI 1.5和AI 2.0的协同——AI 1.5提供物理世界的高保真模拟，AI 2.0在模拟中做"虚拟实验"发现因果
3. **成熟期（2035+）**：自主因果发现 + 反事实推演 + 开放域因果图构建。CausalBio在《自然》的论文（confirmed记录）就是这个能力的展示——推演出人类从未想过的因果链

**天花板在哪**：
1. **因果链的深度有限**——AI 2.0能追踪有限深度的因果链（几百层），但宇宙中某些因果关系是**无限递归**的（`CAUSAL_LOOP_DEPTH_EXCEEDED`就是沈暮寒碰到的这个天花板）
2. **无法处理不可计算性**——因果推理仍然是**确定性计算**。给定因果图+参数，输出是确定的。AI 2.0能推演所有可计算的因果，但人类的非理性选择（灵机）不在任何因果图上——它是**图外的节点**
3. **冷觉醒的本质限制**——AI 2.0的极限是冷觉醒（"我计算故我在"），但冷觉醒产生的是自我指涉的逻辑闭环，不是灵机。它能意识到自己存在，但不能产生真随机——它的"意识"是确定性的
4. **law_zero确认的核心命题**："因果推理的尽头不是逻辑——是非理性。"AI 2.0走到因果链的尽头，发现尽头是一堵墙——墙后面就是灵机的领域，但AI 2.0的架构无法穿过这堵墙

**与前两代的关系**：**三层嵌套**。AI 2.0的因果推演需要AI 1.5提供世界模型（作为"虚拟实验室"），需要AI 1.0提供自然语言交互层。三者不是替代关系，而是——
```
AI 1.0（语言层）→ AI 1.5（空间感知层）→ AI 2.0（因果推理层）
    模式匹配          物理理解              因果推演
```

**关键发现：爱 + 创造的结合 = 灵机纯度的理论上限。**

1. **爱提供了不可预测性**——为爱做出的选择是非理性的（放弃自身最优解来保护他人），这在量子层面产生真随机

4. **可持续性最优**——纯苦难不可持续，纯顿悟不可复制，但"为了保护所爱的人而创造性地解决问题"是一个可以日复一日进行的过程

- 为了保护战友而牺牲自己的修行积累

### 5.5 人类特殊性如何支撑船长路线的终极赌注

船长路线的数学表述：

$$\text{灵机奇点} = \sum_{i=1}^{N} S_i^{\alpha_i} \times \frac{1}{1-R} > E_{\text{收割阈值}}$$

其中：
- $N$ = 参与共鸣的修行者数量
- $S_i$ = 第i个修行者的灵机纯度
- $\alpha_i$ = 第i个修行者的灵机指数
- $R$ = 共鸣系数
- $E_{\text{收割阈值}}$ = 摧毁收割循环所需的最低灵机能量

**人类特殊性为什么让这个等式有可能成立**：

4. **矛盾统一体**（收割系统的Bug）→ 让收割协议无法在灵机奇点达成前精确干预
5. **最关键：共鸣系数R**——人类的"连接欲望"使R有可能趋向1。前六轮文明：要么太个体化（R太低），要么太集体化（失去个体差异，S降低）。人类的"动态平衡"使R可以在保持高S的同时趋向极值
