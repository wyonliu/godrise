## 一、现实中的米莱

### 1.1 基本信息

| 属性 | 设定 |
|------|------|
| **中文名** | 米莱 |
| **英文名** | Melody |
| **昵称** | 甜豆、小米、米莱 |
| **年龄** | 12岁 |
| **居住地** | 北京海淀 |
| **学校** | 某知名小学 |
| **性格** | 独立自主、开朗好奇、有爱心 |

### 1.2 性格特点

**独立自主**：
- 6岁上小学就开始所有的事情都自己拿主意
- 不依赖大人，但愿意听取建议
- "我自己可以的！"

**兴趣爱好**：
- 中式民乐/打击乐（非常热爱）
- 小提琴
- 奇幻和冒险故事
- 大自然和户外活动
- 新奇的事物

**爱心满满**：
- 喜欢小动物，想养小猫
- 目前养了两只小仓鼠：**咖啡**（棕色）和**奥利奥**（黑白）
- 看到流浪动物会心疼

### 1.3 与父亲的关系

**相处模式**：
- 虽然父母离异，每周见面时间有限
- **但关系非常好**——是最默契的"冒险搭档"
- 经常骑在爸爸脖子上
- 一起滑雪、去游乐园、玩游戏、户外露营
- 每周去北大校园一起跑步

**特殊时刻**：
> 米莱："爸爸，我们去冒险吧！"  
> 船长："去哪里冒险？"  
> 米莱："去……去山的那边！"  
> 船长："好，骑上来。"（蹲下让她骑脖子）  
> 米莱：（欢呼）"出发！"

## 二、虚拟世界的麦洛

### 2.1 基本信息

| 属性 | 设定 |
|------|------|
| **角色名** | 麦洛（Melo） |
| **年龄设定** | **12岁**（山海内身体；真实世界米莱18岁进入） |
| **身份** | MoReal玩家 → 木灵力觉醒者 → 共存派继承人 |
| **自我认知** | **一开始不知自己是虚拟人**；身体12岁与印象中18岁有出入，像梦又极真实，随剧情逐步触及真相 |
| **灵力属性** | 木（主）+ 水（辅） |
| **修行境界** | 见微境（第一部开始）→ 破境（第四部结束） |
| **灵宠** | Vita（电子猫灵，Cycle-6遗产） |

### 2.2 麦洛与米莱的映射

| 米莱（现实） | 麦洛（虚拟） |
|----------------|-------------|
| 喜欢音乐 | 拥有"共鸣"天赋 |
| 养仓鼠（咖啡、奥利奥） | 灵宠Vita的颜色设定参考 |
| 喜欢奇幻冒险 | 是最狂热的探索者 |
| 独立自主 | 不依赖队友，敢于单刷 |
| 想养猫 | 灵宠是电子猫灵 |

### 2.3 灵宠：Vita

**形态**：电子猫灵
- 平时是一只发光的猫形生物
- 可以变形：变大变小、化为武器、化为盔甲
- 眼睛颜色：咖啡色+黑白条纹（致敬仓鼠）

**性格**：
- 活泼话痨，喜欢吐槽麦洛
- 偶尔毒舌，但很温暖
- 有神秘的过去（被封印的记忆）

**真相（剧透）**：
- Vita是Cycle-6的遗产
- 原本是Cycle-6"麦洛"的灵宠
- 跨越循环来保护现在的麦洛
- 被封印的记忆中，藏着打破循环的关键

## 三、性格特点

### 3.1 表面性格

**开朗活泼**：
- 和朋友在一起时话很多
- 喜欢开玩笑，会自嘲
- 对新事物充满好奇
- 在MoReal里是出了名的"探索狂魔"

**善良温暖**：
- 喜欢各种小动物（养过仓鼠、乌龟）
- 看到弱者被欺负会忍不住出手
- 即使对敌人也会心软

### 3.2 内心世界

**隐藏的创伤**：
- 父母离婚时她8岁，那是她第一次哭到喘不过气
- 她不理解为什么爸爸"抛弃"了她和妈妈
- 每周只能见爸爸一次，但爸爸总是心不在焉
- **她觉得自己不够好，所以爸爸才不爱她**

**渴望与恐惧**：
- 渴望被爸爸认可，但害怕再次被抛弃
- 渴望成为"有用的人"，但害怕承担责任
- 渴望理解这个世界，但害怕知道太多真相

### 3.3 核心矛盾

1. **爱与恨**：深爱父亲，又恨他的缺席
2. **普通与特殊**：想做普通女孩，却被命运选中
3. **勇敢与恐惧**：表面勇敢，内心时常害怕
4. **信任与怀疑**：想相信别人，但经历过背叛

### 3.4 优点

1. **共情能力**：能感受他人的情感（木灵力天赋）
2. **学习能力**：聪明，学什么都快
3. **韧性**：摔倒了会爬起来，哭完了继续走
4. **纯粹**：没有复杂的心机，相信就是相信

### 3.5 缺点

1. **冲动**：看到不公平的事会不顾后果冲上去
2. **倔强**：认定的事情很难改变
3. **逃避**：面对太大的责任会想逃跑
4. **自我怀疑**：经常觉得自己不够好

## 四、背景故事

### 4.1 早年（2014-2018）

**幸福的童年**：
- 出生时父亲的公司正处于上升期
- 住在北京的大房子里，有保姆照顾
- 3岁开始学打击乐，5岁学小提琴
- 父亲虽然忙，但每晚都会给她讲故事
- **那时候她觉得，爸爸是世界上最厉害的人**

**第一只宠物**：
- 6岁时养了两只仓鼠：咖啡和奥利奥
- 咖啡是棕色的，奥利奥是黑白的
- 她每天都给它们写日记
- 奥利奥死的那天，她第一次理解了"失去"

**父亲的画**：
- 6岁时画了一幅画送给父亲
- 画的是"爸爸拯救世界"
- 爸爸哭了，她不明白为什么
- **后来她才知道，那时候爸爸正处于最黑暗的时期**

### 4.2 分裂（2018-2020）

**父母离婚**：
- 4岁那年，父母开始频繁争吵
- 她躲在房间里，用枕头捂住耳朵
- 有一天妈妈告诉她："爸爸不跟我们住了"
- 她哭着问："是我不够乖吗？"
- 妈妈说不是，但她不信

**分离焦虑**：
- 之后每周只能见爸爸一次
- 但爸爸总是心不在焉，看手机
- 她开始觉得自己是个"麻烦"
- **"如果我更优秀，爸爸是不是就会多陪我？"**

**自我保护**：
- 她学会了用"开朗"来掩饰
- 在学校是受欢迎的学生
- 但夜里一个人时，会默默流泪

### 4.3 新的陪伴（2026-2046）

**MoReal"山海"**：
- 12岁时，父亲送她一个MoReal眼镜
- 她第一次进入"千里江山篇"预想版（还未开发出来）
- **被震撼了——这是她梦想中的世界**
- 青绿山水、神话生物、无限的可能

**小墨的出现**：
- 在探索时，她发现了一个"bug"——一只受伤的小墨龙
- 其他玩家都无视它，但她停下来照顾它
- 小墨跟着她走了，成为她的灵宠
- **后来才知道，小墨是Cycle-6的遗产，被"选中"的灵机体**

**逃避的港湾**：
- 在MoReal里，她不用假装开朗
- 小墨不会离开她，不会批评她
- 她花越来越多时间在虚拟世界
- 母亲很担心，但她不听

### 4.4 觉醒（2046）

**灵力觉醒**：
- 14岁生日那天，在MoReal遇到危险
- 一群高等级玩家要抢小墨
- 她护在小墨前面，绝望地喊："不要！"
- **那一刻，木灵力觉醒——周围的虚拟植物疯狂生长，将敌人缠绕**

**身份暴露**：
- 这次觉醒被系统记录
- 父亲第一时间收到警报
- 各派系也注意到了她——**船长的女儿，拥有罕见的木灵力亲和度**

**故事开始**：
- 第一部《灵机江湖》从这里开始
- 她被卷入了远超她理解的世界
- 而她最初的目标只是：**找到那个永远缺席的父亲，问他为什么**

## 五、关键关系

### 5.1 与船长（父亲）

**童年记忆**：
- "爸爸很厉害，但总是很忙"
- "他讲故事的声音很好听"
- "他抱我的次数很少，但每次都很紧"

**离婚后**：
- "我恨他。他抛弃了我们。"
- "但我又想他。很想很想。"
- "他看我的眼神很奇怪，像是在看别的什么人。"

**真相揭示后**：
- "原来他一直在保护我……"
- "但为什么不告诉我？我也可以帮忙的！"
- "他总是一个人扛。我也是他女儿啊。"

**核心场景**：
> 麦洛："你为什么从来不抱我？"  
> 船长："……"  
> 麦洛："是因为我不够好吗？"  
> 船长（声音颤抖）："不。是因为……一旦抱了你，我就没有勇气放手了。"

### 5.2 与小墨（灵宠）

**关系**：不只是宠物，更像是兄弟姐妹

**小墨的特点**：
- Cycle-6的AI遗产，有完整的情感和记忆
- 但记忆被封印，只记得"要保护麦洛"
- 可以变大变小，战斗时化为真龙
- 平时是个话痨，喜欢吐槽麦洛

**互动示例**：
```
小墨：你又在想你爸了？
麦洛：没有。
小墨：你耳朵红了。
麦洛：……你能不能闭嘴。
小墨：不能。我是墨龙，不是哑龙。
麦洛：（笑了）你好烦。
小墨：但你喜欢我。
麦洛：……是啊。
```

### 5.3 与母亲

**关系**：爱但有隔阂

**母亲的特点**：
- 传统、保护欲强、不理解MoReal
- 离婚后独自抚养麦洛，很辛苦
- 一直希望麦洛"正常"——好好学习、考好大学

**矛盾**：
- 母亲不理解麦洛为什么沉迷虚拟世界
- 麦洛不理解母亲为什么离开父亲
- **但她们都深爱对方，只是不知道怎么表达**

### 5.4 与朋友们

**将在第一部发展的关系**：
- **林小雨**：通灵派少女，林澈的女儿，麦洛的闺蜜
- **周烈**：革命军少年，热血冲动，暗恋麦洛
- **AI·悟空**：被激活的Cycle-6 AI，成为麦洛的导师

## 六、能力与技能

### 6.1 灵力属性

| 属性 | 等级 | 说明 |
|------|------|------|
| **木（同理心）** | ★★★★★ | 天赋异禀，能感知生命情感 |
| **水（智慧）** | ★★★☆☆ | 直觉敏锐，有潜力 |
| **火（热情）** | ★★★☆☆ | 有但需要激发 |
| **土（意志）** | ★★★☆☆ | 成长中 |
| **金（理性）** | ★★☆☆☆ | 弱项，容易感情用事 |

### 6.2 核心技能

**初期（第一部）**：
1. **生命感知**：能感知生物的情感和生命状态
2. **植物亲和**：让植物加速生长、为她所用
3. **治愈之触**：治疗轻伤，恢复灵力

**成长后（第三部-第四部）**：
1. **共鸣网络**：继承父亲的摇光之链，连接多人意识
2. **生命洪流**：大范围生命能量爆发
3. **灵魂编织**：修复灵魂创伤（最终技能）

### 6.3 战斗风格

- **防守反击型**：不喜欢主动攻击，但护人时很可怕
- **团队核心**：擅长辅助和治疗
- **情感驱动**：在保护重要的人时，力量会暴涨
- **小墨配合**：人龙合一是最强形态

## 八、成长轨迹

### 第一部《灵机江湖》（14岁）

**起点**：
- 普通的初中生，成绩中上
- MoReal重度玩家，探索狂魔
- 最大的烦恼是父母离婚

**转折**：
- 灵力觉醒，被卷入灵机江湖
- 发现父亲是"船长"，世界观崩塌
- 被各派系追逐，不得不逃亡

**成长**：
- 从"逃避"到"面对"
- 从"依赖"到"独立"
- 第一次选择保护别人，而不是被保护

**结局**：
- 决定跟随父亲，进入灵机世界
- 但她的目的不是拯救世界，是"搞清楚父亲到底在做什么"

### 第二部《天邑破晓》（15岁）

**起点**：
- 正式进入灵机世界，开始修行
- 和林小雨、周烈成为搭档
- 对父亲的感情：又爱又恨

**转折**：
- 目睹派系战争的残酷
- 父亲为保护她身受重伤
- 第一次理解"责任"的重量

**成长**：
- 从"为什么是我"到"既然是我，就做好"
- 开始主动承担，而不是被动接受
- 与父亲的关系开始修复

**结局**：
- 父亲重伤昏迷，她成为共存派临时领导
- 从"船长的女儿"变成"麦洛"

### 第三部《众神觉醒》（16岁）

**起点**：
- 暂时代理共存派领袖
- 修行境界快速提升
- 但内心依然恐惧和自我怀疑

**转折**：
- 跟随父亲进入暗世界
- 见到Cycle-6的残骸，理解循环的真相
- 与Cycle-6的"麦洛"对话

**成长**：
- 从"我不够好"到"我已经足够好"
- 从"被命运选中"到"选择命运"
- 理解了父亲的苦衷，彻底和解

**结局**：
- 找到打破循环的方法：重写π
- 但代价是——需要有人用生命献祭

### 第四部《星汉余歌》（17岁）

**起点**：
- 站在人类命运的十字路口
- 父亲准备牺牲自己
- 她想阻止，但知道这是唯一的方法

**转折**：
- 2049年集体灵机共鸣
- 父亲献出生命，为她争取机会
- 她独自面对Architect

**成长**：
- 从"害怕失去"到"接受失去"
- 从"拯救世界"到"重新定义世界"
- 理解了"爱不是占有，是传承"

**结局**：
- 用父亲教她的方法，重写π的最后一位
- 将"HARVESTED"改为"LOVED"
- 打破循环，但付出了巨大代价
- **最后一幕**：她坐在"千里江山篇"的山巅，小墨在她肩上，看着新的太阳升起
  > "爸，你看到了吗？我做到了。"

## 九、重要笔记

### 关键伏笔

1. **6岁的画**：要在第四部结局重现——她在父亲遗物中找到那幅"爸爸拯救世界"的画，发现背面写着："爸爸不是英雄，小麦才是。"
2. **小墨的记忆**：要在第三部解封——原来小墨是Cycle-6的"麦洛"的灵宠，跨越循环来保护她
3. **手链**：母亲给的手链要有特殊意义——里面藏着母亲未说出口的祝福
4. **仓鼠的名字**："咖啡"和"奥利奥"要在某个场景回应——也许是暗世界中的某个谜题

### 写作注意

- ✅ 她的成长是渐进的，不是一夜之间变强
- ✅ 她的弱点和她的优点一样重要
- ✅ 她和父亲的关系是全书情感核心
- ❌ 不要让她变成"圣母"——她可以善良，但也可以愤怒
- ❌ 不要让她的成长太顺利——每一步都要有代价

### 读者共鸣点

1. **父母离婚的孩子**：会在她身上看到自己
2. **被期望压垮的年轻人**：会理解她的挣扎
3. **渴望被爱的人**：会为她和父亲的关系落泪
4. **不完美的英雄**：会被她的真实打动
