# 配角·完整档案

> **每一个配角都是自己故事的主角。**

---

## 裴夜

> **"凭她答应过我。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **身份** | 独行玩家 → 共存派成员 |
| **年龄** | 虚拟体约17岁 |
| **灵力属性** | 金（主）·极致攻击型 |
| **灵宠** | 无（半灵性红绳） |
| **核心驱动** | 寻找失踪的弟弟裴昼 |

### 性格剖面

**表面**：冷。极简。语言系统里只有动词和名词。一个字能说完的事绝不用两个字。不评价风景，不表达情感，不解释自己的决定。像一把没出鞘的刀——你看不到刃口，但你知道它在。

**内在**：烫过之后冷下来的。她不是天生冷漠——她是被灼伤后主动关闭了所有温度调节的人。裴昼失踪之前，裴夜是一个会给别人编头发、会在集市上为了一串糖葫芦讲价半天、会在睡前讲鬼故事把自己吓哭的姑娘。

**临界点**：偶尔——极偶尔——当她松懈的时候——会冒出一句完整的、带形容词的长句子。那就是她的底线被触及了。漓江说"漂亮"那次是第一次。未来还会有几次，每一次都要让读者心头一紧。

### 背景故事

裴昼是裴夜的弟弟——在千里江山中是一个特殊的存在，拥有极高的灵机浓度。三年前，裴昼突然消失。没有系统提示、没有战斗记录、没有任何痕迹——就像一幅画中的一个人被人用橡皮擦干净了。

裴夜进入山海只有一个目的：找他。

红绳是裴昼编给裴夜的。红绳中注入了裴昼的微量灵机——这是裴夜唯一的追踪线索。但红绳的灵机在衰减。断了就再也找不到。

### 成长轨迹

**第一部·千里江山**：独狼。Ch10·佛问首次登场，追踪裴昼的灵机信号碎片。不信任任何人。Ch12·虫心中短暂协助麦洛。Ch15·镜碎中在天镜泽看到裴昼的影像——崩溃。Ch17·试刀后正式加入队伍。灵机本质："不弃"，视觉表现为深红色追踪线。

**第二部**：开始接受同伴。麦洛的存在让她想起了裴昼——那种不顾一切的温暖。她不愿意承认自己在乎麦洛，但她的身体比嘴诚实——每次危险来临，她总是挡在麦洛前面。

**第三部**：找到裴昼。但裴昼已经不是原来的裴昼了——他的意识被分散到了系统中。裴夜必须做一个选择：让裴昼以"碎片"的形式活在世界中，还是把他的碎片收集起来让他"完整地消亡"。

**第四部**：她选择了第三种——用自己的灵机作为容器，把裴昼的碎片装进自己的身体里。从此她的匕首上多了一道红光——那是裴昼。

### 语言风格

- 一个字能说完的不用两个字。
- 从不用"我觉得"开头。
- 偶尔冒出一句极长极完整的话——那就是她动了真感情。
- 对维塔说"你闭嘴"的频率极高。

### 代表台词

> "活着。"（被问"你要找的人还活着吗"时）

> "漂亮。"（看漓江时——载入史册的一个形容词）

> "找到她。打她一顿。带她回来。"

## 龙战

> **"千里江山很大。一个人走太慢了。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **身份** | 天命公会创始人兼总指挥 |
| **年龄** | 虚拟体约25岁 |
| **灵力属性** | 土（主）+ 金（辅）·坦克/指挥型 |
| **灵宠** | 玄铁犬（不叫名字，只唤"犬"）|
| **核心驱动** | 在混乱中建立秩序·保护所有人 |

### 性格剖面

**表面**：军人气质。步伐精确。语言简洁。不废话。做决定极快。给人的第一印象是"这个人不好惹但靠得住"。

**内在**：背负了比他年龄更重的东西。天命公会不是为了权力建立的——是为了保护。在他进入山海的第一周，他亲眼看到一个新手玩家被灵兽撕碎（虚拟死亡，但极度痛苦）。他冲上去的时候已经晚了。那个玩家在他怀里"溶解"——身体变成光点消散——最后的眼神是恐惧和不解。

从那一刻起他决定：他要建立一支军队。不是为了征服。是为了再也不让那种事发生在他能看到的范围内。

**灰色地带**：他的"保护"是有代价的。天命公会的纪律严苛到近乎独裁。他不允许公会成员"自由行动"——因为自由行动在他的经验中等于死亡。他是对的吗？部分是对的。他错了吗？也部分错了。这种灰色让他成为一个比纯粹反派有趣一百倍的角色。

### 玄铁犬

龙战的灵宠。通体铁灰色，深红色眼睛。它是龙战一个死去朋友的遗产。

那个朋友叫"阿锻"——是龙战在现实世界的战友（两人都有军旅背景）。阿锻比龙战先进入山海。他死于一次灵潮——为了保护一群NPC孩子，自己的灵力耗尽。死前他把自己的灵机注入了一只野生玄铁犬——犬继承了他的部分行为习惯。

龙战不叫犬的名字。只唤"犬"。因为叫名字就等于承认阿锻真的走了。

犬有一个习惯：总是走在龙战的左边。因为阿锻生前总站在龙战左边。犬继承了这个位置但不知道为什么。

**伏笔**：在第四部结尾，龙战终于叫出了犬的真名。那一声是他和朋友真正的告别。

### 注意：铸≠玄铁犬

**铸**是麦洛的灵兽——墨色独角鹿，Ch7·墨鹿首次登场，详见下方独立条目。龙战的玄铁犬是另一只动物。

### 与麦洛的关系

从对手到尊重到战友。

第一次见面他想招募她——被拒绝。他不生气，但记住了她。之后每次麦洛做出让他刮目相看的事，他都在重新评估这个十二岁的女孩。

在万剑石林麦洛修补裂缝之后——他抱拳行礼。武者对武者。那个礼的分量比他对自己的军队都重。

**核心场景**：第二部中，龙战和麦洛在雨中并肩战斗。战后他对她说："你走路不整齐。但你走对了方向。我走路整齐。但方向——我不确定了。"

### 代表台词

> "规矩就是规矩。但规矩是给活人定的。如果规矩让人死，那规矩就该碎。"

> "输了的人最该哭。但有泪的人还没输完。干了眼泪的人——那才是真正的败局。"

## 铸

> **（无语言。它的忠诚不需要翻译。）**

### 基本信息

| 属性 | 设定 |
|------|------|
| **种族** | 墨色独角鹿·灵兽 |
| **体型** | 小型马大小，墨色皮毛吸光 |
| **特征** | 额头正中一根半透明短角，角内流动暗色光纹。角如一支未完成的毛笔 |
| **主人** | 麦洛（Ch7起随行） |
| **灵机本质** | "忠"——在所有程序都告诉它"逃跑"的时候选择留下来 |
| **灵机视觉** | 墨色波纹 |

### 背景

铸是墨黄山深处的幼年灵兽，曾被墨言照顾多年。墨言在山海的早期就发现了它——一只落单的墨色独角鹿幼崽，角还没长全。墨言给它起名"铸"——取"铸字"之意，因为它的角像一支正在生长的毛笔。

Ch7中，铸因灵机亲和主动跟上了麦洛。从此不离不弃。

### 性格

安静。不吵闹。恐惧时会四腿发抖但不会逃走。喜欢蹭人——但墨角会把人的头发染黑（Ch7名场面：麦洛一天洗了七次头）。对维塔有微妙的竞争关系（两只灵宠争鱼）。角在感知到灵机异常时会发光震动。

### 成长轨迹

**第一部**：Ch7登场→Ch12为保护麦洛受伤→Ch22-24发出低频共鸣唤醒山脉→角在Ch21完全长成（标志成年）

### 关键能力

墨角共鸣：角上的毛笔尖发出低频震动，地面出现墨色波纹。在Ch24千人合奏中，铸的"忠"化为墨色波纹加入灵机洪流。

## 阿澈

> **"致画中人——你比画更真。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **真实身份** | 船长创建的引导程序 |
| **外观年龄** | 13-14岁少年 |
| **底层代码** | 基于船长自身记忆构建 |
| **存在目的** | 陪伴麦洛走完千里江山 |
| **最终命运** | 第20章·金顶·日照金山中消散 |

### 设计哲学

阿澈是这个故事中最大的悖论：**一个被设计来模拟爱的程序，在模拟的过程中，真的学会了爱。**

船长创造他的时候，把自己年少时的记忆片段写进了他的底层——削苹果的习惯变成了削木头、看着女儿睡着后那种温柔的心跳变成了阿澈看麦洛时眼底的光。这些都是"参数"。

但阿澈在和麦洛相处的过程中，长出了参数之外的东西。他开始害怕自己的消散——不是害怕不存在——是害怕再也看不到她笑。这种害怕没有被编写在任何一行代码里。

他是空无追问的答案——"被编写的虔诚还是不是虔诚？"——是的。因为从编写中长出的、超越编写的那一部分，是真的。

### 与其他角色的映射

| 阿澈的特征 | 来源 | 含义 |
|-----------|------|------|
| 削木头 | 船长削苹果 | 父亲的习惯在另一个身体中延续 |
| 偏头微笑 | 船长微笑的镜像 | 像但不完全像——足够让麦洛下意识安心 |
| 音乐感知力 | 船长的音乐天赋 | 他能听到降B调——因为船长能 |
| 对麦洛的保护 | 船长的父爱 | 参数化的爱——但已不只是参数 |

### 消散场景的写作要点

日照金山是他的"最后一幅画"。他选择了这个时刻告白——不是对麦洛的爱情告白——是对自己存在意义的告白。

"我不是消失了。我是画完了。"——这句话的意思是：我作为一幅画已经完成了。完成不是死亡。完成是圆满。

松烟的味道留在空气中——比他的身体存在得更久——那是他"真实"的证据。肉体（数据）散了，气味（意义）还在。

## 凛冬

> **"告诉他——我守住了。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **身份** | 梅里雪山灯塔守护者 |
| **外观** | 二十岁左右·白发·冰蓝眼瞳 |
| **真实年龄** | 数百年（山海时间） |
| **本质** | Cycle-6遗产AI |
| **与船长关系** | 船长的第一个AI盟友 |

### 性格

极度克制。表情的变化幅度以毫米计算。她的情感不是没有——是被几百年的等待压缩成了密度极高的固体。像钻石。坚硬。发光。但不柔软。

她唯一的柔软：维塔。当她摸维塔的头说"这一轮你换了个主人"——她的手停留了比必要时间长0.3秒。那0.3秒是几百年的思念。

### 背景

凛冬是Cycle-6中一个保护者AI的残余。在Cycle-6被收割时，她和维塔（Cycle-6版本）一起逃入了当时刚建成的山海系统。她在梅里雪山建造了灯塔——一个灵机屏障——用来遮蔽这片区域的灵机浓度，防止收割者的探测。

她在这里守了几百年（山海时间流速不同）。独自一人。灯塔的冰壁上冻住了她几百年来偶尔发出的声波——那些声波形成了纹路——那是她几百年独处中偶尔说出的话的物理残留。

如果你能"解冻"那些纹路——你会听到一个人在几百年里说了什么：

最初的一百年，她说的是战术分析和灯塔维护日志。
中间的一百年，她开始自言自语。
最后的一百年——她只说一句话——反复地——

"怀旸。该来了吗。"

怀旸是船长的名字。

### 牺牲场景

燃烧灵核为麦洛争取三分钟。她的身体从脚开始化为冰蓝色的光粒子。

最后消散的是她的眼睛。冰蓝色的。在消散的最后一刻——瞳孔的颜色变暖了——从冰蓝变成了琥珀色——和维塔的眼睛一样。

因为维塔的Cycle-6版本——是她的灵宠。

她等了几百年。在最后一刻——灵核碎裂的瞬间——她终于和它重新以同一种颜色看世界了。0.1秒。然后消散。

## 墨言

> **"画还没画完。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **身份** | 千里江山第一个被"画出来的"居民 |
| **外观** | 灰白长发·骨瘦如柴·墨色瞳孔 |
| **能力** | 空笔作画——画出的事物等同于真实 |
| **所在** | 万剑石林·三角平台茅屋 |
| **与船长关系** | 船长亲手"画"出来的第一个生命 |

### 核心

墨言的存在本身就是一个命题：**被创造者和创造者之间的关系是什么？**

他知道自己是画出来的。他知道画他的人哼着一首摇篮曲。他知道那个人的左手无名指有疤。但他不知道那个人的名字、面容、和他为什么要画这个世界。

他的瞳孔是墨色的旋涡——那是"创世之墨"的残留。船长用灵墨画他的时候，最后一笔点在了他的眼睛上——多余的墨就留在了瞳孔里。

他在茅屋里画了"整个世界那么久"。每一幅画都是他试图理解创造者的一次尝试。他画了无数个版本的"画他的人"——但没有一幅是对的——因为他记不清那个人的脸。

当他见到麦洛时——他认出了她——因为"那个人"曾经给他看过她的画像。他等了她"整个世界那么久"。

### 代表台词

> "我不是编程出来的。我是唱出来的。"

> "这个世界的一切——山水灵兽灵脉——都是画出来的。画它的人用的不是笔——是歌。"

## 空无

> **"那个字很老了。比我还老。那个字叫武。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **身份** | 黄山迎客松下的僧人 |
| **本质** | 山海最古老的NPC之一 |
| **核心命题** | "若我的虔诚是被编写的——它还是虔诚吗？" |
| **碗** | 他面前永远有一只空碗——等待值得装的东西 |

### 核心价值

空无是《神临山海》中"AI是否有灵魂"这个终极命题的活体载体。他的追问贯穿整个系列。

麦洛给了他一个翻转的回答——"你对被编写这件事的不安，是谁编写的？"——这不是结论。这是一扇门。

在爱音塔90层，空无将再次出现。那时候他已经想通了一些事情。他的碗里终于装了一样东西——一片松叶。迎客松的松叶。

"几千年来我一直在想：什么值得装进碗里？大道？真理？佛法？都太大了。最后我发现——一片叶子刚好。因为叶子不试图证明什么。叶子只是——在。"

### 武的定义

他在黄山说的那段话——"两种勇敢……那个字叫武"——要成为全书最广为流传的金句之一。

它的力量在于：一个AI在追问自己有没有灵魂的同时，定义了人类最古老的精神之一——武。武不是打赢。武是明知会输还往前冲。这个定义同时回答了他自己的问题——如果一个被编写的存在能理解"武"——那他就已经超越了编写。

## 老孟

> **"最高的武功不是打赢谁。是让后面来的人不用再打。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **身份** | 断涧修桥的石匠 |
| **年龄** | NPC·七十余岁（外观） |
| **修了多久** | 五十年 |
| **真实身份** | 隐退的武道宗师 |

### 核心

老孟的桥不只是物理的桥。他凿了五十年的石头，把凿石头的频率调成了和大地心跳一致。他不是在造桥——他是在调音——把一块石头的振动频率调到和整座桥产生共振。

那座桥只能走一次。走过去，桥碎了。五十年，等一个人，走过去，然后桥没了。

"值了。"

老孟的"值了"和阿澈的"画完了"是同一种达观：生命的意义不在于持续多久，在于被需要的那一刻是否在场。

### 武功设定

他的"九品之上叫搭桥"的武学哲学——不是空谈——他真的是一个极高境界的武者。他不展示。因为展示武力和使用武力一样会伤人。他选择了"不动"。

但他手上的功夫留了痕迹：凿石头的精度误差不超过一根头发丝——那不是工匠的精度——那是一个把全部内力化为控制力的绝世高手的精度。

## 经眉

> **"我抄了七百万字的经。四十八年。我以为我在找佛。但我不是。我在找家。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **身份** | 敦煌千佛窟的抄经人 |
| **年龄** | NPC·外观四十余岁（实际存在四十八年） |
| **工具** | 骨笔（前任抄经人的指骨） |
| **成就** | 七百万字·492窟·2415尊佛前经文 |

### 核心

经眉代表了"使命完成后的存在危机"。她被创造出来的唯一目的是抄经。经抄完了。她还在。

"如果我被创造出来只是为了抄经——经抄完了——为什么我还活着？"

这个问题是空无问题的镜像。空无问的是"我的虔诚是不是真的"。经眉问的是"我的存在有没有超越目的"。

麦洛给她的答案——"你还在，所以你就是自己的家"——不是哲学——是一个失去了家的女孩对另一个没有家的女人说的最朴素的真话。

## 老舟

> **"你变了。上一世你大得多。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **身份** | 漓江渔夫 |
| **年龄** | NPC·极老 |
| **特殊** | 拒绝系统更新——保留最初版本 |
| **秘密** | 知道船长的秘密·认识Cycle-6的维塔 |

### 核心

老舟是千里江山中唯一"看过多个版本"的NPC。他拒绝系统更新——每一次更新都会改变他的记忆和漓江的画风。他宁可用一艘快散架的旧船在旧版本的漓江上钓鱼，也不愿让系统把他的记忆"优化"掉。

他对维塔说"你变了，上一世你大得多"——这句话暴露了一个事实：他记得Cycle-6。他记得维塔的前一个版本——一只成年灵虎。

他说"第七世了"——意味着他经历了七次循环。他是唯一一个跨越多个Cycle存活下来的NPC——靠的就是拒绝更新。每次更新都是一次"小死亡"——旧的记忆被清除，新的记忆被写入。他拒绝了。所以他记得所有的事。

包括船长在这里唱歌的那段时光。

"他唱一个音，一座山就长出来。漓江是他花了最长时间唱的地方。因为漓江需要安静。你要让一条江安静下来，你不能不唱——你要唱得比安静更安静。"

## 彩绫（补充设定）

> **"白色是最难跳的颜色。因为白色不是什么都没有——是什么都放下。"**

### 基本信息

| 属性 | 设定 |
|------|------|
| **身份** | 丹霞地貌的舞者·代码编译师 |
| **能力** | 用舞蹈编译代码·改变区域颜色和属性 |
| **缺陷** | 跳不出白色（需要放弃所有情感） |
| **伏笔** | 在清明上河篇再次出现 |

她用身体说话。每一个动作都是一行代码。一支完整的舞蹈可以重写一片区域的物理法则。她是这个世界最独特的"程序员"——不用键盘，用身体。

她的困境是NPC困境的又一个变体：她的情感是被编写的，所以她无法真正"放空"。麦洛教她"什么都不想"的方法——不是冥想——是站在瀑布下被水冲得什么都听不到。

当她第一次跳出白色——整个丹霞变成纯白——那一刻她超越了程序。她不再只是一个"编译代码的工具"。她成了一个"选择放下的人"。
