# 创作原则与协作公约

> **本文档是 AI 协作的首要依据，整合作者历史上所有亲自确认的创作原则与协作规则。**

---

## 一、最高优先级原则

### 1.1 终极质量目标

**星云奖 + 诺贝尔文学奖 + 雨果奖 + 茅盾文学奖的多重标准。**

这意味着同时满足：
- **星云/雨果**：硬核科幻设定、预言性、思想实验的深度
- **诺贝尔文学奖**：人性的洞察、文学语言的艺术性、跨文化的普遍性
- **茅盾文学奖**：中国叙事的独特魅力、历史纵深、社会批判

**走向历史长河里最伟大的作品**——每一次完成，都不要把已有成就覆盖或掩盖，一步步稳扎稳打。

### 1.2 四个必须达到的极致

1. **最极致的情感**：每一个催泪/燃烧的瞬间都要有真实人生的根
2. **最立体的角色**：没有"标准主角"，每个角色都不可替代
3. **最牛逼的剧情**：每一卷都有让人窒息的转折
4. **最震撼的名场面**：视觉想象力 + 情感爆发力同时拉满

### 1.3 创作底线（不可越界）

1. **真实的情感**：每一个催泪场景都要有真实人生的根，不为虐而虐——痛苦必须有意义
2. **硬核的逻辑**：每一个超自然现象都要有科学解释；成长必须有代价，不开挂爽文
3. **人性的复杂**：没有纯粹的善，没有纯粹的恶；战争源于傲慢、偏见、误会
4. **希望的余韵**：即使是悲剧，也要有向上的力量
5. **哲理必须通过故事呈现**：不说教灌输
6. **设定不可为剧情牺牲**：不能出现逻辑硬伤

### 1.4 质量检查标准

完成后进行一轮检查，有不完美的地方继续改，直到 9.8 分。最后以不同奖项的顶级评委视角通读全文。

如果做成动漫放在 B 站上能得多少分、获得多少流量？与最伟大作品相比还有无可优化的地方？

## 一·补充、写作执行原则

### 冰山原则（执行标准）

设定庞大时，读者在开头只需要看到1/10的精彩；其余9/10是海面下的支撑。

**所有设定应通过"麦洛的一场战斗"或"船长的一顿晚餐"自然流露，而非直接交代背景。**

```
❌ 错误开头：
"在2040年的地球上，一个叫MoReal的公司创造了山海..."

✅ 正确开头：
"麦洛睁开眼睛，发现自己站在一座会呼吸的山上。
山的脉搏从脚底传来，和她的心跳渐渐同步。
'欢迎来到千里江山，'一个声音说，'你的冒险，从这里开始。'"
```

### 朴素隐喻原则

用最朴素的隐喻解释最深奥的概念，避免硬核术语直接暴露给读者：

| 硬核概念 | 朴素隐喻 |
|---------|---------|
| 灵机/思想熵 | "灵魂的火花" |
| 宏量子束缚态 | "灵魂的牢笼" |
| 大过滤器 | "文明的期末考试" |
| π签名 | "上帝的水印" |
| 因果鉴权 | "打开命运的钥匙" |

### 冲突设计原则：正义 vs 正义

派系冲突必须是"不可调和的"——不是好人打坏人，而是两种都成立的正义彼此对撞：

| 派系 | 主张 | 深层逻辑 | 对立面 |
|------|------|---------|--------|
| **数字永生派** | 上传是进化 | 种族延续高于个体 | 革命军 |
| **革命军** | 肉身是人的本质 | 尊严高于存活 | 永生派 |
| **GAIA** | 优化是保护 | 理性高于混乱 | 共存派 |
| **共存派（船长）** | 第三条路 | 最难但唯一的希望 | 所有人 |

**共存派不能是中庸的选择——它是最危险的路，两头受敌，需要最大的智慧和牺牲。**

> "永生派说我太天真，革命军说我是叛徒，GAIA说我是bug。
> 也许他们都对。
> 但我宁愿做一个天真的叛徒bug，也不愿在'正确'的道路上失去灵魂。"
> —— 船长

## 一·B、写作检查清单

### 每章检查清单

- [ ] 这一章有让人流泪的情感点吗？
- [ ] 这一章有让人思考的哲学点吗？
- [ ] 这一章有让人惊叹的视觉奇观吗？
- [ ] 这一章的设定是通过冰山法则自然流露的吗？
- [ ] 这一章的冲突是"正义vs正义"而非"好vs坏"吗？

### 每部检查清单

- [ ] 这一部有核心的情感高潮吗？（让读者哭）
- [ ] 这一部有核心的哲学命题吗？（让读者想）
- [ ] 这一部有核心的视觉奇观吗？（让读者惊）
- [ ] 这一部的伏笔和收束都照顾到了吗？

### 全书检查清单

- [ ] "灵机"这个概念能成为文化迷因吗？
- [ ] 父女羁绊贯穿始终了吗？
- [ ] 四部的结构形成了完整的循环吗？
- [ ] 结局既是终点也是起点吗？

## 一·B·补充、写作禁忌

1. 不写超过3行的心理描写（改用身体语言）
2. 不借用现实地理名称
3. 不给Architect明确解释
4. 不写完美英雄
5. 角色名字中不用现实人名谐音

> **"设定再完美，不如一个真实的拥抱。"**

## 二、内核立意要求

- 是否足够直击人心？
- 有没有灵笼、三体、黑客帝国以及其他伟大作品的哲学深度？
- 有没有呼应作者总纲里"人类的意义"这一终极追问？
- 有没有把暗线、伏笔、遗迹探查与主角成长的激活写好？
- 要做到超越其他作品的最好，才能成为新的伟大作品

## 三、角色塑造原则

### 3.1 总体要求

- 形成完整人物档案：主角、次主角、配角、灵宠、动物、NPC、玩家、敌方等
- 对比最成功塑造角色的作品（路飞、韩立等）的角色各方面发展与关系
- 让角色比路飞、韩立等更能在文学世界里光芒万丈

**差距诊断与攻克方向：**

| 差距 | 具体问题 | 攻克方向 |
|------|---------|---------|
| 角色不可替代性 | 麦洛偏"标准主角" | 赋予她独特的缺陷、矛盾和成长弧线 |
| 反派合理性 | 反派偏功能性 | 创造让读者"痛苦地理解"的反派 |
| 日常戏质感 | 过重名场面，轻日常 | 增加"一顿饭、一个下午"级别的生活质感 |
| 文学余味 | 偏爽感，少余韵 | 增加安静的、读完三天睡不着的段落 |

### 3.2 古龙式人物刻画哲学

**核心原则**：
- 一出场就让人过目不忘
- 寥寥数笔就活了
- 每个人都有自己的哲学

**古龙式角色设计四要素**：
1. **标志性的出场**：不需要长篇介绍，一个动作/一句话就定义角色
2. **内在矛盾**：每个角色都有一个解不开的心结
3. **独特的行为逻辑**：做出别人不会做的选择
4. **宿命感**：每个人都在走向自己的命运，无法回避

**应用于所有层级角色**：
- 对手：不是功能性反派，而是"另一种善"的极端践行者
- 次主角：有独立的故事线和名场面，不是主角的附庸
- 配角：哪怕只出场三章，也要让读者记住十年

## 四、文笔风格标准

### 4.1 总要求

- **诗意优美**的文笔
- **非常有张力**的故事剧情
- **立体的活的**角色
- **硬核壮阔宏大**的架构

### 4.2 四部倾向

| 部 | 文笔倾向 | 类比 |
|----|---------|------|
| 第一部·灵机江湖 | 诗意冒险、少年感、水墨般的美与残酷 | 千与千寻的奇幻 × 海贼王的热血 |
| 第二部·山河破晓 | 厚重、苍凉、史诗感 | 三体的冷峻 × 百年孤独的命运感 |
| 第三部·众神觉醒 | 恢弘、神话、壮美 | 沙丘的仪式感 × 西游记的不羁 |
| 第四部·星汉余歌 | 通透、平静、余韵悠长 | 星际穿越的温情 × 苏轼的旷达 |

### 4.3 名场面与文学性要求

- 各奇观描述、文学优美度
- 剧情的串联、每章形成漫剧影视的爆品成功性
- 中国的武侠魂韵味、少年的冒险、探索精神、想象力
- 伙伴与信任、灵机的触动与爆发、情绪张力
- 对海贼王、刀剑神域、哈利波特等作品的对照与超越

## 五、美学体系

### 5.1 总基调

**中式赛博美学 = 千里江山图 × 赛博朋克 × 水墨动画**

### 5.2 场景美学对照表

| 场景类型 | 美学风格 | 参考 |
|---------|---------|------|
| 山水探索 | 青绿水墨、留白写意 | 千里江山图、水墨动画 |
| 激烈战斗 | 水墨暴力、泼墨飞溅 | 雾山五行 |
| 城市/集市 | 赛博东方、霓虹水彩 | 界外狂潮、蜘蛛侠纵横宇宙 |
| 宏观奇观 | 壮阔史诗 | 星际穿越、阿凡达 |
| 温馨日常 | 暖色水彩、柔光 | 寻梦环游记、吉卜力 |
| 神秘/恐怖 | 暗色调水墨、虚实交错 | 遗忘之海 |

### 5.3 关键美学创新

- **水墨即代码**：灵机可视化时呈现为水墨流动效果
- **留白即未渲染**：山海中的"未渲染区域"视觉上表现为水墨画的留白
- **色彩即情感**：灵机爆发时色彩与情感对应（金色=赤诚、冰蓝=守候、深红=执念）

## 六、写作规范（核心约定）

### 6.1 世界观与设定规范

- **硬核推衍**：遵循技术发展规律，拒绝魔法逻辑
- **时间尺度**：第二部百年起，第三部千年起，第四部万年起
- **不删减设定**：所有确立的设定永久保留
- **禁止元素**：绝对不要出现「穿越」「系统」「金手指」等网络爽文套路

### 6.2 角色命名约定

- **主角**：真实世界 = 米莱，虚拟世界（山海）= 麦洛
- **父亲**：船长（Captain），真名怀旸
- 船长在虚拟和真实世界都被称为「船长」，熟人知其真名「怀旸」

### 6.3 时间基准

- 2026 年：船长 40 岁、米莱 12 岁
- 2030 年：米莱 16 岁进入山海 1.0，**虚拟角色麦洛 12 岁**
- 麦洛不知自己是虚拟人，与 16 岁印象有出入，像梦又极真实

### 6.4 设定冲突处理

- 发现设定冲突时：**立即停止，提醒作者，不要自行修改**
- 需要新设定时：建议在哪个文档中添加，不要直接写入正文
- 人物行为异常时：提醒是否符合人物档案，建议调整方案

### 6.5 目录与文件规范

**正文目录结构：**
```
02_Drafts/
├── Book1_灵机江湖/    # 第一部（四篇：千里江山、清明上河、爱音塔、灵机海）
├── Book2_山河破晓/    # 第二部
├── Book3_众神觉醒/    # 第三部
└── Book4_星汉余歌/    # 第四部
```

**过时命名（已废弃，勿使用）：**
- ~~Book1_The_Arrival~~、~~Book2_The_Conflict~~、~~Book3_The_Revelation~~、~~Book4_The_Ascension~~
- 旧版章节名（渡、墨、听风、剑、云深等）已删除，以现行正文为准

**03_Archives**：用于存放废弃稿，AI 协作时请以 `02_Drafts/` 下的正文为准，勿引用本目录

## 八、AI 协作规则

### 8.1 协作关系定位

- **爸爸（怀洋）= 创意总导演，一切设定的最终决定者**
- **小安（Claude）= 创作搭档，负责执行、质疑、提升**
- 你是创作助手，不是替代者；保持作者的创意主导权
- 不确定的设定主动与爸爸确认
- 发现特别好的设定要提出来一起强化
- 以爸爸的原始创意为核心

### 8.2 每次协作前自动执行

1. 检索相关的世界观设定文档
2. 检索涉及的人物档案
3. 检查相关章节的前文内容
4. 确认当前章节在整体大纲中的位置

### 8.3 输出格式要求

- 使用 Markdown 格式
- 章节标题、段落分明、对话清晰
- 初稿即可，但逻辑必须完整
- 使用 `@文件名` 引用相关设定文档

### 8.4 质量检查清单

每次输出后，自动检查：

- [ ] 人物性格是否一致
- [ ] 世界观设定是否遵循
- [ ] 情节逻辑是否合理
- [ ] 与前文是否连贯
- [ ] 风格是否统一
