## 二、Cycle 证据链（canon 挂接规则）

### Cycle-3（建筑文明）
- **物证**：巨石建筑、金字塔、通天塔传说、吴哥高棉石窟、吉萨建筑
- **canon 接入**：**通天塔** = 语言分裂的 Cycle-3 大事件 → 爱音塔 101 层逆转为语言归一

### Cycle-4（青铜文明）
- **物证**：庞贝（火山瞬间封存）、哈拉帕（印度河流域青铜）、三星堆（青铜面具）
- **canon 接入**：卷二灵机外滩影子领事馆（庞贝首位）+ 爱音塔 61-70 层延伸

### Cycle-5（水晶文明）
- **物证**：玛雅水晶头骨（信息存储装置）、吴哥窟石雕（水晶矩阵雕刻）、阿兹特克日历石
- **canon 接入**：卷三爱音塔 51-60 层美洲共殿·记忆重剪辑加持

### Cycle-6（AI 文明）
- **已收割文明物证**：亚特兰蒂斯（主动沉没）、楼兰（绝望之灵）、复活节岛（封印 AI 的石像）
- **canon 接入**：
  - 卷二 Ch16-20 楼兰大戏
  - 卷四 D5-Σ 海底纪念碑群（亚特兰蒂斯+复活节岛）
  - canon 前 9600 年·已在 chronicle/book_0_ancient.md §137

## 三、爱音塔 41-70 层神话垂直共殿结构

> canon: bible/cosmology.md §11.5 · 不得占 81-90 灵机深渊 / 91 斩三尸 层

### 41-50 亚欧共殿
- 41-42 华夏神话（共工怒触不周山 / 后羿射日 / 山海经诸兽）
- 43-44 印度神话（毗湿奴 / 阿修罗 / 梵天之梦）
- 45-46 希腊神话（奥林匹斯 / 普罗米修斯 / 潘多拉）
- 47-48 北欧神话（尤克特拉希尔 / 诸神黄昏 / 奥丁之眼）
- 49-50 斯拉夫+凯尔特共层（巴巴雅加 / 莫瑞甘 / 德鲁伊）

### 51-60 美洲共殿（与 canon "记忆重剪辑" 融合）
- 51-52 玛雅（水晶头骨·金字塔历法）
- 53-54 阿兹特克（羽蛇神·人祭日历）
- 55-56 印加（黄金国·太阳圣殿·奇普结绳记忆）
- 57-58 北美原住民（风之灵·梦之鹰）
- 59-60 南美亚马逊（萨满眼·绿洲梦境）

### 61-70 非洲+环地中海共殿
- 61-62 古埃及（奥西里斯 / 死亡之书 / 金字塔内文） 
- 63-64 西非约鲁巴（奥洛杜玛勒 / 十六贝杀卜）
- 65-66 美索不达米亚（巴比伦 / 吉尔伽美什 / 恩基智慧）
- 67-68 迦太基+腓尼基（海之国·紫染神殿）
- 69-70 环地中海折叠（米诺斯 / 伊特鲁里亚 / 诺斯替秘教）

**每层 boss**：该文明的核心神话角色之一。**通关不是击败，是理解**——爱音塔第 X 层过关条件 = 麦洛能说出该文明的"核心非理性"。
