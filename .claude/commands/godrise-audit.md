# 神临山海·一致性审计

对指定设定域进行全域交叉核验，输出矛盾和漏洞清单。

## 输入
审计范围：$ARGUMENTS
（例如："角色年龄"、"力量天花板"、"AI版本时间线"、"全域"）

## 执行流程

1. 读 CLAUDE.md 获取铁律清单
2. 读 bible/ 全部圣旨文件（最高权威）
3. 读指定域的所有相关文件
4. 逐条检查：
   - 时间线是否一致（同一事件在不同文件中的年份）
   - 角色年龄是否对得上（出生年 + 事件年 = 当时年龄）
   - 力量天花板是否遵守（各部/各年代的境界上限）
   - 因果逻辑是否自洽（A事件必须发生在B事件之前）
   - 铁律是否有被违反
   - "待推衍"/"待确认" 标记汇总

5. 输出格式：
   ```
   ## 矛盾清单（必须修正）
   1. [文件A:行号] vs [文件B:行号]：具体矛盾描述
   
   ## 漏洞清单（需要补充）
   1. [文件:行号]：缺失什么信息
   
   ## 待确认清单（需爸爸决策）
   1. 具体问题
   
   ## 通过项
   已检查N条铁律/M个文件，以下无矛盾：...
   ```
