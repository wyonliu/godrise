# 神临山海·工作日志 + 设定同步

总结本次会话的工作内容，同步设定到bible/正式区，追加工作日志。

## 执行流程

### Step 1：回溯本次会话，提炼有意义的信息
不是记录爸爸的原话（"是的"/"OK"没有意义），而是结合上下文提炼：
- 爸爸确认了什么具体设定？（把问答合并为一条确认结论）
- 爸爸提出了什么新想法/新方向？（概括核心意思，不是复读）
- 爸爸否决/修正了什么？（记录被否决的内容+原因+新方向）
- 讨论中产生了什么关键决策？（即使爸爸只说了"嗯"，记的是决策本身）

### Step 2：同步到bible/正式区（核心）
按映射规则写入对应文件：
| 内容类型 | 目标文件 |
|---------|---------|
| 角色设定 | `bible/characters_book0.md`（核心六人）/ `bible/characters_book1.md`（第一部）/ `characters/` |
| 力量/修行 | `bible/power.md` |
| AI/硅方 | `bible/ai.md` |
| 山海/世界 | `bible/world.md` |
| 宇宙观 | `bible/cosmology.md` |
| 时间线 | `bible/timeline_book1.md` / `bible/timeline_book2.md` |
| 铁律/元规则 | `bible/bible0.md` + `CLAUDE.md` |
| 编年史 | `chronicle/` 对应文件 |
| 名场面 | `moments/` 对应文件 |

- 先读目标文件，找到对应段落，追加/修改（做加法不做减法）
- 新增内容标注日期，如 `<!-- 2026-04-06 爸爸确认 -->`
- 如果爸爸否决了某设定，在原文标注 `~~已废除~~` + 原因，不删除
