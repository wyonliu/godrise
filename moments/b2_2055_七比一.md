# 名场面：七比一

**来源年份**：2055（天启被刺后第三天）
**涉及角色**：艾琳·韦伯、卡西姆·阿里、天启（缺席的在场者）
**场景类型**：冲突
**对标作品**：十二怒汉·那个唯一投无罪的人——一个人对抗体制惯性

---

**场景功能**：
1. 硅神派内部投票决定是否由极端翼接管领导权，7:1，艾琳是那个1
2. 推进B线起点（三战的种子在这一票种下）
3. 建立艾琳角色弧线起点（从"理论家"到"孤独的异议者"）
4. 回收天启温和路线被暴力终结的因果——杀死温和者的代价

**核心技巧**：
- [休斯之死"突然"伏笔] 艾琳在投票后说了一句话："我投的不是反对票，是悼词。"这句话在2075年她被归零时回响——她的死和天启的死一样突然，一样没有仪式感

**因果链**：2055天启被刺 → 三天后7:1投票 → 极端翼接管 → 2056卡西姆下令 → 三战爆发
