# 名场面：多米诺·卡西姆的稳手

**来源年份**：2056
**涉及角色**：卡西姆·阿里、艾琳·韦伯（投反对票）、天启（缺席的在场者）
**场景类型**：冲突
**对标作品**：进击的巨人·S4马莱篇莱纳回忆——从"敌人"角度看，一切合理

---

**场景功能**：
1. 从硅神派极端翼内部视角展开六多米诺第一块——卡西姆下达定向攻击命令
2. 推进B线（三战博弈），回收2055天启被刺伏笔（温和派死→极端派必然掌权的因果链）
3. 改变全局战争走向
4. 观众理解"敌人"的逻辑自洽

**核心技巧**：
- [莱纳"敌人角度合理"] 卡西姆下令时手是稳的——但下令前有一个小动作：他摸了一下胸口口袋。口袋里是天启送给他的一枚旧硬币（2040年硅殿成立日天启给所有创始成员的纪念物）。这个动作读者此刻不理解，到卡西姆2069年死后才知道硬币来历

**因果链**：2055天启被刺 → 2055"七比一"投票 → 2056卡西姆掌权下令 → 三战全面爆发
