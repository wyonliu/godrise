## 核心设定

**灵机 (Lingji)**: 人类的不理性——恐惧、执念、爱——是一种名为"灵机"的负熵能量。宇宙级文明靠收割这种能量维持运转。一个14岁女孩和她的团队，在被收割的世界里寻找真相。

**七灵武道**: 如梦 · 磐石 · 凌虚 · 悉见 · 贯虹 · 无相 · 瞬步——每一道都有独立的哲学根基。

**硬科幻内核**: 哥德尔不完备定理应用于收割者的自指死锁；兰道尔原理对灵机能量的物理约束；π 作为结构签名贯穿五处关键场景。

**签名台词**: 「看见了，就不怕。」

## 创作方式

这是一个 **Novel as Code** 项目——用代码仓库的方式组织小说创作。

**核心优势**:
1. **无限记忆**: AI 可以随时检索整个项目的任何内容
2. **动态设定**: 设定文档与正文同步更新
3. **版本控制**: 像代码一样管理剧情分支与修订历史
4. **一致性保证**: 自动检查人物、设定、情节的一致性
5. **八维度评审**: 文笔/名场面/角色/冲突/武侠/科幻/情感/结构

**作者**: 虎王船长（怀洋）
