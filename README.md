## 核心设定一句话

- **灵机**：人类意识中不可计算的成分，在量子层面产生真随机，是对抗收割的最后武器
- **山海**：既是虚拟世界产品，也是灵机采样仪、麦洛的宇宙、人类文明后备箱——四重身份的矛盾不会被解决
- **收割**：不是惩罚，是清洁。Architect 的观察者搜索在 10^500 个弦景观中设参数条件，每轮 Cycle 是一组测试
- **船长的押注**：爱是物理现实的一部分。代价是永远不能对女儿完全透明
