## 一 · POC 期边界与判定条件

### 1.1 时间 / 空间 / 预算边界

| 项 | 边界 |
|---|---|
| 时长 | 12 个月（2026-05-01 → 2027-04-30）· 不延期 |
| 地理 | 单城首店 POC · 第 M7–M12 视首店结果再启动第 2–3 家 pilot（北京 / 上海 / 成都中选）|
| L3 硬件 | 仅做门店内 MR · **不碰消费端头显** |
| L3 店数上限 | 期内 ≤ 3 家 pilot · 任何理由不加码 |
| 预算总盘 | 项目内部规划参照区间 · 见《POC预算表_v1.md》 · 每阶段 tranche 解锁 |

### 1.2 成功判定（M12 同时满足 · 具备继续扩张条件）

- 首店连续 3 个月单店月流水达到内部参照下限（见总纲 §12.4 · POC 期待验具体阈值）
- 合规意见书（公安 / 工信 / 文旅 / 规划 / 市监五线）全部出具 · 无阻断项
- 跨媒介副总裁到岗并完成首个版本季度交付
- L1 30 日留存、L2 信标带 30 日留存落在总纲 §12.2 / §12.3 参照区间内
- 小说卷一封版 · L1 / L2 / L3 叙事锚点一致

### 1.3 失败判定（任一触发 · 进入 §七红线流程）

- 合规意见书任一线出具 "不予" 结论
- 跨媒介副总裁岗位 M6 前未签 offer
- 首店 KPI 连续两月低于内部参照下限 70%
- 现金流剩余月数低于预警阈值且无法在 30 天内补足
- 小说卷一 M9 前未封版

### 1.4 范围外（POC 期不做）

- 消费端头显采购 / 销售 / 订阅
- 多城并行扩张（超过 3 家 pilot）
- 出海 / 文旅旗舰签约 / IP 平台化（§11 P2/P3 阶段内容）
- L2 信标带消费端量产（POC 期仅 MVP 限量 100 台，详见 §四）

## 二 · Week 0–4 启动动作清单

以 W0 = 2026-05-01 为零点。

### W0（启动周）

- [ ] 项目治理结构落地：项目委员会组建 · 周会制 · 月度对账
- [ ] 《POC 期预算表》《本行动清单》两份文件版本锁定并进入版本控制
- [ ] 通知核心协作方（小说、IP、硬件候选）POC 期启动

### W1

- [ ] **合规律所**：发出 RFP 给 3 家律所（具备公安 / 工信 / 文旅 / 规划 / 市监五线案例）· W3 前完成 1 家签约
- [ ] **跨媒介副总裁 JD**：岗位说明书定稿 · 候选画像（十亿级 IP 跨媒介操盘经验）· 挂内推 + 猎头双通道
- [ ] 首城候选清单初评：北京 / 上海 / 成都三城的物业清单各 10 处起

### W2

- [ ] 律所短名单出炉 · 三家面谈 · 合规意见书 SOW 初稿
- [ ] 副总裁候选初筛 · 首批 5 人进入一面
- [ ] 小说组：卷一封版时间表确认（节点 M3 / M6 / M9）

### W3

- [ ] **律所签约**（合规意见书 SOW 定稿 · 见总纲 §13.3 / §13.4 · 公安 LBS / 工信通信 / 文旅 IP / 规划商业 / 市监经营五线）
- [ ] 副总裁二面（项目委员会全员参与）
- [ ] 首城定点（三选一 · 锁定）· 物业接触清单锁定到 8–12 处 LOI 候选

### W4

- [ ] 副总裁 offer 发出（目标 M2 前到岗）
- [ ] 物业 LOI 接洽启动 · 至少 3 处进入尽调
- [ ] L1 手游研发组扩编 JD 上线（服务端 / LLM 台词 / 客户端 / 运营各岗）
- [ ] L2 信标带硬件合伙人 / 代工厂接触（总纲 §14.1 信标带供应链项）

> Week 0–4 红线：若 W4 末律所未签约、副总裁未发 offer、首城未定点 · 项目进入 §七流程。

- [ ] L3 Spark 空间首个 MR 副本内测（总纲 §14.5 POC 待验三条首轮采数）

**场地**
- [ ] 首店装修完工 · 设备联调
- [ ] 本地 CM 三人挂官页（总纲 §14.4 条 3 · 真人真名真脸 · 不合规不开城）

### 4.3 M6（2026-10）· **首店开业 + 首次 go/no-go**

**产品**
- [ ] 首店开业（Spark 档 · 单场 8–12 人 · 总纲 §9.2）
- [ ] L1 公测
- [ ] L2 信标带 MVP 限量 100 台与首店开业仪式绑定发放

**验证**
- [ ] 30 天 KPI 盲测启动（总纲 §14.5 三项待验事项首次完整采数）
- [ ] 玩家 NPS / 复购意愿 / 仪式出席等总纲 §12.4 参照区间开始滚动采集

**决策节点 · M6 go/no-go**
- 继续：首店开业 30 天核心指标落在总纲参照下限及以上 · 合规无阻断
- 调整：任一核心指标低于下限但差距 ≤ 20% · 进入 30 天修正期后复评
- 关停：指标差距 > 20% 或合规出现阻断 · 触发 §七红线

> M4–M6 红线：首店开业前合规意见书新增阻断项、CM 真人真脸未落实、L2 MVP 未到货 · 均触发 §七。

## 七 · 红线清单（触发即停工）

以下任一条件触发 · 项目进入 72 小时紧急评审 · 评审结论三选一：继续（带修正）/ 暂停（30 天重评）/ 关停。

| # | 红线 | 判定口径 | 关联条款 |
|---|---|---|---|
| R1 | 合规意见书五线中任一出具 "不予" 或新增阻断项 | M3 末初版 · 滚动复核 | §三 3.3 / §13.3 |
| R2 | 跨媒介副总裁 M3 前未到岗 | 岗位空缺 · 无代理安排 | §三 3.3 |
| R3 | 首店 KPI 连续两月低于参照下限 70% | 月流水 / NPS / 复购三项中任两项 | §四 / 总纲 §12.4 |
| R4 | 现金流剩余月数 < 预警阈值且 30 天内无补资路径 | 见《POC预算表_v1.md》§3 | 预算表联动 |
| R5 | 小说卷一 M9 前未封版 | L1/L2/L3 叙事锚点无法锁定 | §五 5.3 |
| R6 | 本地 CM 真人真名真脸未落实或 CM 脱岗超 7 天 | 官页无人 · 或顶替操作 | §四 4.2 / §14.4 条 3 |
| R7 | L2 信标带 MVP 未在 M4 末到货 | 开业仪式发放链条断裂 | §四 4.1 / §14.4 条 4 |
| R8 | 合规阻断项未在 30 天内出清 | 律所书面意见 | 通用 |
| R9 | POC 期内擅自启动 ≥ 4 家门店 / 碰消费端头显 | 范围外动作 | §1.4 |

> 任一红线**不允许以 "先推进 · 后补齐" 的理由绕过**。红线意味着暂停而非讨论。

## 八 · 每月必交付物与治理节奏

### 8.1 月报结构（每月 5 日前交项目委员会）

1. **进度**：本月完成项 · 未完成项 · 原因
2. **数据**：对照总纲 §12 参照区间 · 给区间与样本量 · 不给单点承诺
3. **合规**：五线状态（绿 / 黄 / 红）· 新增阻断项
4. **预算**：本月实际 vs 预算 · 剩余月数 · 下月 tranche 解锁请求
5. **红线**：当前是否触发 · 触发则附 72 小时评审纪要
6. **风险**：总纲 §14.1 风险表滚动更新

### 8.2 季度交付物

- Q1（M3）：合规意见书正式版 · 首店合同 · 副总裁首份季度计划
- Q2（M6）：首店开业包 · L1 公测版本 · 30 天盲测首轮数据
- Q3（M9）：双店 / 卷一封版包 · L1 90 日数据 · CM SOP
- Q4（M12）：POC 总报告 · 真实 CAC/LTV · 后续路线图草案

### 8.3 年度节点（M12）

- POC 总报告 · 三店真实 P&L · 合规年审 · 后续融资 deck · 后续 12 个月行动清单 v2
