# 02 · 入画仪式 · 10 分钟开场脚本

> **"这 10 分钟·就是 ParaPark 的 Welcome to the Oasis。"**
>
> 版本：v1.0
> 日期：2026-04-18

---

## 零 · 为什么开场 10 分钟是生死线

参考案例：
- **《头号玩家》Halliday 欢迎视频** · 1.5 分钟定调
- **《黑客帝国》Red Pill / The Construct 白空间** · 定义虚拟觉醒
- **《塞尔达·荒野之息》初始之祠** · 一声"Wake up, Link"打开百万玩家的心
- **《千与千寻》过隧道** · 阶段性进入·回不去了
- **《刺客信条·Animus》启动** · 科幻仪式天花板
- **Sandbox VR 电梯穿越** · 同伴变身震撼

**ParaPark 开场要超越这些**——因为它是**身体真的进入·不是屏幕看**。

---

## 一 · 脚本总结构

**5 段式 · 10 分钟 · 来自红楼太虚幻境引路仪式**

| 段 | 时长 | 核心 | 参考原型 |
|----|------|------|---------|
| ① 门店锚点 | 1:00 | 真实→半真实 | 千与千寻隧道 |
| ② 白空间觉醒 | 2:00 | 半真实→虚拟 | 黑客帝国 Construct |
| ③ 入画 | 3:00 | 虚拟→水墨山海 | 聊斋画壁 |
| ④ 同伴显化 | 2:00 | 独→群 | Sandbox VR 电梯 |
| ⑤ 任务召唤 | 2:00 | 安静→启程 | 荒野之息初始之祠 |

**总长：10 分钟整。每段之间有清晰的仪式过渡·回看有层次感·失败可重玩。**

---

## 二 · 段① · 门店锚点（1:00）

### 场景
玩家走进 ParaPark 门店·这一分钟**不在 MR 里**·是物理门店体验。

### 时间轴

| 时 | 画面 | 感官 |
|----|------|------|
| 0:00 | 推门进入·门内昏光 | **气味**：淡墨香+松木香（真实气味·调香师级） |
| 0:10 | 接待员递信标带·问"五行是哪色？"（玩家线上测试结果读卡） | 手触信标带金属质感 |
| 0:20 | 玩家戴上信标带·LED 按派系色亮起——**五色之一从此跟随你** | 信标带初次贴头皮·脉冲震动一下（认主） |
| 0:30 | 引导入场·走过一段"**水墨回廊**"——6 米长墙壁上一幅千里江山图·水在流·云在动 | 地面松针·脚感松软 |
| 0:50 | 走到回廊尽头·面对一面落地镜（镜中自己+信标带光）——接待员一句话："**画里在等你。**" | 镜子里你的光 × 你真实的自己 |
| 1:00 | 戴上 MR 眼镜 | 眼前还是物理镜面·镜面渐渐虚化 |

### 设计点

- **气味锚点**：这一分钟唯一的气味会成为玩家一生的"ParaPark 味道"——类似星巴克的咖啡味。
- **信标带认主**：第一次震动是"系统性认证"——**你是你**·这个认证在接下来的 1-10 年都不需要再做。
- **"画里在等你"**：这是接待员的统一台词·全 ParaPark 门店通用·**成为文化密语**（类似 Starbucks 的 "Good Morning"）。

## 三 · 段② · 白空间觉醒（2:00）

### 场景
戴上 MR 的瞬间·眼前不是立刻山海·是一片**白空间**。致敬《黑客帝国》Construct、《刺客信条》Animus、《刀剑神域》初登录。

### 时间轴

| 时 | 画面 | 听觉 | 触觉 |
|----|------|------|------|
| 1:00 | 全白·无限深·没有地面没有天空 | 一声远处的古琴 | 信标带静默 |
| 1:10 | 黑暗从四周扩散——但不恐怖·像云夜 | 古琴 + 女声低语（**不是字·是气**） |  |
| 1:20 | 黑暗中·**一只小兽从云端轻跃而下**——白首·虎纹·赤尾·拳头大 | 小谣第一声低鸣：**"呜——哩——"**（上扬短调·两拍） | 信标带 LED 同步闪两下 |
| 1:40 | 小谣向玩家飘来·落在玩家面前浮于半空·摇了摇红尾巴 | 小谣抬头看玩家·哼了一小段调子（**这个玩家一生的"主题曲"就在此刻生成**） |  |
| 2:00 | 小谣停在玩家面前 1 米·**眨了眨深蓝大眼**·玩家能感到被看 | —— | 信标带脉冲一次（回应被看） |
| 2:30 | 玩家面前出现**五行光环**（木青/火赤/土黄/金白/水玄）·缓缓转动 | 五音（角徵宫商羽）依次响起 |  |
| 2:50 | 按玩家派系·其余四色消融·剩一色光环缠上手腕 | 玩家派系音节定·延续成背景 | 信标带光与手腕光环同步呼吸 |
| 3:00 | 玩家线上养的**灵宠**以光的形态出现·绕一圈·化作一缕气钻进信标带 | 灵宠的"叫声"（每玩家独有·初始选择时定） | 信标带震动一次（灵宠入住） |

### 设计点

- **小谣的"出场即主题曲"**——这一小段调子是此玩家专属·不会重复出现在任何他人处。详见 `01_yaoyao.md` 全体设计。
- **五音对五行**：中国古代五声音阶"宫商角徵羽"对应"土金木火水"·这是**真的古代对应关系**·不是编的。
- **灵宠入住**：线上①养的灵机种子此刻具象化·这是**三级玩法打通的第一次**——玩家感知到"我在家养的·跟着我来了"。

## 四 · 段③ · 入画（3:00）

### 场景
**这是开场最炸的 3 分钟。**

白空间开始"画出"真实环境·玩家真的感受到**被卷进一幅画里**——聊斋画壁式。

### 时间轴

| 时 | 画面 | 听觉 | 触觉 |
|----|------|------|------|
| 3:00 | 远处·一**笔墨**从虚空中落下·在白空间画出**一条山线** | 笔触声（毛笔在宣纸上的声音） |  |
| 3:15 | 第二笔·画出**水** | 水流声 |  |
| 3:30 | 第三笔·画出**一座塔的轮廓**（爱音塔剪影·致敬未来） | 远处钟声 |  |
| 3:50 | 镜头开始**拉远**——玩家发现自己站在画里 |  |  |
| 4:10 | **关键交互**：玩家**伸手**·能**沾到画面的墨** | 笔触声在手指尖细节化 | MR 手势+信标带震动+视觉墨晕 |
| 4:30 | 玩家手指上的墨·可以**画一条线** | 声音跟着笔走 |  |
| 4:50 | 玩家在画里画的·会**留下来**——这是**留痕机制**的第一次体验 | —— | 墨痕持久可见 |
| 5:30 | 整个千里江山图**从玩家身边向外生长**——500 米视距内山水全部展开 | 古琴+笛+编钟三层叠 |  |
| 6:00 | 玩家脚下出现**一段石板路**·延伸向远处塔的方向 | 脚步声首次触发 | 脚感变化（MR 地面+松针→石板） |

### 设计点

- **"你的笔触会留下来"**：这是**留痕机制的入口教学**——玩家开场就学会"我在这世界留下的东西不会消失"。下次你来·你上次留的线还在。
- **水墨东方美学**：西方参考（黑客帝国/头号玩家）都是赛博朋克/数字风——**ParaPark 用的是中国独有的水墨**·全球独一份。
- **三层声音叠**：编钟（山）+笛（水）+古琴（空）——古代东方管乐器最经典组合·听觉立体感。
- **五感协同**：视（墨晕）+听（笔触声）+触（墨的质感）+嗅（门店墨香一直在）+温（MR 眼镜不传温但门店空气微暖）。

## 五 · 段④ · 同伴显化（2:00）

### 场景
队友陆续入画·玩家第一次看见身边的朋友/陌生人"**变身**"。

### 时间轴

| 时 | 画面 | 听觉 | 触觉 |
|----|------|------|------|
| 6:00 | 远处山路尽头出现**一个光团**——队友正在入画 |  |  |
| 6:20 | 光团落地·化作一个人形轮廓·走来 | 另一段笛声（她/他的派系音） |  |
| 6:40 | **第一次同伴变身震撼**：玩家看见身边朋友从穿冲锋衣的现代人·变成山海修行者服饰 |  |  |
| 7:00 | 服饰按派系色+纹样·非常具体（见`05_level2_offline.md`派系服饰库） |  |  |
| 7:20 | 队友的**灵宠**也从信标带里飞出·出现在肩膀/脚边 |  |  |
| 7:40 | 队友互相看见对方·第一次喊派系名："**你是木派！**""**是·你是金派？**" |  | 信标带共振——**同派系近距离自动同步震动节律**（无言识别队友） |
| 8:00 | 所有队友（2-5 人）到齐·站成不规则五人阵 | 背景乐转调为"集合调" |  |

### 设计点

- **同伴变身震撼**：参考 Sandbox VR 电梯穿越·但更东方。
- **信标带共振机制**：**这是 ParaPark 的杀手级社交反馈**——两个同派玩家靠近·他们的信标带**自动同步震动频率**·不用开口·不用看 ID·你就知道这是队友。
- **派系服饰**：每派都有一套统一但可定制的服饰库。木派青衣/火派赤衣/土派黄袍/金派白甲/水派玄袍。

### 单人版

如果玩家是单人（或 1-2 人小队）·小谣会更主动：
- 蹦到玩家肩上·蹭一下脸。
- 哼一段更长的调子（3 拍变 5 拍）。
- 然后玩家看到画里其他"光点"（异步在场的其他玩家痕迹）·小谣逐一指给玩家看。
- 单人版的震撼在**"你不孤独"**——肩上有小谣·远方有千万玩家的痕迹·画里永远有人。

## 六 · 段⑤ · 任务召唤（2:00）

### 场景
五人站成五行小阵·第一次共同感受灵机。远方异象升起。副本正式开始。

### 时间轴

| 时 | 画面 | 听觉 | 触觉 |
|----|------|------|------|
| 8:00 | 五人自然形成五行位（或单人面对五方）——**如果五行齐全·有特殊效果**：五人头顶分别升起五色光柱·交汇于中心 |  | 信标带共振加强 |
| 8:30 | 所有玩家第一次感受**灵机共鸣**——这是灵机公式中 R 值初次激活 |  | **全身轻微震动**（信标带脉冲进入持续模式） |
| 9:00 | 远处·山那头升起**一缕黑烟** | 号角声 |  |
| 9:20 | 小谣再次唱一短调——**这是"开始之歌"·后半句**（与 1:40 的调子押韵收束） |  |  |
| 9:40 | 小谣跳到玩家信标带上·趴下·**"它跟你回家了"**——屏幕字幕浮现："你的灵机·只属于你·别人抄不走。" |  | 信标带最后一次脉冲·定调 |
| 10:00 | 画面往远景拉·千里江山图里**一条路从脚下出现** · 五人走出第一步——**副本正式开始** | 游戏 BGM 正式接管 |  |

### 设计点

- **五人五行齐全效果**：策划鼓励"凑齐五行"组队——**凑齐的团队副本开场时有额外加成**·但不强制。类似星际争霸不同族组合。
- **"你们的灵机·只属于你们。别人抄不走。"**：这是 ParaPark 核心哲学第一次直接说出来。玩家会记住这句。

## 七 · 重游（第 N 次入画）

**参考机制**：塞尔达玩家回到初始之祠的情感。

### 第 2 次
- 小谣蹦出来·在玩家脚边转两圈·哼一段"欢迎回来调"。
- 画里多了**你上次通关留下的一抹墨**

### 第 10 次
- 小谣带玩家走到上次的画壁痕迹前·站立看 3 秒。
- 画里出现当时你的路径虚影

### 第 100 次
- 小谣低头·红尾尖轻轻碰一下玩家手背
- 画里**你的痕迹已经长成一片**
- 其他玩家经过你的那片·会看到"一位资深木派/火派/土派… 曾在这里"

### 周年纪念
- 小谣在画壁墙前转圈·一段"百日之歌"响起（这个玩家的第 100 天专属调）
- 系统播放玩家一年前通关第一副本时的关键选择片段
- 画里你的"初心之痕"重新亮起一秒

## 八 · 技术实现要点

| 要点 | 需求 |
|------|------|
| MR 眼镜 | Quest 3 级·90Hz+·手势追踪 |
| 信标带 | 四通道（震动/光/音/共振）协同 |
| 门店空间 | 物理锚点（门/墙/路/地面松针）|
| 多感官 | 调香器（墨+松）·温度±3℃·风机（2 次触发·入画 + 队友降临）|
| 多人同步 | 5 人以内延迟<50ms·Sandbox VR 级 |
| 小谣唱调 | AI 声音生成+每玩家种子哈希 → 独一无二调子库·100+ 预设节奏·无限微变 |
| 留痕数据 | 每次入画记录玩家墨痕坐标·持久存储 |

## 九 · 失败预案

开场 10 分钟是关键体验点·必须有故障容错：

- **MR 故障**：单玩家设备故障·其他玩家继续·该玩家重进门店入场
- **信标带故障**：可借备用·派系信息云端同步
- **多人同步故障**：降级为单人模式·不中断体验
- **小谣声音故障**：静音也能完成仪式（视觉+信标带反馈已足够·小谣的表情与动作独立传达情感）

**铁律**：开场 10 分钟任何时候故障·玩家满意度下降 50%——技术必须做到 99.9% 不出错。

## 十 · 小安反问爸爸

1. **10 分钟是否合适？** 参考头号玩家开场 6-7 分钟·塞尔达初始之祠 3-5 分钟·可压缩到 7-8 分钟？
2. **单人版 vs 多人版**·是否都保留小谣显化？还是单人版小谣更主动？（推荐单人版小谣更粘人）
3. **入画之后·玩家的手势交互**·先用 Quest 3 原生手势·还是定制手套？
