## 一 · 心流顶点：陪伴

L1 顶点是**独处时的温柔** · 不依赖 L2/3 即能让人上瘾。

参照锚：Paradot 的情感深度 × 旅行青蛙的离线惊喜 × Animal Crossing 的慢日循环 × 灵机江湖的 IP 张力。

## 二 · 灵机种子（玩家核心资产）

### 2.1 定义

每个玩家在入画仪式（`02` 文档）结束时获得**一颗灵机种子**——**玩家本人的灵机投影**。

- 视觉：不规则光点 · 介于水滴与火苗之间 · 色随五行偏移
- 听觉：贴耳 0.618Hz 低频呼吸声（与副本信标印同源）
- 触觉：手机震动频率与玩家当日心率同步（健康授权后）

### 2.2 六维属性（对应灵机公式）

| 字母 | 显示名 | 成长来源 | 感知 |
|---|---|---|---|
| S | 根骨 | 入画基准 + 健康 | 半年 |
| α | 悟性 | 抉择一致性 + 冥想 | 月 |
| P | 身法 | 步数 + 对抗战绩 | 周 |
| M | 心神 | 冥想时长 + 感悟答题 | 周 |
| L | 灵机 | 抉择 + 共鸣 + 喂养 | 季度 |
| R | 共鸣 | 社交密度 | 月 |

六维不均衡 = 特性（不是 bug）。玩家自发形成"偏身法 / 偏心神"流派 · 对应 L2 的职责分工。

### 2.3 不可复制（护城河）

- 真随机熵（硬件 + 首次呼吸采样）
- 每日抉择路径分叉指数爆炸
- L2 信标同步数据哈希进种子

**外挂可刷数值 · 刷不出活种子**——这是小说"灵机不可复制"铁律的直接翻译。

## 三 · 双接引人选择（入画即选）

### 3.1 主接引人选择

玩家入画时通过 3 道情境题确定**主接引人**：

- 倾向**宇宙星辰 / 见景爱啸 / 清晨起行 / 一眼看穿 / 嘴快心细** → **星言**
- 倾向**桃花灼灼 / 见人先笑 / 共情拥抱 / 市井烟火 / 直觉先到** → **夭夭**

### 3.2 主接引人做什么

| 时段 | 主接引人出现形式 |
|---|---|
| 应用打开第 1 秒 | 屏幕一角 · 静态出现 · 如"在等你" |
| 冥想 | 主接引人陪坐屏幕旁 · 星言指天 / 夭夭捧脸 |
| 抉择 | 主接引人读完切片后做个表情（不干涉选择） |
| 喂养 | 主接引人的灵宠出现（桃狐 or 衡猩）· 接食动画 |
| 派出 | 主接引人送你的种子上路 · 口头禅一嗓 |
| 共鸣 | 主接引人出现在共鸣动画里 |

### 3.3 副接引人串门

每周 1 天（周三 or 周五 · 季度轮换）· 副接引人串门 24 小时：

- 画面色调暂时切换（星言玩家切桃粉 / 夭夭玩家切星金）
- 副接引人的灵宠也来（衡猩 or 桃狐）
- 带一段另一卷（千里江山 ↔ 清明上河）的剧情线索
- **这是**双接引人双灵宠**的日常曝光机制**——保证全体玩家对两对 IP 都有亲密度

### 3.4 主接引人是否可换

**可以** · 但高代价：
- 每年一次机会
- 换后当前季度未解锁切片作废
- 原主接引人留下"画壁告别信"一封（永久保留）
- 换后 30 日 · 双接引人共同陪伴过渡

## 四 · 10 分钟日循环（五步）

### 4.1 冥想（1 min）

- 睁 / 闭眼均可 · 不可跳过 · 可随时关 app（退 = 今日冥想失败）
- 屏幕中央灵机种子缓慢脉动
- 背景音：主接引人极轻哼调 + 0.2913Hz 低频环境（胎儿呼吸原型）
- 60 秒倒计时不显示数字 · 光圈渐变
- 完成：心神 M +1 · 灵机 L 微增
- 检测到频繁操作（摇晃 / 锁屏）→ "心不在"不计入也不罚（温柔哲学）

### 4.2 抉择（2 min）

- 每日从 **200+ 切片池**抽 1 条
- 切片来源：①小说 IP 名场面侧面碎片 ②主接引人引导的支线 ③其他玩家历史抉择的涌现片段
- **不告诉你哪个对**——只累积"你是谁"
- 选择后显示匿名玩家分布（百分比）
- 一致 → α 多涨（一致性奖励）
- 冲突五行 → 种子偏移 · 可能触发人格漂移

### 4.3 喂养（3 min）

**喂谁**：主接引人的灵宠——**夭夭玩家喂桃狐 · 星言玩家喂衡猩**。

**投喂机制**：

- app 在玩家日常轨迹（GPS 粗粒度）随机生成 1-3 枚"灵机结晶"
- 地图上漂浮 · 手指拖动收集
- 喂养：桃狐踮脚接住「唧」一声 / 衡猩陨石态自转吞入「叮」一声

**成长反馈**：

| 成长维度 | 桃狐线 | 衡猩线 |
|---|---|---|
| 亲密度 | 尾尖桃瓣颜色渐深 | 陨石纹星河流动速度加快 |
| 调子解锁 | 桃狐叫声从 3 种 → 10 种 | 衡猩音从 3 种 → 10 种 |
| 形态小变化 | 桃瓣数增加 · 毛色更粉亮 | 机甲关节更精致 · 星环更清晰 |
| 独家动画 | 摇尾 / 蹭脸 / 蹦跳 / 打滚 | 叮当走 / 翻跟斗 / 合体 / 两态切换 |

**灵宠归属铁律**：桃狐 / 衡猩**不属玩家** · 是接引人的。玩家养成的是**亲密度** · 不是所有权。不可交易 · 不可命名（玩家可用昵称显示 · 全局 ID 锁定）。

### 4.4 派出（2 min）

派灵机种子（不是灵宠）出门 · 五类任务：

| 任务 | 形态 | 产出 |
|---|---|---|
| 门派日课 | 帮派系守护地质节点 | 派系贡献 + 少量 LC |
| 画壁巡游 | 进他人副本记录观察 | 涌现切片 |
| 跨城行走 | 被派到其他城市周边 | Journey 式匿名接触 |
| 副本侦察 | 为下次 L3 副本探路 | 副本 buff |
| 双接引人独游 | 让星言 or 夭夭带你种子独游 | 亲密度 · 稀有画壁 |

玩家**不控制**过程 · 离线时间自行经历。回归时 150-300 字日志 + 1-3 张 AI 生图画壁残页。

### 4.5 共鸣（3 min）

| 形式 | 机制 | 时长 |
|---|---|---|
| 同门问候 | 送一缕灵机给同派玩家 | 30s |
| 跨门共振 | 五行相生派短讯互通 | 1min |
| 陌生擦肩 | 系统匹配陌生人交换画壁残片 | 2min |
| 画壁留痕 | 去他人画壁留一笔 · 不改原 · 只追加 | 2min |
| 寂静共坐 | 同公会成员同时上线 · 不说话 · 种子并排闪 60s | 1min |

- 共鸣 R 涨 · L 涨
- 相克形式触发"灵机涌动"——L 涨 +50% 但 S 临时 -1%（24h 恢复）
- **画壁留痕**永久沉积 · 将来可被其他玩家在 L3 副本遇见

## 五 · 人格漂移（长线系统）

### 5.1 定义

60 日以上抉择持续偏离初始五行 → 种子颜色 / 形状 / 声音缓慢变化 · 系统不通知玩家。

### 5.2 价值

- **对抗刷数值**：数值党的种子都长一样 · 漂移党各自演化独特灵魂
- **IP 哲学一致**：灵机 = 不可计算的成分 · 玩家亲身体验"我不是我以为的我"
- **社交展示**：种子形态 = 灵魂签名 · 不可伪造

### 5.3 极端漂移

- 偏离到原五行对面 · 累计 180 日 → 跃迁事件（换派 or 强制回归）
- 所有漂移可逆 · 但不清零（像人的记忆）

## 六 · 五行相生相克

### 6.1 相生链

```
木 → 火 → 土 → 金 → 水 → 木
（共存 → 通灵 → 观象 → 通宝 → 织天 → 共存）
```

相生派共鸣 → L +20% · 结晶互赠无损耗 · 可建"相生公会"。

### 6.2 相克链

相克派共鸣 → L +50% / S -1%（24h）。L2 对抗自动升级"灵机风暴"赛制。

### 6.3 五行混编

平衡公会（五行全配齐 5 人）→ 激活"五行轮回"buff · 每日冥想时长减半效果翻倍。

## 七 · 经济（L1 部分）

### 7.1 三种资源

| 名称 | 产出 | 用途 | 可交易 |
|---|---|---|---|
| 灵机结晶 | 日常轨迹 | 喂养 / 强化 | 限额可 |
| 画壁残片 | 灵宠派出 + 副本 | 展示 / 装饰 | 自由可 |
| 灵机残片（稀有） | 随机掉落 | 信标印 / 种子突破 | **不可**（铁律） |

### 7.2 跨级流通

- L1 产出 → L2/L3 消耗
- L2/L3 产出 → L1 稀有化
- 灵机残片不可交易——小说铁律直译

## 八 · 双接引人双灵宠在 L1 的爆款钩子

### 8.1 每日高感质触发点

| 时刻 | 主接引人 | 稀缺度 |
|---|---|---|
| 入画首日 | 主接引人第一次抬头看你 · 一嗓口头禅 | 一生一次 |
| Day 7 | 主接引人第一次单独对你说话（短语弹幕式） | 一生一次 |
| Day 30 | 灵宠"亲密度"突破一阶 · 解锁独家蹭脸动画 | 一生一次 |
| Day 60 | 副接引人首次上门 24h · 双灵宠同屏 | 首次一次 |
| Day 90 | 灵宠变色 / 新形态 · 桃狐尾瓣增 1 · 衡猩星环更清 | 一次 |
| Day 180 | 主接引人"生日短信"——玩家生日当天的个性化信件 | 每年一次 |
| Day 365 | 周年庆 · 主接引人真身出现 · 桃狐或衡猩"合照"AI 生图 · 可导出 | 每年一次 |

### 8.2 大招稀缺释放

- **桃狐"生机绽放"**：L1 仅在玩家连续 7 日低情绪抉择后 · 主接引人为夭夭的玩家 · 见一次
- **衡猩"吞噬星河"**：L1 仅在玩家跨级首次进 L3 副本前夜 · 主接引人为星言的玩家 · 见一次
- 两大招在 L1 都是**一生仅见一次** · 保留稀缺性到 L3

### 8.3 双接引人口头禅位置

- 抉择画面底部：**星言**玩家见「看星！」/ **夭夭**玩家见「夭呀！」· 作为弹幕装饰
- 切换主接引人后口头禅随之更换
- 玩家可手动关闭（保护无打扰）· 默认开启

### 8.4 双灵宠短视频资产

- `#我的桃狐` / `#我的衡猩`——玩家每日与灵宠互动录屏
- `#双灵宠合体` / `#生机遇见星河`——副接引人串门日的双灵宠合照
- 生日之歌 / 周年合照 AI 生图可一键分享
- 首年 TikTok/抖音/小红书投放预算 ¥3000 万 · 目标曝光 10 亿

## 十 · L1 爆款顶点（10 个）

1. **入画首日**——主接引人第一嗓「看星！」/「夭呀！」· 灵宠蹲你手心
2. **Day 30 亲密度一阶**——桃狐第一次在屏幕上蹭出 3 秒漫天花瓣 / 衡猩第一次陨石态转机甲态
3. **副接引人首次上门**——屏幕色调换 · 双灵宠同屏 · 一整天双线剧情
4. **人格漂移自我发现**——"咦 · 我的灵机种子怎么变色了？"
5. **画壁留痕被陌生人看到**——三个月前的笔迹今天被广州玩家走过
6. **Journey 式陌生擦肩**——两个种子无语言停 60 秒 · 各自走
7. **五行轮回激活**——5 人公会首次集齐 · 全屏微震
8. **稀有灵机残片首次掉落**——金光 10 秒 · 种子短暂发烫
9. **生日主接引人短信**——一段个性化文字 · 玩家生日当天
10. **周年合照**——主接引人 + 灵宠 + 玩家种子 AI 生图 · 可导出打印

每个顶点都指向一件事：**"我被看见了。"**

## 十一 · 风险与对策

| 风险 | 应对 |
|---|---|
| 日循环 10 分钟太短 | 反过来：10 分钟高质量 > 2 小时低质量 |
| 抉择被"秒选刷" | 触发"心不在"检测 · 不计入 α |
| 人格漂移迷失身份 | 极端触发"回归"剧情 · 可逆 |
| 机器人刷数值 | 灵机种子真随机熵 · 僵尸种子识别排除 |
| 双接引人分流注意力 | 主接引人锁定 · 副接引人每周 1 天串门 · 强制曝光双 IP |
| 灵宠被过度商业化 | 铁律：不卖本尊 · 只卖周边 / 皮肤 / 玩偶 |
