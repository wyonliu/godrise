## 核心公式 v2.0

### 基础能量公式

$$E = S^{\alpha} \times (P + M) \times L \times \frac{1}{1 - R}$$

**变量定义**：
| 符号 | 含义 | 范围 | 单位 |
|------|------|------|------|
| E | 能量输出 | 0 ~ ∞ | 焦耳当量（JE） |
| S | 魂（Soul） | 0 ~ 100 | 无量纲 |
| α | 灵机指数 | 1.0 ~ 3.0 | 无量纲 |
| P | 体力（Physical） | 0 ~ ∞ | 体力值 |
| M | 法力（Magic） | 0 ~ ∞ | 法力值 |
| L | 灵力（Spiritual） | 0 ~ ∞ | 灵力值 |
| R | 共鸣度（Resonance） | -1.0 ~ +1.0 | 无量纲 |

**公式解读**：
- **魂**：决定能量的"质量"，是指数底数
- **灵机指数**：放大魂的效果，修行越高指数越大
- **体力+法力**：决定能量的"承载上限"
- **灵力**：决定能量的"燃料储备"
- **共鸣**：放大倍率，100%时触发"神临"

### 修行曲线（S曲线模型）

**身体素质与能力运用的关系**：

```
能力运用效率 = 1 / (1 + e^(-k(修行年限 - 临界点)))

其中：
- k = 成长速率（天赋决定）
- 临界点 = 突破节点（境界划分）
```

**S曲线的三个阶段**：

1. **蓄势期**（低斜率）
   - 修行初期，进步缓慢
   - 需要大量基础训练
   - 类似运动员的基础体能训练

2. **爆发期**（高斜率）
   - 突破瓶颈后快速提升
   - 灵机觉醒带来质变
   - 类似运动员的技术突破期

3. **瓶颈期**（低斜率）
   - 接近境界上限时放缓
   - 需要更高层次的悟道
   - 类似顶级运动员的微调期

**修行速度约束**：修行速度必须保守。参照韩立200年才元婴，本作灵婴境同样需要漫长积累。不可出现主角短期内连跳数境的情节。

**五段制+半步（v3确认）**：每一境界可分五段：初期→中期→后期→巅峰→圆满。极个别天才可达"半步下一境"——未正式突破但战力可对抗下一境初期。灵婴境在第二部（2026-2126）不存在，2126年人类仅有极个别半步灵婴（米莱+1-2人），真正灵婴需第三部方可出现。

## 一、魂：人类核心

### 定义
**魂 = 赤诚之心（追寻自我的热烈忠诚）+ 混沌意志（情感非理性的变数选择）**

人类个体的非理性超级能量之源,是文明负熵本源。

> **关键约束**：因果推理的尽头是**非理性**（不是直觉！）。混沌意志的本质是非理性选择，不可用"直觉"替代。

---

**定义**：对自我真实情感的绝对忠诚。

**特征**：
- 追求真理、爱或记忆时不计代价
- 不问损益的纯粹状态
- 能量动力源的最高等级
- 能触动宇宙底层的重启协议

**硬核解释**：
```
量子力学角度:
→ 观察者的"坚定意图"会影响量子坍缩方向
→ "赤诚之心"就是"意图纯度"的极致
→ 纯度越高,对现实的干预能力越强

信息论角度:
→ 一个完全忠于自我的选择 = 零信息损耗
→ 无杂质的信号 = 最高传输效率
→ 可以在嘈杂的宇宙背景中被精准接收
```

**剧情应用**：
```

```

### 1.2 混沌意志

**定义**：对应量子力学中的非确定性,在面对算法给出的最优解时,人类会因非理性或情感选择逻辑差解甚至自毁解。

**特征**：
- 无法被预测的跳跃
- 宇宙中唯一能产生新变量的瞬间
- AI眼中的"Bug",人类眼中的"奇迹"

**硬核解释**：
```python
# AI的决策树(确定性)
def AI_choice(situation):
    outcomes = calculate_all_outcomes(situation)
    return max(outcomes, key=lambda x: x.expected_value)

# 人类的混沌意志(非确定性)
def human_choice(situation):
    outcomes = sense_possibilities(situation)
    intuition = quantum_random()  # 真随机
    emotion = current_emotional_state()
    if emotion == "love":
        return outcomes.most_irrational  # 可能是最差选择
    elif intuition > 0.9:
        return outcomes.unexpected  # 算法无法预测
    else:
        return outcomes.balanced
```

**案例**：
```
经典场景(第一部):
爱音塔第99层,主角面对AI的"电车难题":
- 选A: 救100个陌生人,牺牲挚友
- 选B: 救挚友,100人死亡

AI预测: 主角会选A(功利主义最优解)
主角实际: "我全都要!"
→ 选择用自己的生命换取系统漏洞
→ 触发系统未定义的"自我牺牲"分支
→ AI系统瞬间宕机(逻辑溢出)
→ 所有人获救
```

## 五、共鸣：灵机倍率

### 定义
同一瞬间的魂、灵机、灵力、法力、武器、药效的谐振导致能量放大的倍率。

**公式**：
$$\text{共鸣} = \sum (\text{五行匹配度} + \text{心意契合度} + \text{时机精准度})$$

**范围**：-∞ 到 +100%

---

### 共鸣值计算

$$R = \sum_{i=1}^{n} w_i \times \text{匹配度}_i$$

**匹配维度**：
| 维度 | 权重(w) | 说明 |
|------|---------|------|
| 五行相生 | 0.3 | 团队属性配合 |
| 心意契合 | 0.4 | 情感同步程度 |
| 时机精准 | 0.2 | 量子相干态同步 |
| 武器共振 | 0.1 | 装备灵器匹配 |

**共鸣放大表**：
| 共鸣值 | 放大倍率 | 效果 |
|--------|----------|------|
| 0% | 1x | 无放大 |
| 50% | 2x | 双人配合 |
| 80% | 5x | 精英小队 |
| 95% | 20x | 千人阵法 |
| 99% | 100x | 万人共鸣 |
| 99.9% | 1000x | 百万共鸣 |
| 100% | ∞ | 神临（触发维度突破） |

---

### 共鸣机制

#### 正共鸣（增益）
```
条件:
1. 五行属性相生
2. 参与者心意相通
3. 时机精准(量子相干态同步)

效果:
共鸣每提升10% → 能量输出翻倍
共鸣达到100% → 能量输出无限 → 触发"神临"
```

#### 负共鸣（削弱）
```
条件:
1. 五行属性相克
2. 参与者互相猜疑
3. 时机混乱(量子退相干)

效果:
共鸣每降低10% → 能量输出减半
共鸣低于-50% → 技能反噬
```

---

### 神临：共鸣100%的终极形态

**定义**：当共鸣达到+100%时,能量释放公式的分母→0,触发**跨维度因果打击**。

**表现**：
```
视觉: 天空裂开,金色光柱贯穿天地

效果: 可以召唤"高维意识"降临

硬核解释:
→ 100%共鸣 = 所有参与者的量子态完全纠缠
→ 形成一个"宏观量子相干体"
→ 这个相干体的信息密度极高
→ 高到可以"撑开"维度壁垒
→ 让高维存在(神)可以短暂降临低维(现实)
```

**案例**（第三部）：
```
人类vs高等文明最终战:

效果:

→ 震动了N-1层,甚至N层
```
