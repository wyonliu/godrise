## 一、七灵武道

基于佛学"眼、耳、鼻、舌、身、意、空"七根性的战斗技艺体系。七灵武道不依赖灵力消耗，而是通过**身体本能**和**感知强化**实现战斗效果，与灵机修行互补。

---

### 1.1 七道总览

| 武道 | 根性 | 核心 | 三阶 | 应用场景 |
|------|------|------|------|----------|
| **悉见道** | 眼 | 预测 | 捕风→先演→宿命 | 战场预判、伏击侦察 |
| **瞬步道** | 意 | 瞬移 | 闪步→跃迁→神游 | 闪避、突袭、逃生 |
| **磐石道** | 身 | 硬化 | 铁骨→原合→金身 | 防御、扛伤、近战 |
| **贯虹道** | 鼻 | 突击 | 锁踪→刺空→灭因 | 刺杀、穿透、破阵 |
| **无相道** | 舌 | 卸力 | 止水→吸星→归零 | 反弹、消解、化劲 |
| **凌虚道** | 空 | 御空 | 悬空→翔宇→空相 | 飞行、空战、灵活机动 |
| **如梦道** | 耳 | 催眠 | 乱神→念物→梦蝶 | 幻术、控场、情报 |

---

### 1.2 各道详解

#### 悉见道（眼/预测）

**核心原理**：通过强化视觉皮层与前额叶的量子相干，预测敌人行动轨迹。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **捕风** | 预判敌人下0.1秒动作 | 观察落叶飘动规律 |
| 中阶 | **先演** | 预判敌人下3秒动作链 | 与多人对战中保持不被击中 |
| 高阶 | **宿命** | 预见战局走向，锁定必杀时机 | 冥想中预演万种可能 |

**剧情应用**（第一部）：
```
爱音塔第30层，麦洛首次觉醒捕风：
→ 敌人挥刀前0.1秒，她已经开始闪避
→ 敌人以为自己刀法快，其实是被预判
→ 代价：持续消耗精神力，使用过度会剧烈头痛
```

**硬核依据**：
```
神经科学：
→ 人类眼球每秒进行3-4次微跳视
→ 大脑在0.1秒内处理视觉信息并预判
→ 悉见道 = 将这个能力极限强化
```

---

#### 瞬步道（意/瞬移）

**核心原理**：通过意识先行到达目标位置，引导身体快速跟进。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **闪步** | 短距离高速移动（5米内） | 在黑暗中凭借听觉定位移动 |
| 中阶 | **跃迁** | 瞬间位移（50米内） | 在意识中构建空间地图 |
| 高阶 | **神游** | 意识可脱离身体探查（1公里） | 深度冥想中意识外游 |

**剧情应用**（第一部）：
```
清明上河篇，麦洛修炼闪步：
→ 师傅蒙上她的眼睛，只靠听觉定位
→ 在竹林中穿梭，不触碰任何一根竹子
→ 当她能"听到"竹子时，闪步觉醒
```

**与灵机修行的配合**：
```
灵核境 + 跃迁 = 可以携带灵机在位移中发动攻击
化灵境 + 神游 = 意识探查时可以释放灵机干扰敌人
```

---

#### 磐石道（身/硬化）

**核心原理**：通过调控肌肉纤维的排列密度，提升身体硬度。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **铁骨** | 皮肤硬度达到钢铁级别 | 铁砂掌、抗击打训练 |
| 中阶 | **原合** | 身体可抵抗子弹/激光 | 在极端环境中生存训练 |
| 高阶 | **金身** | 物理攻击近乎免疫 | 灵机淬体，重塑骨骼结构 |

**剧情应用**（第三部）：
```
人类vs高等文明，"承载者军团"：
→ 1000名磐石道修行者组成人墙
→ 承受相当于核爆的能量轰击
→ 无人倒下，全员进入"原合"状态
→ 高等文明震惊："这不符合物理定律！"
```

**代价**：
```
磐石道的副作用：
→ 长期使用会导致身体逐渐失去柔韧性
→ 极端情况下可能"石化"
→ 需要配合水灵力进行软化调节
```

---

#### 贯虹道（鼻/突击）

**核心原理**：通过锁定敌人的气息/能量特征，发动必中的穿透攻击。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **锁踪** | 锁定敌人位置（100米内） | 蒙眼追踪训练 |
| 中阶 | **刺空** | 穿透物理护盾的攻击 | 理解物质的分子间隙 |
| 高阶 | **灭因** | 攻击直达敌人"因果源头" | 融入金灵力的解析能力 |

**剧情应用**（第二部）：
```
船长的刺杀任务：
→ 目标藏在全息投影护盾后
→ 船长闭眼，感知目标的"呼吸节律"
→ 一击穿透护盾，直达心脏
→ 敌人甚至不知道自己已经死了
```

**硬核依据**：
```
物理学：
→ 任何物质都有分子间隙
→ 贯虹道 = 让攻击从间隙中穿过
→ 灭因阶 = 攻击的是敌人"存在的因果"
```

---

#### 无相道（舌/卸力）

**核心原理**：通过"品尝"攻击的能量特征，将其化解或反弹。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **止水** | 化解物理攻击 | 太极推手、柔术训练 |
| 中阶 | **吸星** | 吸收敌人灵机攻击 | 感知并"品尝"能量流 |
| 高阶 | **归零** | 将任何攻击归于虚无 | 理解"空"的本质 |

**剧情应用**（第一部）：
```
爱音塔第70层，麦洛vs火属性Boss：
→ Boss释放火焰风暴
→ 麦洛使用止水，将火焰引导到地面
→ "你的火，只是在换个地方燃烧"
```

**哲学根基**：
```
道德经：
→ "上善若水，水善利万物而不争"
→ 无相道 = 不与攻击正面对抗
→ 顺势而为，以柔克刚
```

---

#### 凌虚道（空/御空）

**核心原理**：通过操控身体周围的气场，实现飞行和机动。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **悬空** | 短暂悬浮（30秒） | 轻功训练、减轻重量感知 |
| 中阶 | **翔宇** | 自由飞行（1小时） | 理解气流与重力 |
| 高阶 | **空相** | 在虚空中创造落脚点 | 灵机凝实，创造"虚实锚点" |

**剧情应用**（第三部）：
```
太空战斗场景：
→ 修行者在真空中战斗
→ 凌虚道修行者可以"踩着虚空"机动
→ 不需要推进器，靠灵机创造反作用力
```

**与瞬步道的区别**：
```
瞬步道 = 点对点位移（快但消耗大）
凌虚道 = 持续机动（慢但续航长）
高手会两者配合使用
```

---

#### 如梦道（耳/催眠）

**核心原理**：通过特定频率的声波/念力影响敌人的意识。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **乱神** | 干扰敌人思维（3秒混乱） | 学习多种语言的韵律 |
| 中阶 | **念物** | 植入虚假记忆/幻觉 | 深入理解人类心理 |
| 高阶 | **梦蝶** | 将敌人困在梦境中 | 能在他人梦境中行走 |

**剧情应用**（第二部）：
```
情报战场景：
→ 如梦道高手伪装成敌人的同伴
→ 植入"你应该相信我"的念头
→ 获取核心机密后悄然离开
→ 敌人醒来后完全不记得发生了什么
```

**危险性**：
```
如梦道的禁忌：
→ 不可对同伴使用（违者逐出门派）
→ 过度使用会导致自己分不清梦与现实
→ 最终可能迷失在"庄周梦蝶"的悖论中
```

### 1.3 七道组合技

| 组合 | 效果 | 适用场景 |
|------|------|----------|
| 悉见 + 瞬步 | 预判后瞬移反击 | 1v1决斗 |
| 磐石 + 贯虹 | 硬扛伤害后一击必杀 | 近战肉搏 |
| 无相 + 如梦 | 化解攻击同时迷惑敌人 | 控制场面 |
| 凌虚 + 悉见 | 空中预判，全局视野 | 战场指挥 |
| 七道齐修 | 极少数天才，全能战士 | 传说级别 |

**剧情锚点**：
```
麦洛的成长路径：
第一部：主修悉见道（预判），辅修瞬步道（闪避）
第二部：增加磐石道（抗击打），贯虹道（攻击）
第三部：融合无相道（卸力），凌虚道（飞行）
第四部：触及如梦道（催眠），成为"七道齐修"的传说

船长的路径：
主修贯虹道（刺杀）+ 悉见道（预判）
辅修如梦道（情报战）
不擅长磐石道（他选择避免正面硬扛）
```

### 1.4 七灵武道 vs 灵机修行

| 维度 | 七灵武道 | 灵机修行 |
|------|----------|----------|
| 能量来源 | 身体本能、感知 | 情感、灵力 |
| 消耗 | 精神力、体力 | 灵力储备 |
| 训练方式 | 身体锻炼、感官强化 | 冥想、情感体验 |
| 上限 | 受身体极限制约 | 理论上无上限 |
| 互补关系 | 快速反应、续航 | 爆发伤害、改变法则 |

**关键点**：
- 七灵武道是**战斗技术**，灵机修行是**能量体系**
- 顶级战士两者兼修
- 纯粹的灵机修行者容易被近战专家克制
- 纯粹的武道修行者无法对抗高阶灵机攻击

## 三、战斗演示：三种战斗风格

### 最终公式
$$\text{能量释放} = \frac{\text{魂}^{(\text{灵机})} \times (\text{体力} + \text{法力}) \times \text{灵力}}{1 - \text{共鸣}}$$

> 完整公式与变量定义见 [power_system.md](./power_system.md)

#### 场景：对抗同一个敌人（能量值=10000）

---

**风格A：火力全开（火灵力主导）**
```
参数:
魂 = 100 (赤诚之心饱满)
灵机 = 2 (魂的指数)
体力 = 500 (合械体)
法力 = 3级 (引力/电磁)
灵力 = 1000 (火属性)
共鸣 = 50% (单人,无队友配合)

计算:
能量 = (100^2 × (500 + 1000) × 1000) / (1 - 0.5)
     = (10000 × 1500 × 1000) / 0.5
     = 15000000000 / 0.5
     = 3 × 10^10 焦耳

结果: 瞬间秒杀敌人
代价: 体力透支,昏迷三天
```

---

**风格B：持久战（土灵力主导）**
```
参数:
魂 = 80 (意志坚定,但不极致)
灵机 = 1.5
体力 = 2000 (共相体,灵机强化)
法力 = 2级 (念动力)
灵力 = 5000 (土属性,储备深厚)
共鸣 = 20% (防御姿态,共鸣较低)

计算:
能量 = (80^1.5 × (2000 + 100) × 5000) / (1 - 0.2)
     = (715 × 2100 × 5000) / 0.8
     = 7507500000 / 0.8
     = 9.4 × 10^9 焦耳

结果: 无法秒杀,但可以持续输出10小时
代价: 灵力缓慢消耗,无昏迷风险
```

---

**风格C：团队共鸣（五行配合）**
```
参数:
5人团队,每人:
魂 = 60 (不如单人极致,但心意相通)
灵机 = 1.8
体力 = 1000 (共相体)
法力 = 3级
灵力 = 2000 (五行齐全:木火土金水各400)
共鸣 = 90% (五行相生,心意契合)

单人能量 = (60^1.8 × (1000 + 1000) × 2000) / (1 - 0.9)
         = (约300 × 2000 × 2000) / 0.1
         = 1.2 × 10^9 / 0.1
         = 1.2 × 10^10 焦耳

5人总和 = 5 × 1.2 × 10^10 = 6 × 10^10 焦耳

结果: 能量是风格A的2倍
代价: 5人分担,无人昏迷
额外: 五行循环,产生持续增益效果
```

## 五、写作锚点

### 每次描写修行/战斗时,必须明确：
1. **当前境界**：角色处于哪个修炼阶段
2. **灵力消耗**：每个技能的成本
3. **物理代价**：透支的后果(昏迷/寿命/细胞损伤)
4. **五行属性**：技能的属性及相生相克
5. **共鸣状态**：是单人作战还是团队配合

### 禁止出现：
- 不可 "神秘力量突然涌现"
- 不可 无限连招不消耗
- 不可 主角光环式的"越级挑战"

### 必须呈现：
- 每次突破都有明确的训练过程
- 战斗后的疲惫/受伤
- 灵力储备的管理(类似MP条)
- 团队配合的重要性(1+1>2)
