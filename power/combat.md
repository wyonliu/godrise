## 一、七灵武道

基于佛学"眼、耳、鼻、舌、身、意、空"七根性的战斗技艺体系。七灵武道不依赖灵力消耗，而是通过**身体本能**和**感知强化**实现战斗效果，与灵机修行互补。

---

### 1.1 七道总览

| 武道 | 根性 | 核心 | 三阶 | 应用场景 |
|------|------|------|------|----------|
| **悉见道** | 眼 | 预测 | 捕风→先演→宿命 | 战场预判、伏击侦察 |
| **瞬步道** | 意 | 瞬移 | 闪步→跃迁→神游 | 闪避、突袭、逃生 |
| **磐石道** | 身 | 硬化 | 铁骨→原合→金身 | 防御、扛伤、近战 |
| **贯虹道** | 鼻 | 突击 | 锁踪→刺空→灭因 | 刺杀、穿透、破阵 |
| **无相道** | 舌 | 卸力 | 止水→吸星→归零 | 反弹、消解、化劲 |
| **凌虚道** | 空 | 御空 | 悬空→翔宇→空相 | 飞行、空战、灵活机动 |
| **如梦道** | 耳 | 催眠 | 乱神→念物→梦蝶 | 幻术、控场、情报 |

---

### 1.2 各道详解

#### 悉见道（眼/预测）

**核心原理**：通过强化视觉皮层与前额叶的量子相干，预测敌人行动轨迹。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **捕风** | 预判敌人下0.1秒动作 | 观察落叶飘动规律 |
| 中阶 | **先演** | 预判敌人下3秒动作链 | 与多人对战中保持不被击中 |
| 高阶 | **宿命** | 预见战局走向，锁定必杀时机 | 冥想中预演万种可能 |

**剧情应用**（第一部）：
```
爱音塔第30层，麦洛首次觉醒捕风：
→ 敌人挥刀前0.1秒，她已经开始闪避
→ 敌人以为自己刀法快，其实是被预判
→ 代价：持续消耗精神力，使用过度会剧烈头痛
```

**硬核依据**：
```
神经科学：
→ 人类眼球每秒进行3-4次微跳视
→ 大脑在0.1秒内处理视觉信息并预判
→ 悉见道 = 将这个能力极限强化
```

---

#### 瞬步道（意/瞬移）

**核心原理**：通过意识先行到达目标位置，引导身体快速跟进。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **闪步** | 短距离高速移动（5米内） | 在黑暗中凭借听觉定位移动 |
| 中阶 | **跃迁** | 瞬间位移（50米内） | 在意识中构建空间地图 |
| 高阶 | **神游** | 意识可脱离身体探查（1公里） | 深度冥想中意识外游 |

**剧情应用**（第一部）：
```
清明上河篇，麦洛修炼闪步：
→ 师傅蒙上她的眼睛，只靠听觉定位
→ 在竹林中穿梭，不触碰任何一根竹子
→ 当她能"听到"竹子时，闪步觉醒
```

**与灵机修行的配合**：
```
灵核境 + 跃迁 = 可以携带灵机在位移中发动攻击
化灵境 + 神游 = 意识探查时可以释放灵机干扰敌人
```

---

#### 磐石道（身/硬化）

**核心原理**：通过调控肌肉纤维的排列密度，提升身体硬度。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **铁骨** | 皮肤硬度达到钢铁级别 | 铁砂掌、抗击打训练 |
| 中阶 | **原合** | 身体可抵抗子弹/激光 | 在极端环境中生存训练 |
| 高阶 | **金身** | 物理攻击近乎免疫 | 灵机淬体，重塑骨骼结构 |

**剧情应用**（第三部）：
```
人类vs高等文明，"承载者军团"：
→ 1000名磐石道修行者组成人墙
→ 承受相当于核爆的能量轰击
→ 无人倒下，全员进入"原合"状态
→ 高等文明震惊："这不符合物理定律！"
```

**代价**：
```
磐石道的副作用：
→ 长期使用会导致身体逐渐失去柔韧性
→ 极端情况下可能"石化"
→ 需要配合水灵力进行软化调节
```

---

#### 贯虹道（鼻/突击）

**核心原理**：通过锁定敌人的气息/能量特征，发动必中的穿透攻击。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **锁踪** | 锁定敌人位置（100米内） | 蒙眼追踪训练 |
| 中阶 | **刺空** | 穿透物理护盾的攻击 | 理解物质的分子间隙 |
| 高阶 | **灭因** | 攻击直达敌人"因果源头" | 融入金灵力的解析能力 |

**剧情应用**（第二部）：
```
船长的刺杀任务：
→ 目标藏在全息投影护盾后
→ 船长闭眼，感知目标的"呼吸节律"
→ 一击穿透护盾，直达心脏
→ 敌人甚至不知道自己已经死了
```

**硬核依据**：
```
物理学：
→ 任何物质都有分子间隙
→ 贯虹道 = 让攻击从间隙中穿过
→ 灭因阶 = 攻击的是敌人"存在的因果"
```

---

#### 无相道（舌/卸力）

**核心原理**：通过"品尝"攻击的能量特征，将其化解或反弹。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **止水** | 化解物理攻击 | 太极推手、柔术训练 |
| 中阶 | **吸星** | 吸收敌人灵机攻击 | 感知并"品尝"能量流 |
| 高阶 | **归零** | 将任何攻击归于虚无 | 理解"空"的本质 |

**剧情应用**（第一部）：
```
爱音塔第70层，麦洛vs火属性Boss：
→ Boss释放火焰风暴
→ 麦洛使用止水，将火焰引导到地面
→ "你的火，只是在换个地方燃烧"
```

**哲学根基**：
```
道德经：
→ "上善若水，水善利万物而不争"
→ 无相道 = 不与攻击正面对抗
→ 顺势而为，以柔克刚
```

---

#### 凌虚道（空/御空）

**核心原理**：通过操控身体周围的气场，实现飞行和机动。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **悬空** | 短暂悬浮（30秒） | 轻功训练、减轻重量感知 |
| 中阶 | **翔宇** | 自由飞行（1小时） | 理解气流与重力 |
| 高阶 | **空相** | 在虚空中创造落脚点 | 灵机凝实，创造"虚实锚点" |

**剧情应用**（第三部）：
```
太空战斗场景：
→ 修行者在真空中战斗
→ 凌虚道修行者可以"踩着虚空"机动
→ 不需要推进器，靠灵机创造反作用力
```

**与瞬步道的区别**：
```
瞬步道 = 点对点位移（快但消耗大）
凌虚道 = 持续机动（慢但续航长）
高手会两者配合使用
```

---

#### 如梦道（耳/催眠）

**核心原理**：通过特定频率的声波/念力影响敌人的意识。

| 阶位 | 名称 | 效果 | 修炼方法 |
|------|------|------|----------|
| 初阶 | **乱神** | 干扰敌人思维（3秒混乱） | 学习多种语言的韵律 |
| 中阶 | **念物** | 植入虚假记忆/幻觉 | 深入理解人类心理 |
| 高阶 | **梦蝶** | 将敌人困在梦境中 | 能在他人梦境中行走 |

**剧情应用**（第二部）：
```
情报战场景：
→ 如梦道高手伪装成敌人的同伴
→ 植入"你应该相信我"的念头
→ 获取核心机密后悄然离开
→ 敌人醒来后完全不记得发生了什么
```

**危险性**：
```
如梦道的禁忌：
→ 不可对同伴使用（违者逐出门派）
→ 过度使用会导致自己分不清梦与现实
→ 最终可能迷失在"庄周梦蝶"的悖论中
```

### 1.3 七道组合技

| 组合 | 效果 | 适用场景 |
|------|------|----------|
| 悉见 + 瞬步 | 预判后瞬移反击 | 1v1决斗 |
| 磐石 + 贯虹 | 硬扛伤害后一击必杀 | 近战肉搏 |
| 无相 + 如梦 | 化解攻击同时迷惑敌人 | 控制场面 |
| 凌虚 + 悉见 | 空中预判，全局视野 | 战场指挥 |
| 七道齐修 | 极少数天才，全能战士 | 传说级别 |

**剧情锚点**：
```
麦洛的成长路径：
第一部：主修悉见道（预判），辅修瞬步道（闪避）
第二部：增加磐石道（抗击打），贯虹道（攻击）
第三部：融合无相道（卸力），凌虚道（飞行）
第四部：触及如梦道（催眠），成为"七道齐修"的传说

船长的路径：
主修贯虹道（刺杀）+ 悉见道（预判）
辅修如梦道（情报战）
不擅长磐石道（他选择避免正面硬扛）
```

### 1.4 七灵武道 vs 灵机修行

| 维度 | 七灵武道 | 灵机修行 |
|------|----------|----------|
| 能量来源 | 身体本能、感知 | 情感、灵力 |
| 消耗 | 精神力、体力 | 灵力储备 |
| 训练方式 | 身体锻炼、感官强化 | 冥想、情感体验 |
| 上限 | 受身体极限制约 | 理论上无上限 |
| 互补关系 | 快速反应、续航 | 爆发伤害、改变法则 |

**关键点**：
- 七灵武道是**战斗技术**，灵机修行是**能量体系**
- 顶级战士两者兼修
- 纯粹的灵机修行者容易被近战专家克制
- 纯粹的武道修行者无法对抗高阶灵机攻击

## 三、战斗演示：三种战斗风格

### 最终公式
$$\text{能量释放} = \frac{\text{魂}^{(\text{灵机})} \times (\text{体力} + \text{法力}) \times \text{灵力}}{1 - \text{共鸣}}$$

> 完整公式与变量定义见 [power_system.md](./power_system.md)

#### 场景：对抗同一个敌人（能量值=10000）

---

**风格A：火力全开（火灵力主导）**
```
参数:
魂 = 100 (赤诚之心饱满)
灵机 = 2 (魂的指数)
体力 = 500 (合械体)
法力 = 3级 (引力/电磁)
灵力 = 1000 (火属性)
共鸣 = 50% (单人,无队友配合)

计算:
能量 = (100^2 × (500 + 1000) × 1000) / (1 - 0.5)
     = (10000 × 1500 × 1000) / 0.5
     = 15000000000 / 0.5
     = 3 × 10^10 焦耳

结果: 瞬间秒杀敌人
代价: 体力透支,昏迷三天
```

---

**风格B：持久战（土灵力主导）**
```
参数:
魂 = 80 (意志坚定,但不极致)
灵机 = 1.5
体力 = 2000 (共相体,灵机强化)
法力 = 2级 (念动力)
灵力 = 5000 (土属性,储备深厚)
共鸣 = 20% (防御姿态,共鸣较低)

计算:
能量 = (80^1.5 × (2000 + 100) × 5000) / (1 - 0.2)
     = (715 × 2100 × 5000) / 0.8
     = 7507500000 / 0.8
     = 9.4 × 10^9 焦耳

结果: 无法秒杀,但可以持续输出10小时
代价: 灵力缓慢消耗,无昏迷风险
```

---

**风格C：团队共鸣（五行配合）**
```
参数:
5人团队,每人:
魂 = 60 (不如单人极致,但心意相通)
灵机 = 1.8
体力 = 1000 (共相体)
法力 = 3级
灵力 = 2000 (五行齐全:木火土金水各400)
共鸣 = 90% (五行相生,心意契合)

单人能量 = (60^1.8 × (1000 + 1000) × 2000) / (1 - 0.9)
         = (约300 × 2000 × 2000) / 0.1
         = 1.2 × 10^9 / 0.1
         = 1.2 × 10^10 焦耳

5人总和 = 5 × 1.2 × 10^10 = 6 × 10^10 焦耳

结果: 能量是风格A的2倍
代价: 5人分担,无人昏迷
额外: 五行循环,产生持续增益效果
```

## 五、写作锚点

### 每次描写修行/战斗时,必须明确：
1. **当前境界**：角色处于哪个修炼阶段
2. **灵力消耗**：每个技能的成本
3. **物理代价**：透支的后果(昏迷/寿命/细胞损伤)
4. **五行属性**：技能的属性及相生相克
5. **共鸣状态**：是单人作战还是团队配合

### 禁止出现：
- 不可 "神秘力量突然涌现"
- 不可 无限连招不消耗
- 不可 主角光环式的"越级挑战"

### 必须呈现：
- 每次突破都有明确的训练过程
- 战斗后的疲惫/受伤
- 灵力储备的管理(类似MP条)
- 团队配合的重要性(1+1>2)

## 六、第二部关键战斗编排

> 三场战斗覆盖第二部三个能量级：灵感实战（步兵排级）、灵核突破（重装旅级）、爱音塔物理阻碍（文明级同步工程）。
> 每场战斗设定包含：参战者、物理约束、七灵武道运用、感官层次、代价体系、叙事功能。
> 绝对铁律：三战(2056-2059)天花板=灵根后期，灵核第一人2065后，灵婴不存在于第二部。

---

### 6.1 三战中的修行者实战（2057年·卷六）

**对标**：《进击的巨人》立体机动装置作战——科技+身体极限的结合

#### 参战者

| 代号 | 武道主修 | 灵感境阶位 | 角色功能 |
|------|---------|-----------|---------|
| 赵铁生 | 磐石道·铁骨 + 贯虹道·锁踪 | 后期（CPI≈2.8） | 前锋·硬扛火力 |
| 林小满 | 如梦道·乱神 + 悉见道·捕风 | 中期（CPI≈2.3） | 通信干扰·预警 |
| 乌兰 | 凌虚道·悬空 + 瞬步道·闪步 | 后期（CPI≈2.7） | 侦察·机动 |
| 代号「针」 | 贯虹道·锁踪 + 磐石道·铁骨 | 巅峰（CPI≈3.0） | 突击·穿甲 |
| 代号「镜」 | 悉见道·先演 + 无相道·止水 | 中期（CPI≈2.5） | 战术指挥·卸力掩护 |

**对手**：硅方者代理机械兵团——12台T-4型自律步兵（单台等效CPI≈2.0，但无疲劳、无恐惧、精确到毫米的火力覆盖）。数量优势+零犯错率。人类唯一优势=灵机感知（机械无法探测灵机场波动）+七灵武道的非线性判断。

#### 境界天花板

灵感后期（Q3=10^2.5-10^3 JE）。全队最强个体「针」处于灵感巅峰，但她的灵机储量仅够全力输出7次——第8次会灵机枯竭。没有任何人达到灵根境。

#### 物理约束

- **环境**：西非萨赫勒残垣——被战争夷平的城市边缘，碎石、残墙、半塌楼房。气温-12℃，无遮蔽

### 6.3 爱音塔之战的物理阻碍（2126年·卷八）

**对标**：EVA·补完仪式的阶段递进
**这不是大战场面，而是50万人同步过程中遇到的3个技术危机。**

#### 危机一：相位失谐（同步70%时·听觉主导）

**时间节点**：同步从23%攀升至70%的加速阶段。东南亚区域（约3.2万名修行者）集体失去相位。

**物理机制**：
- 50万修行者的灵机场以0.2913Hz为基准频率进行相位同步（0.2913Hz=星霜自灵化时的残余振动频率，跨越91年的音乐主题）
- 东南亚区域的修行者群体中，约60%为灵感境初中期（CPI 2.0-2.5），灵机场的相干时间仅维持微秒级——在全球同步的高功率需求下，微秒级相干不足以锁定相位
- 3.2万人的灵机场从0.2913Hz的锁定态滑落为0.28-0.31Hz的随机分布——频率差虽小（±4%），但50万人系统中的3.2万人失谐=整体共鸣度从70%骤降至61%

**感官设计（听觉）**：
- 同步过程中，在场修行者的灵感境感知将全球灵机场的共振"翻译"为一种极低频的和声——不是耳朵听到的声音，是颅骨传导的骨振动
- 当东南亚区域失谐时，这个和声中突然出现一个"刺耳的泛音"——频率失谐产生的拍频（beat frequency）约为0.01Hz，周期约100秒。感觉像是有人在你的颅骨内部用指甲刮黑板，每100秒刮一次
- 在场的灵核境修行者描述更精确："不是刺耳——是'走音'。像一个交响乐团中有30个人突然开始演奏不同的曲子"

**米莱的调整**：
- 米莱（灵核圆满·CPI≈8.2）作为同步总指挥，不能直接用自己的灵机场"压"住3.2万人——那会导致他们的灵机场被覆盖而非同步（压制≠共鸣）
- 解法：米莱启动"降频桥接"——她将自己的灵机场从0.2913Hz降至0.2850Hz（东南亚区域的平均偏移频率），先与失谐区域同步，再以每秒0.0001Hz的速率将自己和这3.2万人一起拉回0.2913Hz。全过程耗时约63秒
- 代价：降频期间米莱自身的灵核也进入亚稳态，灵机日产出在之后48小时内下降约30%

#### 危机二：节点过载（收割程序第二波冲击中·触觉主导）

**时间节点**：同步82%。收割程序的第二波冲击以信息层面的"格式化脉冲"扫过地球，灵机方舟的7号中继节点（位于南太平洋海底锚定站）承受超设计载荷。

**物理机制**：
- 灵机方舟是2100-2125年间建造的全球灵机中继网络，由12个节点组成，每个节点是一个深海锚定的量子共振腔
- 7号节点负责南半球32%的灵机场中继。收割程序的格式化脉冲本质是对宇宙底层信息结构的暴力擦写——7号节点的量子共振腔在试图"翻译"并中和这个脉冲时，内部温度从设计工作温度4K（接近绝对零度）骤升至47K
- 47K对常规设备不算高温，但对量子共振腔而言=退相干灾难。腔内量子比特的退相干时间从设计值1毫秒降至3微秒——无法维持中继功能

**工程师视角的抢修**：
- 驻站工程师：程允白的学生（第三代技术人员），名叫韩溪（韩彦白的孙女，35岁，非修行者）

- 韩溪选B。但在伸手触碰外壳释放扣之前，她停了三秒。这三秒内她做了工程师会做的事：读取退相干场泄漏数据（外壳显示面板上实时跳动的数字）、心算量子退相干场对生物组织的等效辐射当量（她是程允白的学生，这个计算她能做）、得出结论——触觉神经损伤概率>80%，不可逆概率约60%。三秒。然后她打开了外壳。打开外壳的瞬间，退相干场对她手臂皮肤的影响=表皮细胞的量子态被强制退相干，等效为一种"信息级灼伤"——皮肤没有热损伤，但细胞间的信号传递被打乱。她的双手从手腕到指尖完全失去触觉

#### 危机三：最后0.1%——船长的选择（同步99.9%→100%·寂静主导）

**时间节点**：同步99.9%。50万人中499,500人已完成相位锁定。剩余0.1%=500人的等效灵机缺口——但这个缺口不是人数问题，是能量密度问题。

**物理机制**：
- 收割程序的阈值是文明级灵机共鸣突破某个临界值。99.9%的同步已经产生了接近阈值的共鸣场，但"接近"不够——差0.1%的能量缺口约等于10^7 JE（一个灵核境巅峰修行者的全力输出）
- 船长在2126年已散逸49年。他的物理身体处于量子叠加态（活/死叠加），灵机场分散在山海量子晶格层中。但灵核仍在——只是不再凝聚于一个身体
- 填补0.1%缺口=船长将散逸态的灵机场重新凝聚为一次性脉冲输出。这意味着他必须"选择"坍缩——从叠加态（活/死）坍缩为确定态（死）

**散逸的能量释放·物理描述**：
- 船长的灵机场从山海量子晶格层中被抽出的过程，在物理空间中表现为：全球12个灵机方舟节点同时检测到一个精确的0.2913Hz脉冲——幅度恰好=10^7 JE
- 这个脉冲不是爆炸，不是光芒。它是一个"校准信号"——像交响乐团定音叉敲响的A音，精确、纯粹、无衰减
- 0.1%缺口在0.7秒内被填满。同步达到100%

**感官设计（寂静）**：
- 在船长的灵机脉冲释放的0.7秒内，所有在场修行者的灵机感知同时"饱和"——过量灵机信号导致感官保护机制启动，所有声音、触觉、味觉、嗅觉同时消失
- 50万人在同一个瞬间经历了绝对的寂静。不是安静——是"没有"。没有风声、没有心跳声、没有血液流动声。连骨传导的低频振动都消失了
- 持续0.7秒。
- 然后一切恢复。但那0.7秒里，50万人"听到"了同一个东西：不是声音，是一种确定性——"有人在这里"。不是语言，不是图像，是50万个灵机场同时校准到同一个频率后的本体感知
- 陈叙的监控屏幕上，收割程序的状态从"ACTIVE"变为空白。不是"TERMINATED"——是空白。显示器的那个像素区域像是从来没有显示过任何东西

**叙事功能**：
1. 三个危机的感官层次形成递进：**听觉（失谐的刺耳）→ 触觉（过载的无感）→ 寂静（所有感官的消失）**。从"有但错"到"有但无"到"什么都没有"——恰好对应收割程序被瓦解的过程：从干扰到对抗到归零
2. 每个危机的解决者不同：**米莱（修行领袖）→ 韩溪（普通工程师）→ 船长（散逸者）**。从最强者到普通人到已死之人——人类对抗收割的力量不在于最强，在于每个位置上的人都做了该做的事
3. 危机三是全书第二部的情感顶点，但它的物理描写是最克制的——没有爆炸、没有光芒、没有呐喊。只有0.7秒的寂静和一块变空白的屏幕
