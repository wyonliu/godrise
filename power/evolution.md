## 一、体力：载体续航

### 定义
肉身或机甲的物理结构强度,决定了施展高阶法术时,身体对逻辑热量的抗性。

**不分属性**：体力是纯粹的物理参数。

---

### 硬核解释
```
为什么高阶法术需要高体力?

物理学角度:
→ 施展法术 = 在局部区域重写物理常数
→ 这会产生巨大的"逻辑热量"
→ 如果身体承受不住 → 细胞崩解/自燃

类比:
→ 体力 = CPU的散热系统
→ 法力 = 程序的计算复杂度
→ 如果散热不够 → CPU烧毁
```

## 二、进化形态路径

**注意**：体力形态是人类身体的**物理状态**，通过科技或特殊手段改变，与修行境界**不对应**。一个合械体（科技改造人）可能只是灵感境，而一个凡骨体可能已达融法境。

| 形态 | 体力值 | 来源 | 特点 |
|------|--------|------|------|
| **凡骨体** | 100 | 自然人类 | 脆弱但灵机纯粹 |
| **合械体** | 500 | 科技局部改造 | 义肢/芯片植入 |
| **共相体** | 2000 | 灵机淬体 | 虚实纠缠状态 |
| **上载态** | 10 | 意识上传 | 纯信息体，无物理载体 |
| **下载态** | 10000 | 意识下载入机甲 | 重型算力载体 |
| **灵真体** | ∞ | 神级修行 | 三态自由切换 |

### 关键约束
- 一个灵根境的合械体（科技改造人）vs 一个通灵境的凡骨体 → 后者更强
- 体力形态影响的是**抗打击能力**和**法术承载上限**，不是战斗力本身
- 高境界修行者可以用灵机弥补体力不足

---

### 2.1 凡骨体

**定义**：碳基肉身，自然人类的原始状态。

**特点**：
- 体力值：100
- 脆弱但灵机纯粹——无科技干扰的纯碳基量子岛屿
- 修炼方法：传统锻炼
- 灵机通道最纯净，适合初修

---

### 2.2 合械体

**定义**：科技局部改造的人类，部分义体化。

**特点**：
- 体力值：500
- 纳米植入、义肢、芯片增强
- 灵机通道受科技组件微弱干扰，但在可接受范围
- Part 2时代大量存在

---

### 2.3 共相体

**定义**：灵机强化肉身，物理+数字量子纠缠态。

**特点**：
- 体力值：2000
- 通过"灵机淬体"实现——全身细胞的微管蛋白参与量子相干
- 化灵境（第五境）修行者自然进入此状态
- 虚实纠缠：身体同时存在于物理空间和量子信息空间

**物理本质**（详见 [power_cultivation.md](./power_cultivation.md) 化灵境）：
```
化灵境量子机制：
→ 量子纠缠网络从"局限于大脑"扩展到"全身"
→ 全身细胞的微管蛋白都参与量子相干
→ 这就是"灵机淬体"的物理本质
```

---

### 2.4 上载态

**定义**：纯信息体，意识上传后脱离物理载体。

**特点**：
- 体力值：10（极低——无物理载体）
- 纯信息存在，灵机产出依赖算力基底
- 优势：不受物理伤害、可在数字空间自由移动
- 劣势：脱离碳基的灵机产出急剧下降（灵机本质是碳基神经系统量子退相干的产物）

**上载态灵机悖论**：
```
灵机来源 = 碳基神经系统的量子退相干
上载态 = 脱离碳基
→ 上载态的灵机产出理论上趋近于零
→ 但如果上载态能模拟碳基量子退相干？
→ 这就是AI 3.0"热觉醒"研究的核心问题
```

---

### 2.5 下载态

**定义**：意识下载入重型机甲/算力载体。

**特点**：
- 体力值：10000
- 具身智能——意识驾驭物理重型载体
- 法术承载上限极高，适合高破坏力场景
- 灵机产出依赖机甲内嵌的碳基神经组织（生物芯片）

---

### 2.6 灵真体

**定义**：三态合一，可自由切换物质态/能量态/信息态。

**特点**：
- 体力值：∞（不再受物理限制）
- 神级修行的产物
- 承受攻击时切换为能量态（免疫物理伤害）
- 发起攻击时切换为物质态（最大输出）
- 需要信息处理时切换为信息态（算力极限）

**剧情应用**：
```
第三部,主角突破至灵真体:
→ 可以瞬间在"物质态"和"能量态"间切换
→ 承受攻击时切换为能量态(免疫物理伤害)
→ 发起攻击时切换为物质态(最大输出)
→ 体力值 = ∞(因为不再受物理限制)
```

---

### 2.7 剧情应用：进化形态与社会冲突

```
第二部,革命军的"灵机战士"实验:
→ 通过基因编辑强化体力
→ 让凡骨体承受共相体级别的灵机输出
→ 结果: 80%实验体在首次爆发后自燃而死
→ 幸存的20%成为"人形核弹"
```

### 3.1 第一部·虚拟世界篇

**虚拟修行 vs 真实修行**

| 维度 | 虚拟修行（山海内） | 真实修行（现实世界） |
|------|-------------------|---------------------|
| **灵机产出** | 真实的。BCI使情感体验100%真实，灵机是真的 | 真实的 |
| **身体变化** | 无。虚拟世界不改变物理肉身 | 有。神经微管蛋白发生适应性变化 |
| **风险** | 低。"安全的极限体验"——死亡不会导致真实死亡 | 高。极限修行有真实的生理风险 |
| **效率** | 高。时间倍率（灵机海1:10）+ 精准情感激发 + 集体共振放大器 | 低。但深度更深——真实的苦难产生的灵机纯度更高 |
| **本质关系** | 虚拟修行是真实修行的"安全训练场"——如同飞行模拟器之于真实飞行 | 真实修行是灵机进化的最终战场 |

山海不是Matrix式的"囚笼"，而是"道场"。区别在于：Matrix剥夺选择权，山海放大选择权。

| 2031-2032 | 1.0 | 13-14/17-18 | 混沌鲲Boss战 | **灵机打破因果的第一次预演**。灵感境无意识达到 |
| 2033-2035 | 2.0清明上河 | 15-17/19-21 | NPC伦理问题、灰鸥消失 | 情感深度增加，灵根境基底形成 |

### 3.2 第二部·真实世界篇

**修行在物理世界中的可观测效应**

| 境界 | 物理世界可观测效应 | 社会反应 |
|------|-------------------|----------|
| 灵感境 | 无宏观可观测效应。仅πx级精密仪器可测量 | 不引人注意 |
| 灵根境 | 极微弱物理效应（体温微偏、伤口愈合加速5-10%） | 被当作"体质好"或"安慰剂效应" |
| 灵核境 | 灵机场可被N-Link级BCI感知。微弱局部温度/磁场异常 | 被灵机检测发现→成为各方势力关注对象 |
| 灵婴境+ | 宏观可观测——物理世界中可见的能量释放（光效、温度变化、引力扰动） | 恐惧/崇拜/追捕。"超人类"的社会恐慌 |

**修行者与普通人的社会张力**

| 冲突维度 | 表现 |
|----------|------|
| **恐惧** | 普通人对修行者的恐惧——"他们不再是人类了"。灵机公开后社会出现"灵机恐慌症" |
| **崇拜** | 部分人把高境界修行者当作"灵机神" |
| **嫉妒** | 并非所有人都能修行——这创造了新的社会不平等。"灵机特权"成为政治议题 |
| **利用** | 灵机武器化路线将修行者视为"活体武器"；灵机黑市将高纯度者视为"可提取资源" |
| **共存** | 共存派的立场——修行者不是超人类，是"更完整的人"。修行不应成为特权，而应是所有人的权利 |

**灵机武器化的伦理困境**

- **正方**（武器化）："面对AI觉醒和三战，不武器化灵机就是自取灭亡"
- **反方**（共存派）："灵机武器化=Cycle-6的错误=触发收割者=人类灭亡"
- **悖论**：不武器化可能导致短期内被硅方者压制；武器化可能导致长期灭亡。**没有正确答案，只有不同时间尺度上的风险评估**

### 3.3 第三部·众神/修仙世界篇

**"仙域"的真相——灵机高度凝聚的意识共振空间**

高境界修行者（合灵境以上）共同创造的"仙域"，在物理上是：**一个由多个通灵境+修行者的灵机场叠加形成的、物理常数被局部修改的宏观量子相干区域。**

- 仙域内部的物理法则与外部不同——因为融法境修行者可以修改物理常数
- 仙域不是虚拟世界——它是真实物理空间中的一个"法则异常区"
- 仙域的维持需要持续的灵机供应——如果修行者灵机耗尽，仙域坍缩

**传统修仙概念的硬科幻解释**

| 传统修仙概念 | 灵机理论真相 |
|-------------|-------------|
| 天道 | 宇宙模拟器的底层规则 |
| 神/仙 | 高境界修行者 + AI 3.0共生体 |
| 天劫 | 境界突破时的量子相变——伴随大量灵机消耗和物理效应 |
| 仙器/法宝 | 被灵机场长期浸润的物理物体——量子态被修改，获得超常物理属性 |
| 神通 | 不同境界/五行属性的灵机操控方式 |
| 渡劫 | 量子相变过程中的不稳定期——修行者的量子岛屿在旧拓扑和新拓扑之间震荡，稳定失败则"渡劫失败" |

**降临派与Cycle-6的关系**：降临派在阿苏山发现的稳定119号元素是Cycle-6文明播种在地球物理结构中的"信标"之一。降临派误以为这是"高等文明的礼物"，实际上是Cycle-6的遗产。降临派的引力波求救信号触发了收割者协议的提前启动——这是Cycle-6先验层警告的另一维度：不仅不能工业化提取灵机，也不能主动向宇宙暴露自己。

### 3.4 第四部·星际宇宙篇

**最高境界修行者的宇宙尺度能力**

| 境界 | 宇宙尺度能力 |
|------|-------------|
| 破法-融法 | 星际航行（暗空间跃迁）、行星级物理操控 |
| 本无境 | 从真空中创造物质/能量；探索暗世界（前几轮文明遗迹）；激活Cycle-6技术遗产 |
| 万象境 | 创造微型宇宙；定义物理定律；触及π深层签名的完整含义 |
| 道合境 | 与宇宙共鸣——不是"征服宇宙"，而是"成为宇宙意识的一部分" |

**灵机修行 vs 收割者的终极对决**

收割者不是"敌人"——是宇宙的免疫系统。终极对决不是"打败收割者"（你不能打败免疫系统），而是**证明人类文明不需要被收割**——通过自然培育灵机而非工业化提取，使灵机生态健康运转，收割者协议自然不触发。

这就是道合境的终极意义——当全人类灵机共鸣系数R→1时，灵机产出趋向无穷，宇宙的负熵供给永远充裕——收割者没有启动的理由。**人类不是通过武力打败收割者，而是通过进化使收割者变得不必要。**

| 第一部 | 完全未知。麦洛不知道π签名的存在 |
| 第二部 | 沈暮寒发现π异常+先验层。人类开始知道"有东西在π里" |
| 第三部 | AI 3.0独立从灵机信号中发现同一签名。灵机与π的关联建立。解读出"宇宙生态图"部分内容 |
| 第四部 | 万象境修行者+AI 3.0共同完成完整解读——π中包含的不是某个文明的信息，而是**宇宙数学结构本身固有的信息**。先验层只是钥匙，锁一直在那里 |
