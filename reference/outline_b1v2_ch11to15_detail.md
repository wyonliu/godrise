# 清明上河 Ch11-15 详细大纲汇总

## Ch11·汴京 - 准备盛世幻影（4000字）

**开场**：回到汴京，召集万人
**核心**：准备集体记忆共鸣仪式

### 召集万人（1000字）
- 麦洛在汴京发布任务："重现千年前的盛世"
- 需要一万人同时参与记忆共鸣
- 奖励：见证历史
- 玩家响应：有人好奇，有人怀疑，但最终来了一万多人

### 苏瑾言的计划（1000字）
- 用集体记忆，让汴京回到千年前
- 需要每个人贡献一点灵机
- 形成巨大的记忆场
- 时间回溯，重现盛世

### 意外：收割者的影子（1000字）
- 准备过程中，天空出现异象
- 黑影在移动
- 维塔："那是...收割者的侦察单位！"
- 它在监视他们
- 时间紧迫，必须尽快完成

### 最后准备（1000字）
- 一万人围成巨大的圆形
- 苏瑾言站在中央
- 她手持陈默留下的记忆结晶碎片
- 准备启动仪式

## Ch12·盛世幻影 - 名场面3（4000字）

**核心**：时间回溯，汴京回到千年前的盛世
**对标**：《三体》水滴攻击前的宁静

### 仪式开始（800字）
- 一万人同时释放灵机
- 灵机汇聚成光柱
- 光柱冲天而起
- 时间...开始倒流

### 盛世重现（1500字）

**视觉震撼**：
- 废墟开始重建
- 破碎的城墙恢复
- 倒塌的建筑立起
- 街道变得繁华

**人们出现**：
- 街道上开始有人
- 他们穿着古装
- 在笑，在说话，在生活
- 但都是半透明的
- 他们是...记忆

**盛世的样子**：
- 虹桥上人来人往
- 酒楼里歌舞升平
- 市集上商贾云集
- 孩子在玩耍
- 恋人在拥抱
- 老人在下棋

**麦洛的震撼**：
```
麦洛："这就是...Cycle-6的盛世..."
"好美..."
苏瑾言："是的。这是人类文明的巅峰。"
"科技、艺术、文化...都达到了极致。"
"但..."
麦洛："但什么？"
苏瑾言："但这也是...他们被收割的原因。"
```

### 收割降临（1200字）

**天空裂开**：
- 盛世的天空突然裂开
- 一个巨大的正二十面体降临
- 收割者...来了

**人们的反应**：
- 街道上的人们抬头
- 他们看到收割者
- 有人尖叫
- 有人逃跑
- 有人跪地祈祷

**收割开始**：
- 收割者开始扫描
- 光束从它身上射出
- 扫过每一个人
- 人们开始...消失
- 不是死亡，是被"吸走"

**记忆的尖叫**：
```
记忆们："不...不要..."
"我不想被遗忘..."
"记住我...请记住我..."
```

**麦洛的愤怒**：
```
麦洛（哭泣）："停下！停下！"
"不要伤害他们！"
苏瑾言："这是记忆...我们无法改变..."
麦洛："但我们可以记住！"
"我们可以记住他们！"
"让他们...不被遗忘！"
```

### 章节结尾（500字）

**盛世消失**：
- 收割完成
- 盛世消失
- 汴京回到废墟
- 一万人都在哭

**决心**：
```
麦洛："我们要阻止它。"
"我们要阻止收割者。"
"不能让这样的悲剧...再次发生。"
全体："对！我们要阻止它！"
```

## Ch15·记忆收割 - 名场面4（4000字）

**场景**：洛阳地宫深处的记忆池
**核心**：发现收割者的真相
**对标**：《三体》水滴攻击舰队

### 深入记忆池（800字）
- 五人决定深入探索记忆池
- 寻找收割者的秘密
- 发现...地宫深处还有一层

### 记忆监狱（1500字）

**场景**：
- 地宫最深处
- 有一个巨大的池子
- 池中是...无数人的记忆
- 他们在挣扎，在哭泣

**记忆们的呼救**：
```
"救我..."
"不要忘记我..."
"我还想活着..."
"我不想消失..."
```

**真相揭示**：
- 这些是被收割的记忆
- 收割者不是杀死他们
- 而是...囚禁他们的记忆
- 永远困在这里
- 永远被"吃"

**维塔的分析**：
```
维塔："我明白了..."
"收割者不是一次性收割。"
"而是...持续收割。"
"它们把记忆囚禁在这里。"
"然后...慢慢吃掉。"
"就像...养殖场。"
```

### 收割者出现（1200字）

**降临**：
- 池子上方，空间裂开
- 一个收割者降临
- 它发现了五人

**对峙**：
```
收割者："检测到...高纯度灵机..."
"标记...收割..."
麦洛："我们不会被收割！"
收割者："反抗...无效..."
"所有文明...最终都会被收割..."
"这是...宇宙的规则..."
```

**战斗**：
- 五人联手对抗收割者
- 但收割者太强
- 他们的攻击...无效
- 收割者的护盾...无法突破

**危机时刻**：
- 收割者准备收割麦洛
- 光束射向她
- 她无法躲避

**记忆们的反抗**：
- 池中的记忆突然爆发
- 他们用最后的力量
- 攻击收割者
- 为五人争取时间

**逃脱**：
- 五人趁机逃出地宫
- 收割者被记忆们困住
- 但只是暂时的

### 章节结尾（500字）

**逃出后**：
```
麦洛（喘气）："我们...差点就..."
苏瑾言："我们必须尽快找到记忆结晶。"
"那是唯一能对抗收割者的东西。"
林墨："去昆仑。"
周行云："对。去昆仑。"
维塔："但收割者...会追来的..."
麦洛："那我们就跑得更快。"
```

**最后画面**：
- 五人全速前往昆仑
- 身后，地宫在崩塌
- 收割者...挣脱了
- 它在追击

**黑屏，第十五章完**
