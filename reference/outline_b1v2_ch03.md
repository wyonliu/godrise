**场景**: 长安古道 → 遇到林墨  
**核心**: 林墨登场 + 画出已死之人 + 林染的故事

## 开场：长安古道（600字）

**场景**：荒凉废土，破碎石碑，倒塌城墙

**苏瑾言的解释**：
```
"这里是'遗忘之地'。"
"Cycle-6文明覆灭后，很多地方被遗忘。"
"没有人记得，就没有灵机维持。"
```

---

## 遇到林墨（1000字）

**第一次见面**：
- 古道旁有人在画画
- 28岁，白色长衫，眼神空洞而专注
- 画的是一个女孩

**对话**：
```
麦洛："你在画什么？"
林墨："我妹妹。"
麦洛："她在哪里？"
林墨："她死了。但只要我一直画她，她就还活着。"
```

**画中的林染**：
- 画在动！女孩的头发在飘，眼睛在眨
- 维塔检测到强烈灵机波动
- 林墨将妹妹的记忆画进了画里

**林墨的能力**：
- 记忆画师
- 将记忆转化为画
- 画中的人会"活"过来
- 代价：每画一次，灵机减少

---

## 林染的故事（1200字）

**三年前**：
- 林染患绝症，只有三个月
- 林墨带她来山海看最美的风景
- 他们一起看虹桥，吃糖葫芦，探险长安

**林染的"死亡"**：
```
林染："哥哥...我好累...我要回去了。"
"我的身体撑不住了。"
"哥哥不要哭。我很开心。"
"这是我最美好的记忆。"
```

- 林染的虚拟身体透明，消失
- 现实世界，林染去世

**林墨的执念**：
- 学习记忆画法
- 画出林染
- 画完成时，林染"活"了
- 她在画里笑，说话："哥哥，我回来了。"

**林墨的誓言**：
```
"我会画一千幅你的画。"
"让全世界都记得你。"
```
