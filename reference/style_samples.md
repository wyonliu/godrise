# 文笔定调·风格基准

> 本文件是《神临山海》正文写作的风格基准。所有章节以此为标尺。
>
> 四大奖标准自检：
> - **星云/雨果**：设定即哲学，不可替换
> - **诺贝尔**：感官精确，节奏即意义，陌生化而非装饰化
> - **茅盾**：用中国人的身体写作，用"做"替代"想"，用沉默替代倾诉
> - **古龙**：先写物后写人，身体替代心理，世界替你说话

---

## 风格戒律（写作时逐条自检）

1. **比喻限额**：每段最多一个比喻。比喻必须是唯一的——如果你在其他书里读过类似的句子，删掉重写。
2. **"像"字警报**：全章"像"字不超过15个。每次用"像"之前问自己：能不能不用比喻直接写？
3. **心理描写红线**：连续心理描写不超过三行。超过就改成身体语言。
4. **动词优先**：名词和形容词退后，让动词做工。"水缠上她的手指"比"温柔的水流"好。
5. **中国体温**：写感官时用中国人的经验——蹲、趴、拧、搓、哈气、缩脖子。不用西式动作。
6. **节奏控制**：长句铺陈后必须接短句收束。三个短句之后可以放一个长句呼吸。
7. **留白即力量**：最关键的情感用"没说"来表达。最震撼的画面用"她没动"来呈现。
8. **不解释**：读者能从画面推断的，不用旁白说明。信任读者的智商。
9. **声音设计**：每个场景要有独特的声音——水声、风声、沉默的质感。声音是最被忽略的感官。
10. **温度设计**：每个场景要有温度——热的、冷的、温的。温度比颜色更能让人"身临其境"。

九色渊的水是黑的。

不是夜晚的黑，不是深水的黑。是颜料用尽之前调色盘底部的那种黑——所有颜色混在一起，浓稠，沉默，像一口倒扣的钟。

渊面没有波纹。风吹不动它。鸟不从上方飞过。岸边的青石生着薄苔，苔痕到水线处齐齐断掉，像是被谁用尺子量过。

山围着渊。青绿色的山，颜色浓得发假，像王希孟还没来得及晕染就被叫走了，颜料直接拍在绢上，石青石绿顺着山脊往下淌。云卡在山腰，不动，实心的，发着银灰色的光。

安静。

安静了很久。

然后水面出现了一点金。

很小。像有人在水底点了一粒火星。金点扩开，变成橘色，橘色边缘泛出浅绿。三种颜色同时在黑水上铺开，像一幅画正被人从水底往上画。

渊底有什么东西在笑。

颜色扩散的速度和笑容的弧度同步——慢慢的，先是嘴角，然后是眼角，然后是整张脸。水面的橘色越来越亮，越来越大，黑水被推到渊的边缘，像帷幕被一双看不见的手拉开。

一个女孩从水里浮上来。

黑头发贴着脸，眼睛还闭着，嘴角翘着。她在笑。不知道在笑什么——也许是梦里有人叫了她的名字，也许是水的温度刚好，也许什么都没有，她就是那种醒来就会笑的人。

水面的颜色铺满了整个渊。

她睁开眼睛。

有一瞬——很短，三秒不到——她的眼神不对。不像十二岁。像是一个走了很远的路的人，突然站在一个完全陌生的路口，在辨认方向。然后那个眼神被好奇心盖过去了，沉到底下，像石子沉进水里。

她试着站起来。腿比预想的短了一截，膝盖磕在石头上。

"……怪了。"

她低头看自己的手。翻过来，翻过去。手太小了。指节细，掌纹浅，指甲是圆的，是小孩的指甲。她攥了攥拳头，又松开，像在确认这双手是不是自己的。

她蹲在岸边，把手伸进水里。

水缠上来了。不是流过——是缠。五色的水顺着指缝钻进去，一根手指一根手指地裹，像在量尺寸。一种轻微的震颤从指尖传到手腕，不疼，是痒，是一扇门在敲她的骨头，问她能不能进来。

她没缩手。

身后的水面动了。不是她弄的。水底有东西在往上浮，慢，带着一种不紧不慢的耐心。水面隆起，像有人从下面顶了一下。五色水往两边分，露出一只眼睛。

竖瞳。每一侧都比她身体宽。瞳孔里流着和池水一样的五色——碧绿、鹅黄、天蓝、淡紫、橘红——像五道年轮在一个圆里转。

那只眼睛没看她。

它看穿她。

目光从她胸口进去，在里面转了一圈，出来。像海关检查，像X光，像一个医生听完心跳说"下一个"。

麦洛站起来，冲它挥了挥手。

"你好。"

没有回声。水面的涟漪往回收——不是散开，是收回去。像倒放的录像。

眼睛闭上了。五色褪去。它沉下去，水面恢复平静，黑色重新从边缘往中间聚。像什么都没发生过。

麦洛蹲回去，歪着头看水面。水里只剩她自己的倒影——一个湿漉漉的、黑头发的、太小了的女孩。

她盯着倒影看了很久。

然后她说："你比我记得的……矮。"

是对倒影说的。也不完全是。

---

她在岸边拧头发里的水。

有什么东西蹭她脚踝。

低头。一只银色的猫，巴掌大，三条尾巴。正用脑袋蹭她的脚踝。蹭一下，抬头看一眼，再蹭一下。

"……你好？"

银猫打了个喷嚏。是银色的喷嚏——小光点从鼻子里喷出来，在空中散开，落在石头上，灭了。

"过敏。"它说。声音尖，快，像指甲划黑板但没那么让人难受。"你身上灵机浓度太高，我过敏。但你暖和。所以我忍了。"

麦洛还没张嘴。

"别摸肚子。别揪尾巴。别给我起名。我叫维塔。V-I-T-A。拉丁语。意思是'生命'。"

三条尾巴一条一条翘起来。

"现在——你是谁？为什么你笑的时候水会变色？你知不知道你刚才差点把整个渊都点亮了？"

麦洛看着它。它看着麦洛。

"你刚才在我口袋里吗？"

"那是系统分配的初始位置！根据《山海灵宠管理条例》第三章——"

"你多重？"

维塔的耳朵竖直了："三斤七两！精确到克！这和目前的议题有什么关系？"

"没关系。"麦洛把它拎起来，放在肩上。动作很顺，像做过很多次。"就想知道你掉下来我能不能接住。"

维塔在她肩膀上找了个平衡点，三条尾巴分别搭在她的领口、肩胛骨和后脑勺上，像三根安全带。

"这种体贴是不必要的。我有悬浮功能。"

"那你刚才为什么蹭我脚？"

维塔沉默了零点四秒。对一个AI来说，这是一个世纪。

"……节能模式下悬浮功能关闭。"

麦洛笑了。

水面又变色了。橘色，从她站着的地方扩出去，一直扩到渊的对岸。

维塔在她肩上回头看了一眼水面。它的传感器记录下这个数据——灵机纯度0.03%到0.15%的跳跃，好奇型波动，与Cycle-6档案匹配度极低。它想说什么。嘴张了一下，又闭上。

"那边是什么？"麦洛指着北面。山谷的方向。竹林的影子在云雾后面摇。

"瑶林。新手引导区。你应该从那里开始——"

"那更远的呢？"

"……更远的不是你现在该——"

"池底那个眼睛是什么？"

维塔的三条尾巴同时卷紧。

"你看见了。"不是问句。

"它不凶。"麦洛说。"就是……大。而且它看我的方式——"她停了一下，想找一个词，"像在等什么人来。但来的不是它等的那个。"

维塔没接话。

风从山谷里送来竹子的味道，青涩的、微苦的，像春天刚拱出土的笋。麦洛深吸了一口气，把那个味道吸进肺里。

"走吧。"她迈开步子。赤脚踩在碎石上，不疼——石头是温的，像被什么东西从下面烘着。

她一边走一边哼歌。跑调的，断断续续的，像是只记得几个音。维塔的耳朵抖了一下——它的声纹数据库里有一条被加密的记录，和这段旋律的相似度是百分之九十一点三。

它没告诉她。

身后，九色渊的水底，那只眼睛重新睁开了。瞳孔里五色流转的速度比刚才快了一点——快了百分之零点七。

像一颗心跳加快了。

## 文笔自检报告

### vs 旧稿对比

| 维度 | 旧稿 | 新稿 | 改进点 |
|------|------|------|--------|
| "像"字密度 | 约40+个 | 约12个 | 比喻精简，每个不可替代 |
| 心理描写 | 多处直接写心理 | 几乎全部用动作/对话替代 | "她没缩手">"她感到好奇" |
| 古龙式出场 | 直接写麦洛的第一人称感受 | 先写水变色→再写人浮出→最后才看清 | 符合角色圣经设计 |
| 维塔出场 | 从口袋里掏出来 | 蹭脚踝+打喷嚏（角色圣经方案） | 更有灵气 |
| 信息暴露 | 结尾大段数据播报 | 融入维塔内心，一句"它没告诉她" | 文学感+留白 |
| 中国体温 | 偏西式感受 | 蹲、拧头发、竹笋味、石头温度 | 更接地气 |
| 节奏 | 长句堆叠 | 长短交替，关键处用短句（"安静。安静了很久。"）| 呼吸感 |
| 声音设计 | 缺失 | 喷嚏声、水缠的声音、竹子味道、风 | 五感立体 |

### 四大奖自检

- **星云/雨果** ✓ 灵机的可视化（笑→水变色）自然嵌入，不硬解释
- **诺贝尔** ✓ "像一口倒扣的钟"——陌生化。"像海关检查，像X光"——精确。节奏有呼吸。
- **茅盾** ✓ 蹲着看水、拧头发、竹笋的味道——中国人的身体。沉默和不说的话承载情感。
- **古龙** ✓ 先水后人。身体替代心理。维塔替麦洛说出读者想知道的事。

### 待改进

1. 巨眼段落可以更克制——目前有一句"像海关检查"略轻佻，正文需要斟酌语气
2. 麦洛的"超龄感"可以更subtle——目前靠一句"你比我记得的矮"点出，但后续需要持续埋设
3. 维塔的数据感知段落（灵机纯度跳跃）保留了，但藏在了叙事里而非结尾播报——正文可以做得更隐蔽

## 正文写作时的文笔基准线

**每写完一章，用以下三个问题过一遍：**

1. 心理描写是否超过三行？→改成身体语言
2. 有没有一句话是你在其他作品中读过的？→重写
3. 最安静的段落是不是最有力量的？→如果不是，调整

**每个场景用以下感官清单检查：**

- [ ] 这个场景有声音吗？（不只是对话——环境声）
- [ ] 这个场景有温度吗？
- [ ] 这个场景有气味吗？
- [ ] 视觉描写是否在3句以内完成核心印象？
- [ ] 角色的情感是通过动作/对话传达的吗？（不是旁白）
