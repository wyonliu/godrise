## 权力的本质（1000字）

### 武则天的独白

**场景**：
- 记忆中的武则天看向五人
- 她知道这是记忆
- 但她还是想说

**独白**：
```
武则天："你们看到了我的一生。"
"你们觉得...我成功了吗？"
麦洛："你成为了皇帝..."
武则天："是的。我成为了皇帝。"
"但我失去了...做人的资格。"
"为了权力，我杀了无数人。"
"包括...我的亲人。"
"我的儿子，我的女儿，我的姐妹。"
"我杀了他们。"
"因为他们...威胁到我的权力。"
```

**权力的代价**：
```
武则天："权力是最甜蜜的毒药。"
"它给你一切。"
"但也夺走一切。"
"它让你站在顶峰。"
"但也让你...永远孤独。"
"因为...没有人能理解你。"
"没有人能信任你。"
"你只能...一个人。"
```

### 苏瑾言的共鸣

**对话**：
```
苏瑾言："我理解你的孤独。"
武则天："你？"
苏瑾言："我也失去了最爱的人。"
"但不是因为权力。"
"是因为...他选择了牺牲。"
武则天："那你恨他吗？"
苏瑾言："不。我爱他。"
"因为他的牺牲...是为了更多人。"
"不是为了权力。"
武则天（沉默）："也许...这就是区别。"
"我的牺牲，是为了自己。"
"他的牺牲，是为了别人。"
"所以...他被记住。"
"而我...被遗忘。"
```
