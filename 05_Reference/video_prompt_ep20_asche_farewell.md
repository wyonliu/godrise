# 视频Prompt · 金顶 · 日照金山 · 阿澈消散（15秒）

风格：水墨动画。参考雾山五行+千里江山图+你的名字。极致写意。笔触可见。万物皆墨。

**【0-3秒·金】**
大远景。黎明前最后的暗。梅里十三峰剪影如墨。天际线一条极细的金边——还没亮——在蓄力。
切近景：观景台。少女站在雪地上。赤脚。灰短衫。散发。灵种藤蔓七色微光在手腕上安静旋转。
她面前站着一个少年。逆光。轮廓是金色的——因为他的身体是半透明的——光直接穿过他。
少年手中举着一面巴掌大的木镜。镜面朝向少女。
第一道日出光线从他身体中穿过——像穿过一扇窗——照在少女脸上。
音效：无声。只有风。和两个人的呼吸。

**【3-7秒·裂】**
太阳从十三峰的第一座后面升起。金光自左向右依次点亮每座峰顶——一、二、三——像巨大的金色火炬被逐一点燃——到第七座时少年的身体开始碎了。
不是爆裂。是**飘散**。从指尖开始。像干透的水墨被风吹起——极细的墨粉——金色的——因为日出的光穿过他正在消散的身体——每一粒碎片都是金的——
少年没有低头看自己消散的手。他的目光一直在少女脸上。木镜一直举着。纹丝不动。
中景：少女伸手。去接他消散的碎片。手穿过了。什么也接不住。
少年微笑。嘴唇动了。无声。但能读出四个字：**"比画更真。"**
音效：古琴。单音。极慢。一个音拉满四秒。弦在颤——像心跳快停时最后的挣扎。

**【7-10秒·散】**
少年的身体从脚到头依次消散——金色碎片升入朝霞——和日照金山的光融在了一起——你分不清哪些是阳光哪些是他——
木镜从他消失的手中落下。旋转。慢镜。
镜面在旋转中依次反射：十三座金峰、少女的脸、天空、少女的脸、天空——
少女双手接住木镜。跪下。膝盖砸在雪地上。两个膝印。
她把镜子贴在胸口。低下头。头发垂下来遮住脸。
没有哭声。肩膀没有抖。
但——雪地上——在她膝盖前面——出现了一滴水。融化了雪。
然后第二滴。第三滴。
音效：古琴停。世界静了一秒。然后——一声极远的——唢呐。不是嘹亮的唢呐。是一支老旧的、被吹了很多年的唢呐的最低音——闷的——哑的——像一个不会哭的人发出的声音。

**【10-13秒·鸣】**
少女抬头。泪在脸上。但嘴是闭着的。她没有哭出声。
她站起来。把木镜揣进胸口衣襟里。
银猫从她身后走过来——走得很慢——尾尖的金色光环不再闪烁——是稳定的暗金——像一盏调到最低的灯——它也在忍。
猫跳上她的肩。用头蹭了一下她的耳朵。
少女伸手摸了一下猫。
然后她张嘴——不是说话——是**唱**——
如梦道。
一段没有歌词的旋律。从胸腔振出来。不是给活人听的。是给刚刚消散的那个人听的。
旋律的频率——让梅里十三峰上的积雪开始共振——雪粒从峰顶缓缓扬起——不是暴风雪——是十三条极轻的白色雪带——从十三座峰顶同时升起——在朝霞中旋转——像十三条白色的纱巾在金色的光中飘舞——
音效：女声旋律。无词。纯音。和唢呐合奏。一柔一刚。一生一死。融在一起。

**【13-15秒·立】**
大远景拉开。梅里十三峰。日照金山。满屏金色。
十三条白色雪带在金光中舞蹈。
少女的小小身影站在观景台上。银猫在肩。
脚下是雪。头顶是金。
她的身影在日出中拉出一条极长的影子——影子投在雪地上——影子的轮廓里——如果你仔细看——有一个更淡的影子叠在里面——
少年的影子。
已经消散了。但影子还没有完全走。像一个人离开了房间——但他用过的那把椅子上——还留着一点温度。
末帧定格：少女的剪影。猫的剪影。少年影子里的影子。十三座金峰。日出。
字幕浮现。竖排。草书。四个字——
**致画中人**
古琴最后一声。落。
