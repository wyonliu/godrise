# 视频Prompt · 泼墨 · 漓江灵机爆发+维塔变身（15秒）

风格：水墨动画。参考雾山五行+千里江山图。笔触可见，万物皆画。

**【0-3秒·静】**
大远景。暮色漓江。一叶破舟漂于墨水之上。两岸馒头山以一笔写成，山体在缓缓呼吸。少女立于船头——短黑发、灰色短衫、赤脚、左腕一圈翠色藤蔓微光。银猫蹲在她肩上。一条墨鱼跃水，拖出一道墨线逗号悬于半空不散。
古琴一声。极沉极远。

**【3-6秒·裂】**
少女张口——不是歌——是翠绿灵机暴从体内炸出——冲击波掀翻一切——
河水倒流升天，成为头顶的第二条江。鱼在两层水之间的空气中游。
颜色从万物身上剥离：墨黑脱水成雾球、石绿脱山成纱幕、云白化为坠落的光雨——整个世界变成被拆散的画，层层分离悬浮——
少女被灵机托入半空。藤蔓光从腕部疯长至肩颈。双瞳翠绿如星。
音效：古琴弦断。大地心跳骤加速。空灵女声不唱词——唱墨流动的声音。

**【6-10秒·变】**
倾斜甲板上。银色小猫迈步。逆灵机风暴。毛发压平。它渺小如尘——但它在走。
第一步：尾尖蓝光变金。金色沿脊背上行。
第二步：背弓隆起，肩胛外突，体型翻倍再翻倍。
第三步：银毛转银金色。皮下浮现虎纹——金色活体电路，脉动如活物。
第四步：一丈银金灵虎。肌肉线条毕现。金属利爪。
始终是同一双琥珀竖瞳——眼睛不变——证明是同一个灵魂。
侧向跟拍一镜到底。背景是悬浮的七色碎片世界。
太鼓心跳节奏渐强。每增大一级配一声重低音。虎形完成时一声巨响——切入半秒绝对寂静。

**【10-13秒·抱】**
灵虎跃起。0.5倍速。银金弧线穿过悬浮色彩的碎片世界。倒流河水炸为千颗水珠悬停半空——每一颗都映出飞跃中的虎影。
虎扑住悬空的少女。四肢环绕。尾卷其腰。虎头抵在她胸口。
金色虎纹猛亮——吞噬溢出的翠绿灵机——光从少女体内流入虎体——纹路从尾至头逐道点亮——
灵机风暴骤停。墨黑砸回水中。石绿弹回山上。头顶倒悬之江倾落，满屏水花。
音效：二胡琵琶古琴齐鸣一秒——一刀切入寂静——一声水花——一口呼吸。

**【13-15秒·落】**
二者坠落。虎在缩。金退为银。体量递减。触及甲板时已恢复原形——巴掌大银猫。
少女双手接住。跪于湿木板上。散发。赤脚。把猫贴在胸口。猫眼半阖。毛间升起细烟。
大远景拉开：墨江平。山静。暮沉为夜。半空中墨鱼的逗号仍在，缓缓消散。
少女脸上不是怕——是爱——差点失去后才懂的那种。
末帧：猫尾尖光环微闪——蓝、金、蓝、金——定在金色。弱。但稳。
古琴一声。同开场之音。但暖了。
