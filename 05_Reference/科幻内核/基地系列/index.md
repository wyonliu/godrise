# 基地系列内核提取

## 核心概念

### 1. 心理史学
**原文概念**：通过数学方法预测大规模人群的行为。

**内核提取**：
- 历史发展的可预测性
- 统计学在历史中的应用
- 个体与群体的关系

**改造方向**：
- 可以表现为：AI对历史趋势的分析
- 但强调"意外因素"的重要性，不完全决定论

### 2. 银河帝国
**原文概念**：横跨银河的庞大政治实体。

**内核提取**：
- 超大规模政治体的管理
- 中央与地方的关系
- 帝国的兴衰周期

**改造方向**：
- 在《神临山海》中可以表现为：跨星系联邦的治理难题

### 3. 历史必然性
**原文概念**：某些历史事件是必然发生的。

**内核提取**：
- 历史发展的规律性
- 偶然与必然的关系
- 个人在历史中的作用

**改造方向**：
- 可以探讨：技术发展是否必然导致某些结果

## 叙事技巧

### 多线叙事
- 不同时间点的故事
- 不同角色的视角
- 最终汇聚

### 历史感
- 时间跨度大
- 历史事件的因果
- 宏观视角
