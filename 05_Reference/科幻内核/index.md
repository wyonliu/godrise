# 科幻作品内核提取参考库

> **目的**：从经典科幻作品中提取核心概念、设定思路、叙事技巧，为《神临山海》提供创作灵感

## 📖 参考作品分类

### 硬科幻经典

#### [三体系列](./三体系列/index.md)
- **作者**：刘慈欣
- **核心概念**：黑暗森林法则、降维打击、文明冲突
- **叙事特点**：宏大叙事、时间跨度大、硬科幻逻辑
- **可借鉴**：宇宙社会学、技术爆炸、文明等级

#### [基地系列](./基地系列/index.md)
- **作者**：艾萨克·阿西莫夫
- **核心概念**：心理史学、银河帝国、历史必然性
- **叙事特点**：多线叙事、时间跳跃、历史感
- **可借鉴**：宏观历史视角、社会结构设计

#### [沙丘系列](./沙丘系列/index.md)
- **作者**：弗兰克·赫伯特
- **核心概念**：生态哲学、政治权谋、宗教与权力
- **叙事特点**：多视角、复杂政治、生态思考
- **可借鉴**：世界观深度、政治体系设计

#### [海伯利安系列](./海伯利安系列/index.md)
- **作者**：丹·西蒙斯
- **核心概念**：时间坟墓、终极智能、人类进化
- **叙事特点**：多线叙事、文学性、哲学思辨
- **可借鉴**：叙事结构、哲学深度

#### [神经漫游者系列](./神经漫游者系列/index.md)
- **作者**：威廉·吉布森
- **核心概念**：赛博空间、人工智能、后人类
- **叙事特点**：赛博朋克美学、技术细节、反乌托邦
- **可借鉴**：技术描写、未来感营造

## 🎯 提取原则

1. **概念提取**：提取核心科幻概念，而非直接复制
2. **本土化改造**：结合中国文化背景进行改造
3. **逻辑自洽**：确保提取的概念符合《神临山海》的世界观
4. **创新融合**：多个概念融合，创造新意

## 📝 使用方式

在创作时，可以参考这些内核，但必须：
- 进行创新性改造
- 符合《神临山海》的世界观
- 保持原创性和独特性
