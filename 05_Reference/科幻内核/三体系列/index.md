# 三体系列内核提取

## 核心概念

### 1. 黑暗森林法则
**原文概念**：宇宙就是一座黑暗森林，每个文明都是带枪的猎人。

**内核提取**：
- 文明间的猜疑链
- 技术爆炸的不确定性
- 生存是第一需要

**改造方向**：
- 在《神临山海》中，可以表现为：未知接触的谨慎、信息封锁的必要性
- 但可以加入"合作可能性"的探讨，不完全悲观

### 2. 降维打击
**原文概念**：高维文明对低维文明的碾压式攻击。

**内核提取**：
- 技术代差的绝对性
- 维度概念的物理化
- 文明等级的不可逾越

**改造方向**：
- 可以表现为空间技术的不同层次
- 曲率引擎的不同等级对应不同"维度"的理解

### 3. 技术爆炸
**原文概念**：低级文明可能在短时间内技术突飞猛进。

**内核提取**：
- 技术发展的非线性
- 关键技术的突破性
- 文明跃迁的可能性

**改造方向**：
- 在《神临山海》中可以表现为：某个关键技术的突破改变了整个格局

## 叙事技巧

### 宏大叙事
- 时间跨度大
- 空间范围广
- 多线并进

### 硬科幻逻辑
- 严格遵守物理法则
- 技术细节真实
- 逻辑严密
